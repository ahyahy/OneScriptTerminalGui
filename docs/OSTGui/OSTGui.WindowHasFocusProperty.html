Окно.Сфокусирован (Window.HasFocus) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, имеет ли объект фокусировку.

Примечание

Пример

Сфокусирован = Окно4.Сфокусирован;
