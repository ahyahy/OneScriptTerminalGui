ТерминалФормыДляОдноСкрипта.Отправитель (OneScriptTerminalGui.Sender) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает источник (объект) последнего события.

Примечание

Пример

Отправитель = АргументыСобытия1.Отправитель;
