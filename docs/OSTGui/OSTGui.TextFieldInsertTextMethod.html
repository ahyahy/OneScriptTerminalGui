ПолеВвода.ВставитьТекст (TextField.InsertText) Метод

Синтаксис

ВставитьТекст(<Текст>, <ИспользоватьСтаруюПозициюКурсора>)

Параметры

Текст (обязательный)
Тип: Строка.
Текст для добавления.
ИспользоватьСтаруюПозициюКурсора (необязательный)
Тип: Булево.
Использовать старую позицию курсора. Значение по умолчанию - Истина.

Возвращаемое значение

Описание

Вставляет указанный текст в текущую позицию курсора, как если бы пользователь только что его набрал.

Примечание

Пример

ИспользоватьСтаруюПозициюКурсора = Ложь;
ПолеВвода1.ВставитьТекст("Добавленный текст. ", ИспользоватьСтаруюПозициюКурсора);
Полный пример кода 



Перем ТФ, ПолеВвода1, Текст;

Процедура ПриОткрытии() Экспорт
	ИспользоватьСтаруюПозициюКурсора = Ложь;
	ПолеВвода1.ВставитьТекст("Добавленный текст. ", ИспользоватьСтаруюПозициюКурсора);
	Текст = ПолеВвода1.Текст;
	ТФ.Уведомление().Показать("Текст = " + Текст, -1);
