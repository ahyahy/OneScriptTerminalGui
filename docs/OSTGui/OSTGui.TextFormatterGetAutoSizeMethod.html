ОформительТекста.ПолучитьАвтоРазмер (TextFormatter.GetAutoSize) Метод

Синтаксис

ПолучитьАвтоРазмер()

Возвращаемое значение

Тип: Размер (Size).

Описание

Возвращает размер, чтобы соответствовать всему тексту.

Примечание

Пример

АвтоРазмер = ОформительТекста1.ПолучитьАвтоРазмер();

	Окно1.Высота = ТФ.Величина.Абсолютно(АвтоРазмер.Высота);
