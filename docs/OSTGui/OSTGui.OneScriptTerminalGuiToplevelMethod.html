ТерминалФормыДляОдноСкрипта.Верхний (OneScriptTerminalGui.Toplevel) Метод

Синтаксис

Верхний(<Прямоугольник>)

Параметры

Прямоугольник (необязательный)
Тип: Прямоугольник (Rect).
Прямоугольник, определяющий положение и размер объекта относительно родителя.

Возвращаемое значение

Тип: Верхний (Toplevel).

Описание

Создаёт новый экземпляр класса Верхний (Toplevel).

Примечание

Если параметр Прямоугольник задан, экземпляр инициализируется со свойством СтильКомпоновки (LayoutStyle) установленным в значение Абсолютно (Absolute).

Если не задан параметр Прямоугольник, экземпляр инициализируется со свойством СтильКомпоновки (LayoutStyle) установленным в значение Вычислено (Computed).

Верхний1 = ТФ.Верхний();
