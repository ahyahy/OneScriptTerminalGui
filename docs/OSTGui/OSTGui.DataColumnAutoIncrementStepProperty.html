КолонкаДанных.ШагПриращения (DataColumn.AutoIncrementStep) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает шаг приращения для колонки, свойству которого АвтоПриращение (AutoIncrement) задано значение Истина. Значение по умолчанию - 1.

Примечание

Пример

Колонка5.ШагПриращения = 5;
Полный пример кода 



Перем ТФ, Колонка5;

Функция РазобратьСтроку(Строка, Разделитель)
	Стр = СтрЗаменить(Строка,Разделитель,символы.ПС);
	М = Новый Массив;
	Если ПустаяСтрока(Стр) Тогда
		Возврат М;
	КонецЕсли;
	Для Ч = 1 По СтрЧислоСтрок(Стр) Цикл
		М.Добавить(СтрПолучитьСтроку(Стр,Ч));
	КонецЦикла;
	Возврат М;
КонецФункции

Процедура ПриОткрытии() Экспорт
	ШагПриращения = Колонка5.ШагПриращения;
	ТФ.Уведомление().Показать("ШагПриращения = " + ШагПриращения, -1);
