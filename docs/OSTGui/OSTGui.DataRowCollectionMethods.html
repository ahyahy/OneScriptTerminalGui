СтрокиДанных (DataRowCollection) Методы

Методы

Вставить (InsertAt) Вставляет новую строку в коллекцию по указанному индексу.
Добавить (Add) Добавляет указанный объект СтрокаДанных (DataRow) в коллекцию.
Очистить (Clear) Очищает коллекцию от всех строк.
Получить (Get) Возвращает значение по указанному индексу из коллекции.
Удалить (Remove) Удаляет строку из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет строку по указанному индексу из коллекции.
Элемент (Item) Устанавливает и/или возвращает указанный объект СтрокаДанных (DataRow) по указанному индексу в коллекции.
