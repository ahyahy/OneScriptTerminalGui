Дерево.ДобавитьУзлы (TreeView.AddObjects) Метод

Синтаксис

ДобавитьУзлы(<МассивУзлов>)

Параметры

МассивУзлов (обязательный)
Массив

Возвращаемое значение

Описание

Добавляет множество новых объектов корневого уровня. Объекты, которые уже являются корневыми, игнорируются.

Примечание

Пример

Массив = Новый Массив();
Для А = 9 По 60 Цикл
	Массив.Добавить(ТФ.УзелДерева("Узел" + А));
КонецЦикла;
Дерево1.ДобавитьУзлы(Массив);
Полный пример кода 



Перем ТФ, Дерево1, Узел1;

Процедура ПриОткрытии() Экспорт
	Массив = Новый Массив();

	Корневые = Дерево1.Корневые;
