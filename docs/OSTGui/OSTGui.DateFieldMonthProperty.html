ПолеДаты.Месяц (DateField.Month) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество месяцев.

Примечание

Пример

ПолеДаты1.Месяц = 12;
Месяц = ПолеДаты1.Месяц;
