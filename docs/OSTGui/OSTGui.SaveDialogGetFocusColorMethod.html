ДиалогСохранения.ЦветФокуса (SaveDialog.GetFocusColor) Метод

Синтаксис

ЦветФокуса()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему для текущего фокуса.

Примечание

Пример

ЦветФокуса = ДиалогСохранения1.ЦветФокуса();
Полный пример кода 



Перем ТФ, ДиалогСохранения1;

Процедура ПриОткрытии() Экспорт
	ЦветФокуса = ДиалогСохранения1.ЦветФокуса();
	Стр = "ЦветФокуса() = " + ЦветФокуса + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветФокуса.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветФокуса.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
