Флажок (CheckBox) Конструктор

Синтаксис

Флажок(<Текст>, <Икс>, <Игрек>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст для флажка. Значение по умолчанию - "Флажок".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла флажка. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла флажка. Значение по умолчанию - 1.

Описание

Создаёт новый экземпляр класса Флажок (CheckBox).

Примечание

Пример

Флажок1 = ТФ.Верхний.Добавить(ТФ.Флажок("Ф_лажок1"));
