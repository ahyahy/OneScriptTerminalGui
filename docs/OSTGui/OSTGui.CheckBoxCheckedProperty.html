Флажок.Помечен (CheckBox.Checked) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, помечен ли флажок.

Примечание

Пример

Флажок1.Помечен = Истина;
Полный пример кода 



Перем ТФ, Флажок1, Помечен;

Процедура ПриОткрытии() Экспорт
	Помечен = Флажок1.Помечен;
	ТФ.Уведомление().Показать("Помечен = " + Помечен, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Флажок1 = ТФ.Верхний.Добавить(ТФ.Флажок("Ф_лажок1"));
Флажок1.Помечен = Истина;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");
