Верхний.Границы (Toplevel.Bounds) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rect).

Возвращает прямоугольник, используемый для элемента.

Примечание

Пример

Верхний1.Текст = "Верхний1.Границы = " + Верхний1.Границы + "
|Верхний1.Границы.Икс = " + Верхний1.Границы.Икс + "
|Верхний1.Границы.Игрек = " + Верхний1.Границы.Игрек + "
|Верхний1.Границы.Ширина = " + Верхний1.Границы.Ширина + "
|Верхний1.Границы.Высота = " + Верхний1.Границы.Высота + "
|";

Верхний1.Ширина = ТФ.Величина.Абсолютно(80);

ЦветоваяСхема1 = ТФ.ЦветоваяСхема();

Верхний1.Текст = "Давным-давно в городке на берегу Средиземного моря жил старый столяр Джузеппе по прозванию Сизый Нос.";
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
ТФ.Верхний.Добавить(Верхний1);

Прямоугольник1 = ТФ.Прямоугольник(1, 9, 30, 5);
Верхний2 = ТФ.Верхний(Прямоугольник1);
Верхний2.Текст = "– Неплохая вещь, – сказал сам себе Джузеппе, – можно смастерить из него что-нибудь вроде ножки для стола…";
Граница2 = ТФ.Граница();
Граница2.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний2.Граница = Граница2;
Верхний2.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Верхний2.Ширина = ТФ.Величина.Процент(70);
// Мы создали Верхний2 используя в конструкторе прямоугольник.
// Чтобы сработала установка новой ширины через ТФ.Величина.Процент(70) нужно 
// стиль компоновки поменять на ТФ.СтильКомпоновки.Вычислено
Верхний2.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
ТФ.Верхний.Добавить(Верхний2);

ТФ.Запуск();
