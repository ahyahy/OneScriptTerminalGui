Кнопка.ЦветВыделенного (Button.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветВыделенного = Кнопка0.ЦветВыделенного();
Полный пример кода 



Перем ТФ, Кнопка0;

Процедура Кнопка1_Нажатие() Экспорт
	ЦветВыделенного = Кнопка0.ЦветВыделенного();
	Кнопка0.Текст = "ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ЦветВыделенного.ОсновнойЦвет + "
	|ЦветФона = " + ЦветВыделенного.ЦветФона + "
	|";
КонецПроцедуры
