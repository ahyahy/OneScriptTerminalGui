Переключатель.Высота (RadioGroup.Height) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает высоту элемента.

Примечание

Пример

Высота = Переключатель1.Высота;
Полный пример кода 



Перем ТФ, Высота, Переключатель1;

Процедура ПриОткрытии() Экспорт
	Высота = Переключатель1.Высота;
	ТФ.Уведомление().Показать("Высота = " + Высота, -1);
