Атрибут (Attribute) Конструктор

Синтаксис

Атрибут(<ЦветОсновнойИФоновый>)

Атрибут(<ОсновнойЦвет>, <ЦветФона>)

Параметры

ЦветОсновнойИФоновый (обязательный)
Тип: Цвет (Color).
ОсновнойЦвет (обязательный)
Тип: Цвет (Color).
ЦветФона (обязательный)
Тип: Цвет (Color).

Описание

Создаёт новый экземпляр класса Атрибут (Attribute).

Примечание

Конструктор с параметром (ЦветОсновнойИФоновый) инициализирует новый экземпляр с одним и тем же цветом для переднего плана и фона.

Конструктор с двумя параметрами (ОсновнойЦвет и ЦветФона) инициализирует новый экземпляр с цветами для переднего плана и фона.

Пример

Атрибут1 = ТФ.Атрибут(ТФ.Цвет.ЯркоГолубой, ТФ.Цвет.Синий);
Атрибут2 = ТФ.Атрибут(ТФ.Цвет.Красный);
Полный пример кода 



Перем ТФ, Атрибут1, ОсновнойЦвет;

Процедура ПриОткрытии() Экспорт
	ОсновнойЦвет = Атрибут1.ОсновнойЦвет;
	ТФ.Уведомление().Показать("Атрибут1.ОсновнойЦвет = " + ТФ["Цвет"].Имя(ОсновнойЦвет), -1);
