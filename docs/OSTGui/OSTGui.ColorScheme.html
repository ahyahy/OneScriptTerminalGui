ЦветоваяСхема (ColorScheme) Класс

Описание

Определяет цветовые свойства для общих видимых элементов. Контейнеры, такие как окно и кадр, используют цветовую схему чтобы определить цвета, используемые подэлементами.

Все элементы настроены с использованием цветовой схемы, которая будет работать как на цветных терминалах, так и на более ограниченных чёрно-белых терминалах.

Различные стили (цветовые схемы) представлены в классе Цвета (Colors) и определяют цветовые схемы для верхнего уровня, обычных элементов, строки меню, всплывающих диалоговых окон и диалоговых окон ошибок.

Цветовая схема представляет собой четыре значения:

  • Цвет, используемый для обычного текста.
  • Цвет, используемый для обычного текста, когда элемент находится в фокусе.
  • Цвет для клавиш вызова в режиме фокусировки.
  • Цвет для клавиш вызова в режиме без фокусировки.

    • Важно: Объект ЦветоваяСхема (ColorScheme) нельзя переиспользовать. То есть при обращении к свойству элемента, имеющему тип значения ЦветоваяСхема (ColorScheme) или при присваивании свойству элемента значения этого типа каждый раз будет создан новый экземпляр объекта с типом ЦветоваяСхема (ColorScheme).
