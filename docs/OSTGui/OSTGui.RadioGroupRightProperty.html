Переключатель.Право (RadioGroup.Right) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси X, являющейся суммой значений свойств Икс (X) и Ширина (Width) данного элемента.

Примечание

Пример

Право = Переключатель1.Право;
Полный пример кода 



Перем ТФ, Право, Переключатель1;

Процедура ПриОткрытии() Экспорт
	Право = Переключатель1.Право;
	ТФ.Уведомление().Показать("Право = " + Право, -1);
