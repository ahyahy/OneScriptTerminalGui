ВертикальноеВыравниваниеТекста (VerticalTextAlignment) Перечисление

Описание

Указывает вертикальное выравнивание текста.

Члены

Имя члена Описание Значение
Верх (Top) Текст выравнивается по верхней части кадра. 0
Низ (Bottom) Текст выравнивается по нижней части кадра. 1
Подобранный (Justified) Подобранное выравнивание текста в кадре. 3
Середина (Middle) Текст выравнивается в кадре по середине. 2
