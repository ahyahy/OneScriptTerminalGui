Кнопка.Границы (Button.Bounds) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rect).

Возвращает прямоугольник, используемый для элемента.

Примечание

Пример

Кнопка1.Текст = "Кнопка1.Границы = " + Кнопка1.Границы + "
|Кнопка1.Границы.Икс = " + Кнопка1.Границы.Икс + "
|Кнопка1.Границы.Игрек = " + Кнопка1.Границы.Игрек + "
|Кнопка1.Границы.Ширина = " + Кнопка1.Границы.Ширина + "
|Кнопка1.Границы.Высота = " + Кнопка1.Границы.Высота + "
|";

Окно1.Ширина = ТФ.Величина.Абсолютно(50);

Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
