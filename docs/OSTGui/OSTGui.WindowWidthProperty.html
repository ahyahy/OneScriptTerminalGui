Окно.Ширина (Window.Width) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает ширину элемента.

Примечание

Пример

Окно0.Ширина = ТФ.Величина.Абсолютно(90);
Полный пример кода 



Перем Окно1, Окно0;

Процедура ПриОткрытии() Экспорт
	Ширина = Окно1.Ширина;
	Окно0.Текст = "Окно1.Ширина = " + Ширина;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Окно0 = ТФ.Окно("Окно0");
Окно0.Икс = ТФ.Позиция.Абсолютно(25);
Окно0.Игрек = ТФ.Позиция.Абсолютно(1);
Окно0.Ширина = ТФ.Величина.Абсолютно(90);
Окно0.Высота = ТФ.Величина.Абсолютно(27);
ТФ.Верхний.Добавить(Окно0);

Окно1 = ТФ.Окно();
