ДиалогОткрытия.УдалитьСочетаниеКлавиш (OpenDialog.RemoveShortcut) Метод

Синтаксис

УдалитьСочетаниеКлавиш(<СочетаниеКлавиш>)

Параметры

СочетаниеКлавиш (обязательный)
Тип: Клавиши (Keys); Число.
Сочетание клавиш.

Возвращаемое значение

Описание

Удаляет сочетание клавиш для элемента.

Примечание

Пример

ДиалогОткрытия1.УдалитьСочетаниеКлавиш(ТФ.Клавиши.d_low);
