Дерево.ОбеспечитьОтображение (TreeView.EnsureVisible) Метод

Синтаксис

ОбеспечитьОтображение(<Узел>)

Параметры

Узел (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Описание

Обеспечивает отображение на экране указанного в параметре узла.

Примечание

Пример

Дерево1.ОбеспечитьОтображение(Узел);
Полный пример кода 



Перем ТФ, Дерево1;

Процедура ПриОткрытии() Экспорт
	Массив = Новый Массив();
	Для А = 9 По 60 Цикл
		Массив.Добавить(ТФ.УзелДерева("Узел" + А));
	КонецЦикла;
	Дерево1.ДобавитьУзлы(Массив);
	
	Узел = Дерево1.ПолучитьПоИндексу(55);
	Дерево1.ОбеспечитьОтображение(Узел);
КонецПроцедуры
