Кнопка.Выше (Button.PlaceTop) Метод

Синтаксис

Выше(<Элемент>, <Смещение>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент, относительно которого производим смещение.
Смещение (обязательный)
Тип: Число.
Величина смещения в целых числах нижнего края смещаемого элемента от верхнего края объекта, указанного в параметре Элемент.

Возвращаемое значение

Описание

Смещает выше этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.

Примечание

Свойство СтильКомпоновки (LayoutStyle) будет установлено в значение Вычислено (Computed).

Пример

Кнопка5.Выше(Кнопка1, 1);
