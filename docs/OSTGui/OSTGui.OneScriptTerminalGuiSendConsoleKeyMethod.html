ТерминалФормыДляОдноСкрипта.ОтправитьКлавишуКонсоли (OneScriptTerminalGui.SendConsoleKey) Метод

Синтаксис

ОтправитьКлавишуКонсоли(<Клавиша>, <МодификаторShift>, <МодификаторAlt>, <МодификаторCtrl>)

Параметры

Клавиша (обязательный)
Тип: КлавишиКонсоли (ConsoleKey).
МодификаторShift (необязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Shift. Значение по умолчанию - Ложь.
МодификаторAlt (необязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Alt. Значение по умолчанию - Ложь.
МодификаторCtrl (необязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Ctrl. Значение по умолчанию - Ложь.

Возвращаемое значение

Описание

Отправляет клавишу консоли без использования клавиатуры.

Примечание

Важно: Данный метод можно применить только после запуска приложения. То есть после применения методов:
ТФ.Запуск();
ТФ.ЗапуститьИЗавершить();

Важно: Не работает в Unix системах.

Пример

ТФ.ОтправитьКлавишуКонсоли(ТФ.КлавишиКонсоли.Tab);
