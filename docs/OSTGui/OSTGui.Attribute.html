Атрибут (Attribute) Класс

Описание

Представляет класс, задающий цвет переднего плана и фона.

Класс необходим для сопоставления цветов с возможностями терминала, в котором может не быть цветов. Он кодирует как цвет переднего плана, так и цвет фона и используется в классе ЦветоваяСхема (ColorScheme) для определения цветовых схем, которые можно использовать в приложении.
