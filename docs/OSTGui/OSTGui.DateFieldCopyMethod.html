ПолеДаты.Копировать (DateField.Copy) Метод

Синтаксис

Копировать()

Возвращаемое значение

Описание

Копирует выделенный текст в буфер обмена.

Примечание

Пример

ПолеДаты1.Копировать();
Полный пример кода 



Перем ТФ, ПолеДаты1;

Процедура ПриОткрытии() Экспорт
	ПолеДаты1.ВыбратьВсе();
	ПолеДаты1.Копировать();
	ТФ.Уведомление().Показать("ТФ.ТекстБуфераОбмена = " + ТФ.ТекстБуфераОбмена, -1);
КонецПроцедуры
