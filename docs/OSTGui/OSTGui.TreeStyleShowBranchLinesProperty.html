СтильДерева.ПоказатьВертикальные (TreeStyle.ShowBranchLines) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее отображение вертикальных линий под развёрнутыми узлами. Установите значение Истина для отображения вертикальных линий под развёрнутыми узлами, чтобы показать, к какому родительскому узлу относится тот или иной узел. Ложь - для использования только пробелов.

Примечание

Пример

СтильДерева1.ПоказатьВертикальные = Ложь;
Полный пример кода 



Перем ТФ, СтильДерева1;

Процедура ПриОткрытии() Экспорт
	ПоказатьВертикальные = СтильДерева1.ПоказатьВертикальные;
	ТФ.Уведомление().Показать("ПоказатьВертикальные = " + ПоказатьВертикальные, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Дерево1 = ТФ.Верхний.Добавить(ТФ.Дерево(1, 3, 40, 15));
Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Дерево1.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);

Узел1 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел1"));
Узел2 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел2"));
Узел3 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел3"));
Узел1_1 = Узел1.Добавить(ТФ.УзелДерева("Узел1_1"));
Узел1_2 = Узел1.Добавить(ТФ.УзелДерева("Узел1_2"));
Узел1_1_1 = Узел1_1.Добавить(ТФ.УзелДерева("Узел1_1_1"));
Дерево1.РазвернутьВсе();

СтильДерева1 = ТФ.СтильДерева();
СтильДерева1.ПоказатьВертикальные = Ложь;
Дерево1.Стиль = СтильДерева1;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
