Текстовый.ТекстТекущейСтроки (TextView.GetCurrentLine) Метод

Синтаксис

ТекстТекущейСтроки()

Возвращаемое значение

Тип: Строка.

Описание

Возвращает символы текущей строки (где находится курсор).

Примечание

Пример

ТекстТекущейСтроки = Текстовый1.ТекстТекущейСтроки();
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.ПозицияКурсора = ТФ.Точка(0, 4);
	ТекстТекущейСтроки = Текстовый1.ТекстТекущейСтроки();
	ТФ.Уведомление().Показать("ТекстТекущейСтроки = " + ТекстТекущейСтроки, -1);
