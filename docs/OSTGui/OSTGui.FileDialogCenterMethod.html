ФайловыйДиалог.Центр (FileDialog.Center) Метод

Синтаксис

Центр(<СмещениеИкс>, <СмещениеИгрек>)

Параметры

СмещениеИкс (необязательный)
Тип: Произвольный.
Величина смещения по оси икс.
СмещениеИгрек (необязательный)
Тип: Число.
Величина смещения по оси игрек.

Возвращаемое значение

Описание

Центрирует элемент управления внутри своего родителя с учетом параметров смещения.
