Толщина.Верх (Thickness.Top) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает ширину верхней стороны ограничивающего прямоугольника.

Примечание

Пример

Верх = Толщина2.Верх;
Полный пример кода 



Перем ТФ, Толщина2;

Процедура ПриОткрытии() Экспорт
	Верх = Толщина2.Верх;
	ТФ.Верхний.Текст = "Толщина2.Верх = " + Верх;
