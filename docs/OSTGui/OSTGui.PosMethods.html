Позиция (Pos) Методы

Методы

Абсолютно (At) Возвращает объект Позиция (Pos) с абсолютным значением из указанного целого значения.
Верх (Top) Возвращает объект Позиция (Pos), отслеживающий верхнюю (Игрек) координату указанного элемента.
Вычесть (Subtract) Вычитает из одного значения экземпляра класса Позиция (Pos) другое значение экземпляра класса Позиция (Pos), получая в результате экземпляр класса Позиция (Pos).
Игрек (Y) Возвращает объект Позиция (Pos), отслеживающий верхнюю (Игрек) позицию указанного элемента.
Икс (X) Возвращает объект Позиция (Pos), отслеживающий левую (Икс) позицию указанного элемента.
Лево (Left) Возвращает объект Позиция (Pos), отслеживающий левую (Икс) позицию указанного элемента.
Низ (Bottom) Возвращает объект Позиция (Pos), отслеживающий нижнюю (Игрек+Высота) координату указанного элемента.
Право (Right) Возвращает объект Позиция (Pos), отслеживающий правую (Икс+Ширина) координату указанного элемента.
Процент (Percent) Возвращает объект Позиция (Pos) на основе указанного процента.
Сложить (Summation) Суммирует два значения экземпляра класса Позиция (Pos) получая в результате экземпляр класса Позиция (Pos).
Центр (Center) Возвращает объект Позиция (Pos), который можно использовать для центрирования элемента.
ЯкорьКонец (AnchorEnd) Возвращает объект Позиция (Pos), привязанный к концу (правой стороне или нижней части) измерения, что полезно для выравнивания макета по правому краю или нижней части.
