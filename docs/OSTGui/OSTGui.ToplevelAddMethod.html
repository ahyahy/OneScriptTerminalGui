Верхний.Добавить (Toplevel.Add) Метод

Синтаксис

Добавить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Добавляет дочерний элемент к этому элементу.

Примечание

Пример

ТФ.Верхний.Добавить(Верхний1);
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Центр();
Верхний1.Игрек = ТФ.Позиция.Центр();
Верхний1.Ширина = ТФ.Величина.Процент(60);
Верхний1.Высота = ТФ.Величина.Абсолютно(1);
ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Верхний1.ЦветоваяСхема = ЦветоваяСхема1;
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
ТФ.Верхний.Добавить(Верхний1);

Верхний1.Текст = "ТФ.Верхний.Подэлементы.Количество = " + ТФ.Верхний.Подэлементы.Количество;

ТФ.Запуск();
