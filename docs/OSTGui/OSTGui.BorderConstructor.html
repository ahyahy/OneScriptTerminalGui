Граница (Border) Конструктор

Синтаксис

Граница()

Параметры

Описание

Создаёт новый экземпляр класса Граница (Border).

Примечание

Важно: Объект Граница (Border) нельзя переиспользовать. То есть созданную и назначенную границу одного элемента нельзя присваивать другому элементу. Правильно будет для каждого элемента создавать и присваивать уникальный объект Граница (Border).

Если вы создали объект Верхний (Toplevel) вложенный в объект Верхний (Toplevel), который соответствует значению свойства ТерминалФормыДляОдноСкрипта.Верхний (OneScriptTerminalGui.Top), то созданные в нем кнопки могут некорректно рисовать заданные программно границы.

Пример

Граница1 = ТФ.Граница();
Полный пример кода

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();
ТФ.Активировать();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(1);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(1);
Верхний1.Ширина = ТФ.Величина.Абсолютно(20);
Верхний1.Высота = ТФ.Величина.Абсолютно(5);
ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Верхний1.ЦветоваяСхема = ЦветоваяСхема1;
Верхний1.Текст = "Давным-давно в городке на берегу Средиземного моря жил старый столяр Джузеппе по прозванию Сизый Нос.";
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Верхний1.Граница = Граница1;
ТФ.Верхний.Добавить(Верхний1);

ТФ.Запуск();
