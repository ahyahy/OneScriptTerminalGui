АргументыСобытия.ЭтоAlt (EventArgs.IsAlt) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, была ли нажата клавиша Alt (реальная или синтезированная).

Примечание

Пример

АргументыСобытия = ТФ.АргументыСобытия;
ТФ.Верхний.Текст = "Кнопка1 сейчас в фокусе. Нажмите любую клавишу. С модификаторами или без.
|Кнопка1_КлавишаНажата = " + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
|Отправитель = " + АргументыСобытия.Отправитель + "
|Клавиша = " + АргументыСобытия.Клавиша + " (" + ТФ.Клавиши.ВСтроку(АргументыСобытия.Клавиша) + ")
|КлавишаСтрокой = " + АргументыСобытия.КлавишаСтрокой + "
|ЭтоAlt = " + АргументыСобытия.ЭтоAlt + "
|ЭтоCapslock = " + АргументыСобытия.ЭтоCapslock + "
|ЭтоCtrl = " + АргументыСобытия.ЭтоCtrl + "
|ЭтоNumlock = " + АргументыСобытия.ЭтоNumlock + "
|ЭтоScrolllock = " + АргументыСобытия.ЭтоScrolllock + "
|ЭтоShift = " + АргументыСобытия.ЭтоShift + "
|";
