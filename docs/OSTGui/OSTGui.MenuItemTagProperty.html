ЭлементМеню.Метка (MenuItem.Tag) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает объект, содержащий дополнительные данные.

Примечание

Пример

Структура1 = Новый Структура("Буратино","Деревянная кукла, наивная, добродушная и озорная.");
Структура1.Вставить("Карло", "Шарманщик, который вырезал Буратино из полена.");
Структура1.Вставить("Дуремар", "Продавец лечебных пиявок и друг Карабаса.");
Структура1.Вставить("Тортила", "Добрая черепаха, которая подарила Буратино золотой ключик.");
ЭлементМеню1.Метка = Структура1;
Полный пример кода 



Перем ТФ, ЭлементМеню1;

Процедура ПриОткрытии() Экспорт
	Метка = ЭлементМеню1.Метка;
	Текст = "
	|
	|
	|
	|
	|
	|
	|ЭлементМеню1.Метка = " + Метка + "
	|";
	ТФ.Верхний.Текст = Текст;
