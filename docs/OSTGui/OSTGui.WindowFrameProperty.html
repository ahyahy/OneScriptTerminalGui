Окно.Кадр (Window.Frame) Свойство

Использование

Чтение и запись.

Значение

Тип: Прямоугольник (Rect).

Возвращает или задает кадр для элемента. Кадр определяется относительно родителя.

Примечание

Пример

Окно1.Кадр = ТФ.Прямоугольник(1, 1, 50, 15);
Полный пример кода 



Перем Окно1;

Процедура ПриОткрытии() Экспорт
	Окно1.Текст = "Окно1.Кадр = " + Окно1.Кадр + "
	|Окно1.Кадр.Икс = " + Окно1.Кадр.Икс + "
	|Окно1.Кадр.Игрек = " + Окно1.Кадр.Игрек + "
	|Окно1.Кадр.Ширина = " + Окно1.Кадр.Ширина + "
	|Окно1.Кадр.Высота = " + Окно1.Кадр.Высота + "
	|";
