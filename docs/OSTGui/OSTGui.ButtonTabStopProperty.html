Кнопка.ТабФокус (Button.TabStop) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли пользователь перевести фокус на данный элемент при помощи клавиши Tab. Применимо только если свойство Фокусируемый CanFocus также установлено в Истина.

Примечание

Пример

Кнопка1.ТабФокус = Истина;
Полный пример кода 



Перем ТФ, Кнопка1, ТабФокус;

Процедура ПриОткрытии() Экспорт
	ТабФокус = Кнопка1.ТабФокус;
	ТФ.Верхний.Текст = "Кнопка1.ТабФокус = " + ТабФокус;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка("Кнопка1");
Кнопка1.Икс = ТФ.Позиция.Абсолютно(4);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(4);
Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Кнопка1.Ширина = ТФ.Величина.Процент(50);
Кнопка1.Высота = ТФ.Величина.Процент(18);
ТФ.Верхний.Добавить(Кнопка1);

Кнопка1.Фокусируемый = Истина;
Кнопка1.ТабФокус = Истина;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
