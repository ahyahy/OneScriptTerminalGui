ЦветоваяСхема.ГорячийНормальный (ColorScheme.HotNormal) Свойство

Использование

Чтение и запись.

Значение

Тип: Атрибут (Attribute).

Возвращает или задает цвет переднего плана и фона текста при выделении элемента.

Примечание

Пример

ГорячийНормальный = ЦветоваяСхема1.ГорячийНормальный;
Полный пример кода 



Перем ТФ, ЦветоваяСхема1;

Процедура ПриОткрытии() Экспорт
	ГорячийНормальный = ЦветоваяСхема1.ГорячийНормальный;
	ТФ.Верхний.Текст = "ЦветоваяСхема1.ГорячийНормальный = " + ГорячийНормальный;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
