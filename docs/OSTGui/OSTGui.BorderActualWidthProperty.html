Граница.АктуальнаяШирина (Border.ActualWidth) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает отображаемую ширину границы.

Примечание

Пример

АктуальнаяШирина = Верхний1.Граница.АктуальнаяШирина;

Перем ТФ, Верхний1, АктуальнаяШирина;

Процедура ПриОткрытии() Экспорт

	Верхний1.Текст = "Верхний1.Граница.АктуальнаяШирина = " + АктуальнаяШирина;
