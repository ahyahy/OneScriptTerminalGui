Таблица.ПолучитьВсеВыбранныеЯчейки (TableView.GetAllSelectedCells) Метод

Синтаксис

ПолучитьВсеВыбранныеЯчейки()

Возвращаемое значение

Тип: Массив.

Описание

Возвращает все ячейки в любом ВыбранныеРегионы (MultiSelectedRegions) (если МножественныйВыбор (MultiSelect) включен) и выбранную ячейку.

Примечание

Возвращается массив точек Точка (Point). Каждая точка в координате Икс содержит индекс колонки, в координате Игрек содержит индекс строки таблицы.

Пример

Массив = Таблица1.ПолучитьВсеВыбранныеЯчейки();
