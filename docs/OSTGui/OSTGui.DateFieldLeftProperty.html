ПолеДаты.Лево (DateField.Left) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси X левого края элемента.

Примечание

Пример

Лево = ПолеДаты1.Лево;
Полный пример кода 



Перем ТФ, Лево, ПолеДаты1;

Процедура ПриОткрытии() Экспорт
	Лево = ПолеДаты1.Лево;
	ТФ.Уведомление().Показать("Лево = " + Лево, -1);
