Величина (Dim) Методы

Методы

Абсолютно (Sized) Возвращает объект Величина (Dim) с абсолютным значением из указанного целого значения.
Высота (Height) Возвращает объект Величина (Dim), соответствующий высоте указанного элемента.
Вычесть (Subtract) Вычитает из одного значения экземпляра класса Величина (Dim) другое значение экземпляра класса Величина (Dim), получая в результате экземпляр класса Величина (Dim).
Заполнить (Fill) Возвращает объект Величина (Dim), который заполняет размерность, но оставляет указанный отступ.
Процент (Percent) Возвращает объект Величина (Dim) на основе указанного процента.
Сложить (Summation) Суммирует два значения экземпляра класса Величина (Dim) получая в результате экземпляр класса Величина (Dim).
Ширина (Width) Возвращает объект Величина (Dim), соответствующий ширине указанного элемента.
