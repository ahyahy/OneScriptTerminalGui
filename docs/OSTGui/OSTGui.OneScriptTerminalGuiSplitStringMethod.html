Синтаксис

РазобратьСтроку(<РазделяемаяСтрока>, <Разделитель>)

Параметры

РазделяемаяСтрока (обязательный)
Тип: Строка.
Строка, которую требуется разделить.
Разделитель (обязательный)
Тип: Строка.
Строка, которая будет служить разделителем.

Возвращаемое значение

Тип: Массив.

Массив подстрок.

Описание

Разделяет строку на массив подстрок.
