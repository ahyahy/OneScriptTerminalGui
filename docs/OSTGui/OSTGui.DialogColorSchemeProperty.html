Диалог.ЦветоваяСхема (Dialog.ColorScheme) Свойство

Использование

Чтение и запись.

Значение

Тип: ЦветоваяСхема (ColorScheme).

Возвращает или задает цветовую схему для элемента. Если она не определена, будет возвращена цветовая схема родителя.

Примечание

Пример

ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Диалог1.ЦветоваяСхема = ЦветоваяСхема1;
Полный пример кода 



Перем ТФ, ЦветоваяСхема, Диалог1;

Процедура ПриОткрытии() Экспорт
	ЦветоваяСхема = Диалог1.ЦветоваяСхема;
	ТФ.Уведомление().Показать("ЦветоваяСхема = " + ЦветоваяСхема, -1);
КонецПроцедуры
