ВыравниваниеТекста (TextAlignment) Перечисление

Описание

Указывает выравнивание текста.

Члены

Имя члена Описание Значение
Лево (Left) Текст выравнивается слева по кадру. 0
Подобранный (Justified) Подобранное выравнивание текста в кадре. 3
Право (Right) Текст выравнивается справа по кадру. 1
Центр (Centered) Текст выравнивается по центру кадра. 2
