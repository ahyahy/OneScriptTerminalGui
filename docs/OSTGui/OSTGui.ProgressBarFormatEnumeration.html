ФорматИндикатора (ProgressBarFormat) Перечисление

Описание

Указывает формат, который используется в индикаторе для визуального отображения.

Члены

Имя члена Описание Значение
Кадр (Framed) Отображение в рамке, на которой отображается только индикатор выполнения. 2
КадрПроценты (FramedPlusPercentage) Отображение в рамке с индикатором выполнения и процентным соотношением. 3
КадрПроцентыОтступы (FramedProgressPadded) Отображение в рамке с индикатором выполнения, помещенным в рамку, и процентным соотношением. 4
Простой (Simple) Простой индикатор без рамки. 0
ПростойПроценты (SimplePlusPercentage) Простой индикатор с процентным соотношением. 1
