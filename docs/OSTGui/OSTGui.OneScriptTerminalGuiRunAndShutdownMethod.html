ЗапуститьИЗавершить()

Возвращаемое значение

Описание

Запускает приложение и после выполнения возникших событий завершается работа приложения.

Примечание

Метод создан для упрощения тестирования библиотеки и выполняет завершение работы приложения после выполнения события ТерминалФормыДляОдноСкрипта.ПриОткрытии (OneScriptTerminalGui.NotifyNewRunState).

Метод не обеспечивает подавление сообщения об ошибке при использовании метода ТерминалФормыДляОдноСкрипта.Завершить (OneScriptTerminalGui.Shutdown).
