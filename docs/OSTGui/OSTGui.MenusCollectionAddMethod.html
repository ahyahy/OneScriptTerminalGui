КоллекцияПодменю.Добавить (MenusCollection.Add) Метод

Синтаксис

Добавить(<ПунктМеню>)

Параметры

ПунктМеню (обязательный)
Тип: ПунктМеню (MenuBarItem).

Возвращаемое значение

Описание

Добавляет ПунктМеню (MenuBarItem) в конец коллекции КоллекцияПодменю (MenusCollection).

Примечание

Пример

ПанельМеню1.ПодМеню.Добавить(ПунктМеню2);
Полный пример кода 
Копировать     Выделить всё


Перем Текст, ПанельМеню1, Окно1;

Процедура ПриОткрытии() Экспорт
	КоллекцияПодменю = ПанельМеню1.ПодМеню;
	Стр = "";
	Для А = 0 По КоллекцияПодменю.Количество - 1 Цикл
		Стр = Стр + "
		|Подменю.Заголовок = " + КоллекцияПодменю.ПунктМеню(А).Заголовок;
	КонецЦикла;
	Окно1.Текст = "Для ПанельМеню1 созданы подменю:
	|" + Стр;
КонецПроцедуры

Процедура Файл_Нажатие() Экспорт

ПанельМеню1.ПодМеню.Добавить(ПунктМеню2);

Окно1 = ТФ.Окно();

|";
