ДиалогСохранения (SaveDialog) Конструктор

Синтаксис

ДиалогСохранения(<Заголовок>, <Сообщение>, <Икс>, <Игрек>, <Ширина>, <Высота>, <ЯзыкНадписей>)

Параметры

Заголовок (необязательный)
Тип: Строка.
Заголовок для диалога сохранения. Значение по умолчанию - "Диалог сохранения".
Сообщение (необязательный)
Тип: Строка.
Сообщение для диалога сохранения. Значение по умолчанию - "Сообщение".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла диалога сохранения. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла диалога сохранения. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина диалога сохранения. Значение по умолчанию - 80.
Высота (необязательный)
Тип: Число.
Высота диалога сохранения. Значение по умолчанию - 20.
ЯзыкНадписей (необязательный)
Тип: ЯзыкНадписей (LabelLanguage).
Язык надписей на элементах. Значение по умолчанию - Русский (Russian).

Описание

Создаёт новый экземпляр класса ДиалогСохранения (SaveDialog).

Примечание

Пример

ДиалогСохранения1 = ТФ.ДиалогСохранения(, , , 5);

	АргументыСобытия = ТФ.АргументыСобытия;

		|Элемент = " + АргументыСобытия.Элемент + "

	Иначе
		Текст = "
		|РезультатДиалога = " + АргументыСобытия.РезультатДиалога;
	КонецЕсли;
	ТФ.Верхний.Текст = Текст;
	ДиалогСохранения = АргументыСобытия.Элемент;
