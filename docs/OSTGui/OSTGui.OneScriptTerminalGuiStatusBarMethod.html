ТерминалФормыДляОдноСкрипта.СтрокаСостояния (OneScriptTerminalGui.StatusBar) Метод

Синтаксис

СтрокаСостояния()

Возвращаемое значение

Тип: СтрокаСостояния (StatusBar).

Описание

Создаёт новый экземпляр класса СтрокаСостояния (StatusBar).

Примечание

Пример

СтрокаСостояния1 = ТФ.СтрокаСостояния();
Полный пример кода 



Перем ТФ, СтрокаСостояния1;

Процедура ПриОткрытии() Экспорт
	СтрокаСостояния = СтрокаСостояния1;
	ТФ.Верхний.Текст = "СтрокаСостояния1 = " + СтрокаСостояния;
КонецПроцедуры

Процедура ЭлементСтрокиСостояния1_Нажатие() Экспорт
	ТФ.Верхний.Текст = "Элемент1 Нажатие " + ТекущаяУниверсальнаяДатаВМиллисекундах();
КонецПроцедуры

Процедура ЭлементСтрокиСостояния2_Нажатие() Экспорт
	ТФ.Верхний.Текст = "Элемент2 Нажатие " + ТекущаяУниверсальнаяДатаВМиллисекундах();
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

СтрокаСостояния1 = ТФ.СтрокаСостояния();
ТФ.Верхний.Добавить(СтрокаСостояния1);

ЭлементСтрокиСостояния1 = ТФ.ЭлементСтрокиСостояния(ТФ.Клавиши.s_low, "Элемент1");
// Нажатие клавиши s для ЭлементСтрокиСостояния1 равносилен клику по нему мышью.
// Только следите за раскладкой (s сработает только при англ. раскладке)
ЭлементСтрокиСостояния1.Нажатие = ТФ.Действие(ЭтотОбъект, "ЭлементСтрокиСостояния1_Нажатие");
СтрокаСостояния1.Элементы.Добавить(ЭлементСтрокиСостояния1);

ЭлементСтрокиСостояния2 = ТФ.ЭлементСтрокиСостояния(ТФ.Клавиши.d_low, "Элемент2");
ЭлементСтрокиСостояния2.Нажатие = ТФ.Действие(ЭтотОбъект, "ЭлементСтрокиСостояния2_Нажатие");
СтрокаСостояния1.Элементы.Добавить(ЭлементСтрокиСостояния2);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
