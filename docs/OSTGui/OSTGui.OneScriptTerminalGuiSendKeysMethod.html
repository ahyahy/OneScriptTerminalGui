ТерминалФормыДляОдноСкрипта.ОтправитьКлавиши (OneScriptTerminalGui.SendKeys) Метод

Синтаксис

ОтправитьКлавиши(<Символ>, <Клавиша>, <МодификаторShift>, <МодификаторAlt>, <МодификаторCtrl>)

Параметры

Символ (обязательный)
Тип: Строка.
Отправляемый символ.
Клавиша (обязательный)
Тип: КлавишиКонсоли (ConsoleKey); Число.
Клавиша, с которой отправляется символ, или числовое значении клавиши.
МодификаторShift (обязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Shift.
МодификаторAlt (обязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Alt.
МодификаторCtrl (обязательный)
Тип: Булево.
Указывает нужно ли использовать модификатор Ctrl.

Возвращаемое значение

Описание

Отправляет сообщения без использования клавиатуры.

Примечание

Если в перечислении КлавишиКонсоли (ConsoleKey) (параметр Клавиша), нет нужного значения, тогда можно в параметре Клавиша задать числовое значение для клавиши клавиатуры. Обычно это необходимо при раскладке клавиатуры, отличной от английской.

Пример

ТФ.ОтправитьКлавиши("к", 1050, Ложь, Ложь, Ложь);
