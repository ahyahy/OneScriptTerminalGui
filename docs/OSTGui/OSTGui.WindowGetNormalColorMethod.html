Окно.ЦветОбычного (Window.GetNormalColor) Метод

Синтаксис

ЦветОбычного()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветОбычного = Окно0.ЦветОбычного();
Полный пример кода 



Перем Окно0, ЦветОбычного;

Процедура ПриОткрытии() Экспорт
	ЦветОбычного = Окно0.ЦветОбычного();
	Окно0.Текст = "Окно0.ЦветОбычного() = " + ЦветОбычного + "
	|ОсновнойЦвет = " + ЦветОбычного.ОсновнойЦвет + "
	|ЦветФона = " + ЦветОбычного.ЦветФона + "
	|";
