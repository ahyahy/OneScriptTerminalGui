Окно.Удалить (Window.Remove) Метод

Синтаксис

Удалить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Удаляет подэлемент, добавленный методом Добавить (Add).

Примечание

Пример

Окно0.Удалить(Окно3);
Полный пример кода 
Копировать     Выделить всё


Перем Окно0, Количество, Окно3;

Процедура ПриОткрытии() Экспорт
	Окно0.Удалить(Окно3);
	Количество = Окно0.Подэлементы.Количество;
	Окно0.Текст = "Окно0.Подэлементы.Количество = " + Количество;
КонецПроцедуры
