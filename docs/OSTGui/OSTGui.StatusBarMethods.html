СтрокаСостояния (StatusBar) Методы

Методы

ВерхнийРодитель (GetTopSuperView) (унаследовано от Элемент (View)) Возвращает самого верхнего родителя.
ВставитьПоИндексу (AddItemAt) Вставляет элемент в панель состояния по указанному индексу.
ВСтроку (ToString) Возвращает строковое представление данного объекта.
ТочкаНаЭлементе (ScreenToView) (унаследовано от Элемент (View)) Вычисляет местоположение указанной точки на терминале в координаты на элементе.
Удалить (RemoveItem) Удаляет элемент из панели состояния по указанному индексу.
УдалитьВсе (RemoveAll) (унаследовано от Элемент (View)) Удаляет все подэлементы, добавленные методом Добавить (Add).
ЦветВыделенного (GetHotNormalColor) (унаследовано от Элемент (View)) Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.
ЦветОбычного (GetNormalColor) (унаследовано от Элемент (View)) Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.
ЦветФокуса (GetFocusColor) (унаследовано от Элемент (View)) Возвращает текущую цветовую схему для текущего фокуса.
