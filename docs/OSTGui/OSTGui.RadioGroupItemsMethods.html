Методы

Добавить (Add) Добавляет метку в конец коллекции.
Очистить (Clear) Удаляет все элементы из коллекции.
Получить (Get) Возвращает значение по указанному индексу из коллекции.
Удалить (Remove) Удаляет метку из коллекции.
