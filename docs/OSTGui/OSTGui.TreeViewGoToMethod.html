Дерево.Перейти (TreeView.GoTo) Метод

Синтаксис

Перейти(<Узел>)

Параметры

Узел (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Описание

Изменяет выбранный узел на переданный в параметре и прокручивает страницу, чтобы он был виден. Не действует, если узел, переданный в параметре, не отображается в дереве (например, его родительские элементы свернуты).

Примечание

Пример

Дерево1.Перейти(Узел6);
Полный пример кода 



Перем ТФ, Дерево1, Узел6;

Процедура ПриОткрытии() Экспорт
	Дерево1.Перейти(Узел6);
	Выбранный = Дерево1.Выбранный;
	ТФ.Уведомление().Показать("Выбранный = " + Выбранный.Текст, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Дерево1 = ТФ.Дерево(, , 40, 15);
Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Дерево1.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Черный, ТФ.Цвет.Серый);
Дерево1.ЦветоваяСхема.Фокус = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.ЯркоГолубой);

Узел1 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел1"));
Узел2 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел2"));
Узел3 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел3"));
