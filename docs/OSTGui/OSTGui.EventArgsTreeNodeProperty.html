АргументыСобытия.УзелДерева (EventArgs.TreeNode) Свойство

Использование

Только чтение.

Значение

Тип: УзелДерева (TreeNode).

Процедура Активирован() Экспорт

	|";
	ТФ.Уведомление().Показать(Текст, -1);
