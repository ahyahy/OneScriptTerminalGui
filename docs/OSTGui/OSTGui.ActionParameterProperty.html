Действие.Параметр (Action.Parameter) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает объект, заданный при установке обработчика события, для передачи в обработчик события.

Примечание

Пример

Структура1 = Новый Структура("Буратино","Деревянная кукла, наивная, добродушная и озорная.");
Структура1.Вставить("Карло", "Шарманщик, который вырезал Буратино из полена.");
Структура1.Вставить("Дуремар", "Продавец лечебных пиявок и друг Карабаса.");
Структура1.Вставить("Тортила", "Добрая черепаха, которая подарила Буратино золотой ключик.");

Действие1 = ТФ.Действие();
Действие1.ИмяМетода = "Кнопка1_Нажатие";
Действие1.Параметр = Структура1;
Действие1.Сценарий = ЭтотОбъект;
Кнопка1.Нажатие = Действие1;
Полный пример кода 



Перем ТФ, Счетчик, Действие1;

Процедура ПриОткрытии() Экспорт
	Кнопка1_Нажатие();
КонецПроцедуры

Процедура Кнопка1_Нажатие() Экспорт
	Счетчик = Счетчик + 1;
	АргументыСобытия1 = ТФ.АргументыСобытия;
	Параметр = АргументыСобытия1.Параметр;
	ТФ.Верхний.Текст = "Кнопа Нажата " + Счетчик + " раз.
	|Действие1.ИмяМетода = " + Действие1.ИмяМетода + "
	|Действие1.Параметр = " + Действие1.Параметр + "
	|Действие1.Сценарий = " + Действие1.Сценарий;
