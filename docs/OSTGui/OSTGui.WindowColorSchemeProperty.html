Окно.ЦветоваяСхема (Window.ColorScheme) Свойство

Использование

Чтение и запись.

Значение

Тип: ЦветоваяСхема (ColorScheme).

Возвращает или задает цветовую схему для этого элемента. Если она не определена, будет возвращена цветовая схема родителя.

Примечание

Пример

ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Окно1.ЦветоваяСхема = ЦветоваяСхема1;
