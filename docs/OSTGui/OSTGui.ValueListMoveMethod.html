СписокЗначений.Сдвинуть (ValueList.Move) Метод

Синтаксис

Сдвинуть(<Элемент>, <Сдвиг>)

Параметры

Элемент (обязательный)
Тип: ScriptEngine.HostedScript.Library.ValueList.ValueListItem.
Сдвигаемый элемент из списка значений.
Сдвиг (обязательный)
Тип: Число.
Количество позиций, на которое сдвигается элемент. Если значение положительное - сдвиг вниз, иначе вверх.

Возвращаемое значение

Описание

Сдвигает элемент на указанное количество позиций.

Примечание

Пример

ЭлементСписка = СписокЗначений1.Вставить(0, 150, "Сто пятьдесят");
СписокЗначений1.Сдвинуть(ЭлементСписка, 3);
Индекс = СписокЗначений1.Индекс(ЭлементСписка);
