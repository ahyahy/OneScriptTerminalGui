Размер.Добавить (Size.Add) Метод

Синтаксис

Добавить(<Размер1>, <Размер2>)

Параметры

Размер1 (обязательный)
Тип: Размер (Size).
Первая структура размера для добавления.
Размер2 (обязательный)
Тип: Размер (Size).
Вторая структура размера для добавления.

Возвращаемое значение

Тип: Размер (Size).

Структура Размер (Size).

Описание

Добавляет ширину и высоту структуры одного размера к ширине и высоте структуры другого размера.

Примечание

Пример

Добавить = ТФ.Размер().Добавить(Размер1, Размер2);
