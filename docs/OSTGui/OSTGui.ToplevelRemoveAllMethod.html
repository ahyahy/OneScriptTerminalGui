Верхний.УдалитьВсе (Toplevel.RemoveAll) Метод

Синтаксис

УдалитьВсе()

Возвращаемое значение

Описание

Удаляет все подэлементы, добавленные методом Добавить (Add).

Примечание

Пример

ТФ.Верхний.УдалитьВсе();
Полный пример кода 



Перем ТФ;

Процедура Кнопка1_Нажатие() Экспорт
	ТФ.Верхний.УдалитьВсе();
	ТФ.Верхний.Текст = "Осталось подэлементов = " + ТФ.Верхний.Подэлементы.Количество;
