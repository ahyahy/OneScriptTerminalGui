КоллекцияПодэлементов.Количество (SubviewCollection.Count) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает общее количество объектов в коллекции.

Примечание

Пример

Количество = Окно0.Подэлементы.Количество;
Полный пример кода 
Копировать     Выделить всё


Перем Окно0, Количество;

Процедура ПриОткрытии() Экспорт
	Стр = "Подэлементы для Окно0:
	|";
	Стр = Стр + "====================
	|Цикл Для ... По ... Цикл
	|";
	Подэлементы1 = Окно0.Подэлементы;
	Для А = 0 По Подэлементы1.Количество - 1 Цикл
		Стр = Стр + "Элемент(" + А + ") = " + Подэлементы1.Элемент(А).Заголовок + Символы.ПС;
	КонецЦикла;
	Стр = Стр + "====================
	|Цикл Для Каждого ... Из ... Цикл
	|";
	Для Каждого Эл Из Подэлементы1 Цикл
		Стр = Стр + "Эл = " + Эл.Заголовок + Символы.ПС;
	КонецЦикла;
	Окно0.Текст = Стр;
	
	Количество = Окно0.Подэлементы.Количество;
