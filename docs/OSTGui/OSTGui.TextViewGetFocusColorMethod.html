Текстовый.ЦветФокуса (TextView.GetFocusColor) Метод

Синтаксис

ЦветФокуса()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему для текущего фокуса.

Примечание

Пример

Текстовый1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Текстовый1.ЦветоваяСхема.Фокус = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Текстовый1, ЦветФокуса;

Процедура ПриОткрытии() Экспорт
	ЦветФокуса = Текстовый1.ЦветФокуса();
	Стр = "ЦветФокуса() = " + ЦветФокуса + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветФокуса.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветФокуса.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
