Позиция.Вычесть (Pos.Subtract) Метод

Синтаксис

Вычесть(<Уменьшаемое>, <Вычитаемое>)

Параметры

Уменьшаемое (обязательный)
Тип: Позиция (Pos).
Вычитаемое (обязательный)
Тип: Позиция (Pos).

Возвращаемое значение

Тип: Позиция (Pos).

Описание

Вычитает из одного значения экземпляра класса Позиция (Pos) другое значение экземпляра класса Позиция (Pos), получая в результате экземпляр класса Позиция (Pos).

Примечание

Пример

Позиция = ТФ.Позиция.Вычесть(ТФ.Позиция.Верх(Кнопка1), ТФ.Позиция.Абсолютно(2));
Полный пример кода 



Перем ТФ, Кнопка1, Кнопка2;

Процедура ПриОткрытии() Экспорт
	Позиция = ТФ.Позиция.Вычесть(ТФ.Позиция.Верх(Кнопка1), ТФ.Позиция.Абсолютно(2));
	Кнопка2.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
	Кнопка2.Игрек = Позиция;
	
	Игрек1 = Кнопка1.Кадр.Игрек;
	Игрек2 = Кнопка2.Кадр.Игрек;
	ТФ.Верхний.Текст = "
	|Позиция = " + Позиция + "
	|";
