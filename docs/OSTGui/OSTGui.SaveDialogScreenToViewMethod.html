ДиалогСохранения.ТочкаНаЭлементе (SaveDialog.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Координата икс.
Игрек (обязательный)
Тип: Число.
Координата игрек.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Точка = ДиалогСохранения1.ТочкаНаЭлементе(45, 10);
Полный пример кода 



Перем ТФ, ДиалогСохранения1;

Процедура ПриОткрытии() Экспорт
	Точка = ДиалогСохранения1.ТочкаНаЭлементе(45, 10);
	Стр = "Левый верхний угол ДиалогСохранения1 имеет X = 40 и Y = 5 
	|Для точки на терминале X = 45 и Y = 10 
	|ДиалогСохранения1.ТочкаНаЭлементе(45, 10).Икс = " + Точка.Икс + "
	|ДиалогСохранения1.ТочкаНаЭлементе(45, 10).Игрек = " + Точка.Игрек;
	
	ТФ.Уведомление().Показать(Стр, -1);
