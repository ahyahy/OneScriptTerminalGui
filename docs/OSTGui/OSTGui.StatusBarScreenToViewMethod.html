СтрокаСостояния.ТочкаНаЭлементе (StatusBar.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Игрек (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Игрек = СтрокаСостояния1.Кадр.Игрек;
Точка = СтрокаСостояния1.ТочкаНаЭлементе(5, Игрек);
Полный пример кода 



Перем ТФ, СтрокаСостояния1;

Процедура ПриОткрытии() Экспорт
	Игрек = СтрокаСостояния1.Кадр.Игрек;
	Точка = СтрокаСостояния1.ТочкаНаЭлементе(5, Игрек);
	Стр = "Левый верхний угол СтрокаСостояния1 имеет X = 0 и Y = " + Игрек + " 
	|Для точки на терминале X = 5 и Y = " + Игрек + " 
	|СтрокаСостояния1.ТочкаНаЭлементе(5, " + Игрек + ").Икс = " + Точка.Икс + "
	|ПанельМеню1.ТочкаНаЭлементе(5, " + Игрек + ").Игрек = " + Точка.Игрек;
	Икс = Точка.Икс;
	Игрек = Точка.Игрек;
	ТФ.Верхний.Текст = ТФ.Верхний.Текст + Стр;
