ТерминалФормыДляОдноСкрипта.Запуск (OneScriptTerminalGui.Run) Метод

Синтаксис

Запуск()

Возвращаемое значение

Описание

Запускает приложение.

Примечание

Метод обеспечивает подавление сообщения об ошибке при использовании метода ТерминалФормыДляОдноСкрипта.Завершить (OneScriptTerminalGui.Shutdown).

Поэтому в некоторых тестах, где необходимо в обработчике события сценария прервать по условию работу приложения, лучше использовать этот метод, вместо метода ТерминалФормыДляОдноСкрипта.ЗапуститьИЗавершить (OneScriptTerminalGui.RunAndShutdown) Метод.

Пример

ТФ.Запуск();
