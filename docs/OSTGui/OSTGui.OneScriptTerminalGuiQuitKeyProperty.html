ТерминалФормыДляОдноСкрипта.КлавишаВыхода (OneScriptTerminalGui.QuitKey) Свойство

Использование

Чтение и запись.

Значение

Тип: Клавиши (Keys); Число.

Возвращает или задает клавишу для выхода из приложения.

Примечание

Клавишу q не используйте. Она зарезервирована и не сработает.

Пример

ТФ.КлавишаВыхода = ТФ.Клавиши.a_low;
