ДиалогОткрытия.ЦветВыделенного (OpenDialog.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветВыделенного = ДиалогОткрытия1.ЦветВыделенного();
Полный пример кода 



Перем ТФ, ДиалогОткрытия1;

Процедура ПриОткрытии() Экспорт
	ЦветВыделенного = ДиалогОткрытия1.ЦветВыделенного();
	Стр = "ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветВыделенного.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветВыделенного.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
