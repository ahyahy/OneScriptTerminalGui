ДиалогСохранения.Игрек (SaveDialog.Y) Свойство

Использование

Чтение и запись.

Значение

Тип: Позиция (Pos).

Возвращает или задает объект Позиция (Pos), соответствующий координате по оси Y.

Примечание

Пример

ДиалогСохранения1.Игрек = 6;
Полный пример кода 



Перем ТФ, ДиалогСохранения1;

Процедура ПриОткрытии() Экспорт
	ДиалогСохранения1.Игрек = 6;
	Игрек = ДиалогСохранения1.Кадр.Верх;
	ТФ.Уведомление().Показать("Игрек = " + Игрек, -1);
