Уведомление.Интервал (Balloons.Interval) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает время показа в миллисекундах окна уведомления.

Примечание

Пример

Уведомление1.Интервал = 5000;
Интервал = Уведомление1.Интервал;
Полный пример кода 



Перем ТФ, Уведомление1, Интервал;

Процедура ПриОткрытии() Экспорт
	Интервал = Уведомление1.Интервал;
	Уведомление1.Показать("Интервал = " + Интервал, Интервал);
