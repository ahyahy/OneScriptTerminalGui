ЭлементыПунктаМеню (MenuBarItemChildren) Методы

Методы

Добавить (Add) Добавляет ЭлементМеню (MenuItem) в конец коллекции.
Очистить (Clear) Удаляет все элементы из коллекции.
Удалить (Remove) Удаляет ЭлементМеню (MenuItem) из коллекции ЭлементыПунктаМеню (MenuBarItemChildren).
ЭлементПунктаМеню (ItemMenuBarItem) Возвращает ЭлементМеню (MenuItem) с указанным индексом в коллекции.
