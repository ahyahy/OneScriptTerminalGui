Верхний.ТочкаНаЭлементе (Toplevel.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Игрек (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Точка = Верхний1.ТочкаНаЭлементе(45, 10);
Полный пример кода 



Перем ТФ, Верхний1;

Процедура ПриОткрытии() Экспорт
	Точка = Верхний1.ТочкаНаЭлементе(45, 10);
	Стр = "Левый верхний угол Верхний1 имеет X = 40 и Y = 5 
	|Для точки на терминале X = 45 и Y = 10 
	|Верхний1.ТочкаНаЭлементе(45, 10).Икс = " + Точка.Икс + "
	|Верхний1.ТочкаНаЭлементе(45, 10).Игрек = " + Точка.Игрек;
	ТФ.Верхний.Текст = Стр;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1.Текст = "Верхний1";
Верхний1.Икс = ТФ.Позиция.Абсолютно(40);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(5);
Верхний1.Ширина = ТФ.Величина.Абсолютно(70);
Верхний1.Высота = ТФ.Величина.Абсолютно(20);
Граница1 = ТФ.Граница();

Верхний1.Граница = Граница1;
