Верхний.Подэлементы (Toplevel.Subviews) Свойство

Использование

Только чтение.

Значение

Тип: КоллекцияПодэлементов (SubviewCollection).

Возвращает коллекцию подэлементов, содержащую дочерние элементы этого элемента.

Примечание

Пример

Стр = "";
Подэлементы1 = ТФ.Верхний.Подэлементы;
Для Каждого Эл Из Подэлементы1 Цикл
	Стр = Стр + Эл.Текст + Символы.ПС;
	Стр = Стр + "========================" + Символы.ПС;
КонецЦикла;
Полный пример кода 



Перем ТФ, Верхний1, Окно1;

Процедура ПриОткрытии() Экспорт
	Верхний1.Текст = "Верхний1.Кадр = " + Верхний1.Кадр + "
	|Верхний1.Кадр.Икс = " + Верхний1.Кадр.Икс + "
	|Верхний1.Кадр.Игрек = " + Верхний1.Кадр.Игрек + "
	|Верхний1.Кадр.Ширина = " + Верхний1.Кадр.Ширина + "
	|Верхний1.Кадр.Высота = " + Верхний1.Кадр.Высота;
	
	Стр = "";
	Подэлементы1 = ТФ.Верхний.Подэлементы;
	Для Каждого Эл Из Подэлементы1 Цикл
		Стр = Стр + Эл.Текст + Символы.ПС;
		Стр = Стр + "========================" + Символы.ПС;
	КонецЦикла;
	Окно1.Текст = Стр;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();
ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(2);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(2);
Верхний1.Ширина = ТФ.Величина.Абсолютно(80);
Верхний1.Высота = ТФ.Величина.Абсолютно(5);
ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Верхний1.ЦветоваяСхема = ЦветоваяСхема1;
Верхний1.Текст = "Давным-давно в городке на берегу Средиземного моря жил старый столяр Джузеппе по прозванию Сизый Нос.";
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
ТФ.Верхний.Добавить(Верхний1);

Верхний2 = ТФ.Верхний(Верхний1.Кадр.Икс, Верхний1.Кадр.Игрек, Верхний1.Кадр.Ширина, Верхний1.Кадр.Высота);
ТФ.Верхний.Добавить(Верхний2);

Верхний2.Ниже(Верхний1, 1);
Верхний2.Текст = "– Неплохая вещь, – сказал сам себе Джузеппе, – можно смастерить из него что-нибудь вроде ножки для стола…";

Граница2 = ТФ.Граница();
Граница2.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний2.Граница = Граница2;

Окно1 = ТФ.Окно();
Окно1.Заголовок = "Окно1 Исходный текст всех элементов до запуска";
Окно1.Текст = "Когда я был маленький – очень, очень давно, – я читал одну книжку: 
|она называлась «Пиноккио, или Похождения деревянной куклы» 
|(деревянная кукла по-итальянски – буратино).";
ТФ.Верхний.Добавить(Окно1);

Окно1.Ниже(Верхний2, 1);

ТФ.Запуск();
