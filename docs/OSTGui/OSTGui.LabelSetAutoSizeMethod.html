Надпись.УстановитьАвтоРазмер (Label.SetAutoSize) Метод

Синтаксис

УстановитьАвтоРазмер()

Возвращаемое значение

Описание

Устанавливает размер элемента, чтобы соответствовать всему тексту.

Примечание

Пример

Надпись1.УстановитьАвтоРазмер();
Полный пример кода 



Перем ТФ, Надпись1;

Процедура ПриОткрытии() Экспорт
	Надпись1.УстановитьАвтоРазмер();
