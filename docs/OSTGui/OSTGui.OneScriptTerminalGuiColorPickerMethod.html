ТерминалФормыДляОдноСкрипта.ВыборЦвета (OneScriptTerminalGui.ColorPicker) Метод

Синтаксис

ВыборЦвета(<Заголовок>, <Икс>, <Игрек>)

Параметры

Заголовок (необязательный)
Тип: Строка.
Заголовок для выбора цвета. Значение по умолчанию - "Выбор цвета".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла выбора цвета. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла выбора цвета. Значение по умолчанию - 1.

Возвращаемое значение

Тип: ВыборЦвета (ColorPicker).

Описание

Создаёт новый экземпляр класса ВыборЦвета (ColorPicker).

Примечание
