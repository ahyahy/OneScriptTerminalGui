СписокЗначений.НайтиПоЗначению (ValueList.FindByValue) Метод

Синтаксис

НайтиПоЗначению(<Значение>)

Параметры

Значение (обязательный)
Тип: Произвольный.
Искомое значение элемента из списка значений.

Возвращаемое значение

Тип: ScriptEngine.HostedScript.Library.ValueList.ValueListItem.

Описание

Возврашает элемент списка значений, найденный по его значению.

Примечание

Пример

ЭлементСписка = СписокЗначений1.НайтиПоЗначению("Вера Павловна - сестра Дяди Фёдора.");
Полный пример кода

Процедура ПриОткрытии() Экспорт

	ТФ.Уведомление().Показать("ЭлементСписка = " + ЭлементСписка.Представление, -1);
