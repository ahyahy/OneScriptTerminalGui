Величина.Высота (Dim.Height) Метод

Синтаксис

Высота(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Тип: Величина (Dim).

Описание

Возвращает объект Величина (Dim), соответствующий высоте указанного элемента.

Примечание

Пример

Окно2.Высота = ТФ.Величина.Высота(Окно1);
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Окно1 = ТФ.Окно();
Окно1.Заголовок = "Окно1";
Окно1.Икс = ТФ.Позиция.Абсолютно(1);
Окно1.Игрек = ТФ.Позиция.Абсолютно(1);
Окно1.Ширина = ТФ.Величина.Абсолютно(65);
Окно1.Высота = ТФ.Величина.Абсолютно(5);
ТФ.Верхний.Добавить(Окно1);

Окно2 = ТФ.Окно();
Окно2.Заголовок = "Окно2";
Окно2.Текст = "Окно2.Высота = ТФ.Величина.Высота(Окно1);";
Окно2.Ниже(Окно1, 1);
Окно2.Икс = Окно1.Икс;
Окно2.Ширина = ТФ.Величина.Ширина(Окно1);
Окно2.Высота = ТФ.Величина.Высота(Окно1);
ТФ.Верхний.Добавить(Окно2);

ТФ.Запуск();
