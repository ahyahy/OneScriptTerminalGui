ДиалогСохранения.ПолучитьСочетаниеКлавиш (SaveDialog.GetShortcut) Метод

Синтаксис

ПолучитьСочетаниеКлавиш()

Возвращаемое значение

Тип: СписокЗначений.

Список значений из пространства имен односкрипта.

Описание

Возвращает сочетания клавиш, назначенные для элемента.

Примечание

Пример

Объект = ТФ.АргументыСобытия.Параметр;
Список = Объект.ПолучитьСочетаниеКлавиш();
Стр = "";
Для Каждого Эл Из Список Цикл
	Стр = Стр + "
	| " + Эл.Значение + " (" + Эл.Представление + ") " + ТекущаяУниверсальнаяДатаВМиллисекундах();
КонецЦикла;
