Кнопка.УстановитьАвтоРазмер (Button.SetAutoSize) Метод

Синтаксис

УстановитьАвтоРазмер()

Возвращаемое значение

Описание

Устанавливает размер элемента, чтобы соответствовать всему тексту.

Примечание

Пример

Кнопка1.УстановитьАвтоРазмер();

Перем ТФ, Кнопка1;

Процедура ПриОткрытии() Экспорт
