КоллекцияПодэлементов.Элемент (SubviewCollection.Item) Метод

Синтаксис

Элемент(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.
Индекс элемента управления в коллекции.

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает элемент с указанным индексом в коллекции.

Примечание

Пример

Элемент = Окно0.Подэлементы.Элемент(0);
