АргументыСобытия.Размер (EventArgs.Size) Свойство

Использование

Только чтение.

Значение

Тип: Размер (Size).

Возвращает экземпляр класса Размер (Size), предсталяющий новый размер терминала.

Примечание

Пример

АргументыСобытия = ТФ.АргументыСобытия;
ТФ.Верхний.Текст = "ТФ_РазмерИзменен = " + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
|Отправитель = " + АргументыСобытия.Отправитель + "
|Размер = " + АргументыСобытия.Размер + "
|Размер.Ширина = " + АргументыСобытия.Размер.Ширина + "
|Размер.Высота = " + АргументыСобытия.Размер.Высота + "
|";
