Дерево.Кадр (TreeView.Frame) Свойство

Использование

Чтение и запись.

Значение

Тип: Прямоугольник (Rect).

Возвращает или задает кадр для элемента. Кадр определяется относительно родителя.

Примечание

Пример

Кадр = Дерево1.Кадр;
Полный пример кода 



Перем ТФ, Кадр, Дерево1;

Процедура ПриОткрытии() Экспорт
	Кадр = Дерево1.Кадр;
	ТФ.Уведомление().Показать("Кадр = " + Кадр, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
