Позиция.Игрек (Pos.Y) Метод

Синтаксис

Игрек(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Элемент (View).

Возвращаемое значение

Тип: Позиция (Pos).

Описание

Возвращает объект Позиция (Pos), отслеживающий верхнюю (Игрек) позицию указанного элемента.

Примечание

Пример

Позиция = ТФ.Позиция.Игрек(Кнопка1);
Полный пример кода 



Перем ТФ, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Позиция = ТФ.Позиция.Игрек(Кнопка1);
	
	ТФ.Верхний.Текст = "
	|Позиция = " + Позиция + "
	|";
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка("Кнопка1");
Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(10);
Кнопка1.Граница = ТФ.Граница();
Кнопка1.Граница.СтильГраницы = ТФ.СтильГраницы.Закругленная;
ТФ.Верхний.Добавить(Кнопка1);

Кнопка2 = ТФ.Кнопка("Кнопка2");
Кнопка2.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка2.Игрек = ТФ.Позиция.Абсолютно(13);
Кнопка2.Граница = ТФ.Граница();
Кнопка2.Граница.СтильГраницы = ТФ.СтильГраницы.Закругленная;
ТФ.Верхний.Добавить(Кнопка2);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
