Элемент (View) Методы

Методы

ВерхнийРодитель (GetTopSuperView) Возвращает самого верхнего родителя.
ВСтроку (ToString) Возвращает строковое представление данного объекта.
Выше (PlaceTop) Смещает выше этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.
ВышеЛевее (PlaceTopLeft) Смещает выше и левее этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
ВышеПравее (PlaceTopRight) Смещает выше и правее этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
Добавить (Add) Добавляет дочерний элемент к этому элементу.
ДобавитьСочетаниеКлавиш (AddShortcut) Добавляет сочетание клавиш для элемента. Сочетание клавиш можно использовать в качестве глобального ярлыка для вызова действия, если действие указано.
Заполнить (Fill) Заполняет размерность объекта, но оставляет указанные отступы.
Левее (PlaceLeft) Смещает левее этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.
ЛевееВыше (PlaceLeftTop) Смещает левее и выше этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
ЛевееНиже (PlaceLeftBottom) Смещает левее и ниже этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
НаЗаднийПлан (SendSubviewToBack) Перемещает подэлемент в системе компоновки на задний план.
НаПереднийПлан (BringSubviewToFront) Перемещает подэлемент в системе компоновки на передний план.
НаШагВперед (BringSubviewForward) Перемещает подэлемент в системе компоновки на один шаг вперед.
НаШагНазад (SendSubviewBackwards) Перемещает подэлемент в системе компоновки на один шаг в обратном направлении.
Ниже (PlaceBottom) Смещает ниже этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.
НижеЛевее (PlaceBottomLeft) Смещает ниже и левее этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
НижеПравее (PlaceBottomRight) Смещает ниже и правее этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
ПолучитьАвтоРазмер (GetAutoSize) Возвращает размер, чтобы соответствовать всему тексту, если включено свойство АвтоРазмер (AutoSize).
ПолучитьСочетаниеКлавиш (GetShortcut) Возвращает сочетания клавиш, назначенные для элемента.
Правее (PlaceRight) Смещает правее этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.
ПравееВыше (PlaceRightTop) Смещает правее и выше этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
ПравееНиже (PlaceRightBottom) Смещает правее и ниже этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.
ТочкаНаЭлементе (ScreenToView) Вычисляет местоположение указанной точки на терминале в координаты на элементе.
Удалить (Remove) Удаляет подэлемент, добавленный методом Добавить (Add).
УдалитьВсе (RemoveAll) Удаляет все подэлементы, добавленные методом Добавить (Add).
УдалитьСочетаниеКлавиш (RemoveShortcut) Удаляет сочетание клавиш для элемента.
УстановитьАвтоРазмер (SetAutoSize) Устанавливает размер элемента, чтобы соответствовать всему тексту.
УстановитьФокус (SetFocus) Устанавливает фокус.
ЦветВыделенного (GetHotNormalColor) Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.
ЦветОбычного (GetNormalColor) Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.
ЦветФокуса (GetFocusColor) Возвращает текущую цветовую схему для текущего фокуса.
Центр (Center) Центрирует элемент управления внутри своего родителя.
