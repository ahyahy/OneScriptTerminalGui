Надпись.ЦветОбычного (Label.GetNormalColor) Метод

Синтаксис

ЦветОбычного()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

Надпись1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Надпись1.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Надпись1, ЦветОбычного;

Процедура ПриОткрытии() Экспорт
	ЦветОбычного = Надпись1.ЦветОбычного();
	Стр = "ЦветОбычного() = " + ЦветОбычного + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветОбычного.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветОбычного.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
