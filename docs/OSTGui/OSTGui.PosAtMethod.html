Позиция.Абсолютно (Pos.At) Метод

Синтаксис

Абсолютно(<Координата>)

Параметры

Координата (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Позиция (Pos).

Описание

Возвращает объект Позиция (Pos) с абсолютным значением из указанного целого значения.

Примечание

До применения этого метода установите свойство элемента СтильКомпоновки (LayoutStyle) в значение Абсолютно (Absolute).

Пример

Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Полный пример кода 



Перем ТФ, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Позиция = Кнопка1.Икс;
	Икс = Кнопка1.Кадр.Икс;
	ТФ.Верхний.Текст = 
	"Кнопка1.Икс = " + Позиция + "
	|Кнопка1.Кадр.Икс = " + Икс + "
	|";
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка("Кнопка1");
Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(10);

ТФ.Запуск();
