Верхний.МышьНадЭлементом (Toplevel.MouseEnter) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события, когда указатель мыши находится над элементом.

Примечание

Аргументы события

Игрек (Y) Возвращает координату Y для события мыши.
Икс (X) Возвращает координату X для события мыши.
Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
ФлагиМыши (Flags) Возвращает флаги мыши для события мыши.
Элемент (View) Возвращает элемент.

Пример

Верхний1.МышьНадЭлементом = ТФ.Действие(ЭтотОбъект, "Верхний1_МышьНадЭлементом");
