КоллекцияВкладок.Удалить (TabsCollection.Remove) Метод

Синтаксис

Удалить(<Вкладка>)

Параметры

Вкладка (обязательный)
Тип: Вкладка (TabPage).

Возвращаемое значение

Описание

Удаляет указанную в параметре вкладку из коллекции вкладок панели вкладок.

Примечание

Пример

КоллекцияВкладок.Удалить(Вкладка2);
Полный пример кода 



Перем ТФ, ПанельВкладок1, Вкладка2;

Процедура ПриОткрытии() Экспорт
	КоллекцияВкладок = ПанельВкладок1.Вкладки;
	КоллекцияВкладок.Удалить(Вкладка2);
	ТФ.Уведомление().Показать("Количество = " + КоллекцияВкладок.Количество, -1);
