Верхний.УстановитьАвтоРазмер (Toplevel.SetAutoSize) Метод

Синтаксис

УстановитьАвтоРазмер()

Возвращаемое значение

Описание

Устанавливает размер элемента, чтобы соответствовать всему тексту.

Примечание

Пример

Верхний1.УстановитьАвтоРазмер();
