АргументыСобытия.ЭлементМеню (EventArgs.MenuItem) Свойство

Использование

Только чтение.

Значение

Тип: ЭлементМеню (MenuItem).

Возвращает элемент меню.

Примечание

Пример

АргументыСобытия = ТФ.АргументыСобытия;

|ЭлементМеню.Заголовок = " + АргументыСобытия.ЭлементМеню.Заголовок + "
|";
