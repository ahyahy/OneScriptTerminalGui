Кнопка.Заполнить (Button.Fill) Метод

Синтаксис

Заполнить(<ОтступСправа>, <ОтступСнизу>)

Параметры

ОтступСправа (необязательный)
Тип: Число.
Отступ между правой границей объекта и правой границей окна родителя.
ОтступСнизу (необязательный)
Тип: Число.
Отступ между нижней границей объекта и нижней границей окна родителя.

Возвращаемое значение

Описание

Заполняет размерность объекта, но оставляет указанные отступы.

Примечание

Пример

Кнопка1.Заполнить(1, 1);
Полный пример кода 



Перем ТФ, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Ширина = Кнопка1.Кадр.Ширина;
	Кнопка1.Текст = "Кнопка1 Ширина = " + Ширина + "
	|Кнопка1 Высота = " + Кнопка1.Кадр.Высота;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка();
Кнопка1.Текст = "Кнопка1";
Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(1);
Кнопка1.Ширина = ТФ.Величина.Абсолютно(25);
Кнопка1.Высота = ТФ.Величина.Абсолютно(3);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Кнопка1.Граница = Граница1;
ТФ.Верхний.Добавить(Кнопка1);

Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Кнопка1.Заполнить(1, 1);
	
ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
