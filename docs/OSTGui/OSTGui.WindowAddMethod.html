Окно.Добавить (Window.Add) Метод

Синтаксис

Добавить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Добавляет дочерний элемент к этому элементу.

Примечание

Пример

Окно0.Добавить(Окно4);
Полный пример кода
