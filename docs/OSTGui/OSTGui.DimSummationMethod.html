Величина.Сложить (Dim.Summation) Метод

Синтаксис

Сложить(<ПервоеСлагаемое>, <ВтороеСлагаемое>)

Параметры

ПервоеСлагаемое (обязательный)
Величина (Dim).
ВтороеСлагаемое (обязательный)
Величина (Dim).

Возвращаемое значение

Тип: Величина (Dim).

Описание

Суммирует два значения экземпляра класса Величина (Dim) получая в результате экземпляр класса Величина (Dim).

Примечание

Пример

Окно2.Ширина = ТФ.Величина.Сложить(Окно1.Ширина, Окно1.Ширина);
