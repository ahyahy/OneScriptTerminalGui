КоллекцияВкладок.Получить (TabsCollection.Get) Метод

Синтаксис

Получить(<Индекс>)

Параметры

Индекс (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Вкладка (TabPage).

Описание

Возвращает значение по указанному индексу из коллекции.

Примечание

Пример

Вкладка = КоллекцияВкладок.Получить(1);
Полный пример кода 



Перем ТФ, ПанельВкладок1;

Процедура ПриОткрытии() Экспорт
	КоллекцияВкладок = ПанельВкладок1.Вкладки;
	Вкладка = КоллекцияВкладок.Получить(1);
	ТФ.Уведомление().Показать("Вкладка = " + Вкладка, -1);
