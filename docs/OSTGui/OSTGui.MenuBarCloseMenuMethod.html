ПанельМеню.Закрыть (MenuBar.CloseMenu) Метод

Синтаксис

Закрыть()

Параметры

Возвращаемое значение

Описание

Программно закрывает меню, если оно открыто и видимо (как при нажатии F9).

Примечание

Пример

ПанельМеню1.Закрыть();
Полный пример кода 



Перем ТФ, ПанельМеню1, Текст, Открыто;

Процедура ПриОткрытии() Экспорт
	ПанельМеню1.Закрыть();
	Открыто = ПанельМеню1.Открыто;
	ТФ.Верхний.Текст = ТФ.Верхний.Текст + "
	|ПанельМеню1.Открыто = " + Открыто + "
	|";
КонецПроцедуры

Процедура Файл_Нажатие() Экспорт
	ТФ.Верхний.Текст = Текст + "Файл " + ТекущаяУниверсальнаяДатаВМиллисекундах();
КонецПроцедуры

КонецПроцедуры

Процедура Открыть_Нажатие() Экспорт
	ТФ.Верхний.Текст = Текст + "Открыть " + ТекущаяУниверсальнаяДатаВМиллисекундах();
КонецПроцедуры
