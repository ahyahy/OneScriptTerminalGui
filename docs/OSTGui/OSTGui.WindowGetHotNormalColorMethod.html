Окно.ЦветВыделенного (Window.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветВыделенного = Окно0.ЦветВыделенного();
Полный пример кода 



Перем Окно0, ЦветВыделенного;

Процедура ПриОткрытии() Экспорт
	ЦветВыделенного = Окно0.ЦветВыделенного();
	Окно0.Текст = "Окно0.ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ЦветВыделенного.ОсновнойЦвет + "
	|ЦветФона = " + ЦветВыделенного.ЦветФона + "
	|";
