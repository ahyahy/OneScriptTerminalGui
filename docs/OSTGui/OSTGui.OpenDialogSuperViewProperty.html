ДиалогОткрытия.Родитель (OpenDialog.SuperView) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает контейнер (родителя) для элемента или Неопределено, если этот элемент не был добавлен в контейнер.

Примечание

Пример

Родитель = ДиалогОткрытия1.Родитель;
Полный пример кода 



Перем ТФ, ДиалогОткрытия1;

Процедура ПриОткрытии() Экспорт
	Родитель = ДиалогОткрытия1.Родитель;
	ТФ.Уведомление().Показать("Родитель = " + Родитель, -1);
