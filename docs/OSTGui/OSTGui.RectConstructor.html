Прямоугольник (Rect) Конструктор

Синтаксис

Прямоугольник()

Прямоугольник(<Точка>, <Размер>)

Прямоугольник(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Точка (обязательный)
Тип: Точка (Point).
Координата точки для левого верхнего угла прямоугольника.
Размер (обязательный)
Тип: Размер (Size).
Объект класса Размер (Size) для значений ширины и высоты прямоугольника.
Икс (обязательный)
Тип: Число.
Координата икс для левого верхнего угла прямоугольника.
Игрек (обязательный)
Тип: Число.
Координата игрек для левого верхнего угла прямоугольника.
Ширина (обязательный)
Тип: Число.
Ширина прямоугольника.
Высота (обязательный)
Тип: Число.
Высота прямоугольника.

Описание

Создаёт новый экземпляр класса Прямоугольник (Rect).

Примечание

Пример

Прямоугольник1 = ТФ.Прямоугольник();
Прямоугольник2 = ТФ.Прямоугольник(ТФ.Точка(2, 2), ТФ.Размер(5, 5));
Прямоугольник3 = ТФ.Прямоугольник(2, 2, 5, 5);
