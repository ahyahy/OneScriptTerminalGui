Прямоугольник (Rect) Конструктор

Синтаксис

Прямоугольник(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата икс для левого верхнего угла прямоугольника. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата игрек для левого верхнего угла прямоугольника. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина прямоугольника. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота прямоугольника. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса Прямоугольник (Rect).

Пример
