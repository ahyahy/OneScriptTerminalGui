Таблица.ИзменитьВыделенноеДоКонцаСтроки (TableView.ChangeSelectionToEndOfRow) Метод

Синтаксис

ИзменитьВыделенноеДоКонцаСтроки(<Расширить>)

Параметры

Расширить (обязательный)
Тип: Булево.
Значение Истина, чтобы расширить текущий выбор (если есть) вместо замены.

Возвращаемое значение

Описание

Изменяет выделенную область до последней ячейки в текущей строке.

Примечание

Пример

Таблица1.ИзменитьВыделенноеДоКонцаСтроки(Истина);
Полный пример кода

КонецПроцедуры

	Выделена = Таблица1.Выделена(5, 0);
