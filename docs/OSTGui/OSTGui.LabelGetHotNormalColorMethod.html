Надпись.ЦветВыделенного (Label.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

Надпись1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Надпись1.ЦветоваяСхема.ГорячийНормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Надпись1, ЦветВыделенного;

Процедура ПриОткрытии() Экспорт
	ЦветВыделенного = Надпись1.ЦветВыделенного();
	Стр = "ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветВыделенного.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветВыделенного.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
