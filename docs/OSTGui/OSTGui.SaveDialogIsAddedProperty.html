ДиалогСохранения.Добавлено (SaveDialog.IsAdded) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает информацию о том добавлен ли элемент к родителю.

Примечание

Пример

Добавлено = ДиалогСохранения1.Добавлено;
Полный пример кода 



Перем ТФ, ДиалогСохранения1, Добавлено;

Процедура ПриОткрытии() Экспорт
	Добавлено = ДиалогСохранения1.Добавлено;
	ТФ.Уведомление().Показать("Добавлено = " + Добавлено, -1);
КонецПроцедуры
