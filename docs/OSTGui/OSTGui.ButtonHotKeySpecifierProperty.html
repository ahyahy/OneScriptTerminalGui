Кнопка.СимволКлавишиВызова (Button.HotKeySpecifier) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает символ для клавиши вызова (например, "_"). Установите Неопределено, чтобы отключить поддержку клавиши вызова для этого элемента. По умолчанию это Неопределено.

Примечание

Символ клавиши вызова определяет (указывает на) символ в тексте элемента, который будет соответствовать клавише вызова. Символ клавиши вызова должен находится перед символом в тексте элемента, который будет соответствовать клавише вызова.

Пример

Кнопка1.СимволКлавишиВызова = "_";

	Окно1.Заголовок = Окно1.Заголовок + " Кнопка1.КлавишаВызова = " + КлавишаВызова;
