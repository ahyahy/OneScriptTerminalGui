Кнопка.НаправлениеТекста (Button.TextDirection) Свойство

Использование

Чтение и запись.

Значение

Тип: НаправлениеТекста (TextDirection).

Возвращает или задает направление текста элемента. Изменение этого свойства повторно отобразит элемент.

Примечание

Пример

Кнопка1.НаправлениеТекста = ТФ.НаправлениеТекста.СправаНалевоСверхуВниз;
Полный пример кода 



Перем Кнопка1, НаправлениеТекста, Окно1;

Процедура ПриОткрытии() Экспорт
	НаправлениеТекста = Кнопка1.НаправлениеТекста;
	Окно1.Заголовок = Окно1.Заголовок + " Кнопка1.НаправлениеТекста = " + НаправлениеТекста;
