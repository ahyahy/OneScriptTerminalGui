ЭлементыПунктаМеню.Добавить (MenuBarItemChildren.Add) Метод

Синтаксис

Добавить(<Элемент>)

Параметры

Элемент (необязательный)
Тип: ЭлементМеню (MenuItem).

Возвращаемое значение

Описание

Добавляет ЭлементМеню (MenuItem) в конец коллекции или разделитель пунктов меню.

Примечание

Если использовать метод без параметра, будет добавлен разделитель пункта меню.

Пример

ПунктМеню1.Элементы.Добавить(ЭлементМеню2);
