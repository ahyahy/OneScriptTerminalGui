Элемент (View) События

События

ВидимостьИзменена (VisibleChanged) Возвращает или задает код для события при изменении свойства Отображать (Visible).
ДоступностьИзменена (EnabledChanged) Возвращает или задает код для события при изменении свойства Доступность (Enabled).
КлавишаНажата (KeyPress) Возвращает или задает код, выполняемый при нажатии клавиши в то время как элемент имеет фокус.
МышьНадЭлементом (MouseEnter) Возвращает или задает код для события, когда указатель мыши находится над элементом.
МышьПокинулаЭлемент (MouseLeave) Возвращает или задает код для события, когда мышь покинула элемент.
ПриВходе (Enter) Возвращает или задает код для события, когда элемент получает фокус.
ПриНажатииМыши (MouseClick) Возвращает или задает код для события, когда клавиша мыши нажата.
ПриУходе (Leave) Возвращает или задает код для события, когда элемент теряет фокус.
СочетаниеКлавишДействие (ShortcutAction) Возвращает или задает код обработчика события для сочетания клавиш, установленного для элемента.
