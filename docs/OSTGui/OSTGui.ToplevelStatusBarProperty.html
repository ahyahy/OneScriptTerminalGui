Верхний.СтрокаСостояния (Toplevel.StatusBar) Свойство

Использование

Чтение и запись.

Значение

Тип: СтрокаСостояния (StatusBar).

Возвращает или задает строку состояния для элемента.

Примечание

Пример

Верхний2.СтрокаСостояния = СтрокаСостояния1;
Полный пример кода 
Копировать     Выделить всё


ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();
ТФ.Активировать();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(1);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(1);
Верхний1.Ширина = ТФ.Величина.Абсолютно(60);
Верхний1.Высота = ТФ.Величина.Абсолютно(5);
ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
Верхний1.ЦветоваяСхема = ЦветоваяСхема1;
Верхний1.Текст = "Верхний1";
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
ТФ.Верхний.Добавить(Верхний1);

Верхний2 = ТФ.Верхний(1, 14, 60, 5);
ТФ.Верхний.Добавить(Верхний2);
Верхний2.Текст = "Верхний2";
Граница2 = ТФ.Граница();
Граница2.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний2.Граница = Граница2;
Верхний2.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Верхний2.Ниже(Верхний1, 1);

СтрокаСостояния1 = ТФ.СтрокаСостояния();
ЭлементСтрокиСостояния1 = ТФ.ЭлементСтрокиСостояния(ТФ.Клавиши.s_low, "Элемент1");
СтрокаСостояния1.Элементы.Добавить(ЭлементСтрокиСостояния1);
ЭлементСтрокиСостояния2 = ТФ.ЭлементСтрокиСостояния(ТФ.Клавиши.d_low, "Элемент2");
СтрокаСостояния1.Элементы.Добавить(ЭлементСтрокиСостояния2);

Верхний2.СтрокаСостояния = СтрокаСостояния1;

ТФ.Запуск();
