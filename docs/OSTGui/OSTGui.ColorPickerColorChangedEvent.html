ВыборЦвета.ПриИзмененииЦвета (ColorPicker.ColorChanged) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события при выборе цвета.

Примечание

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.

Пример

ВыборЦвета1.ПриИзмененииЦвета = ТФ.Действие(ЭтотОбъект, "ВыборЦвета1_ПриИзмененииЦвета");
