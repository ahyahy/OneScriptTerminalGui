Прокручиваемый.ТабФокус (ScrollView.TabStop) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли пользователь перевести фокус на данный элемент при помощи клавиши Tab. Применимо только если свойство Фокусируемый CanFocus также установлено в Истина.

Примечание

Пример

ТабФокус = Прокручиваемый1.ТабФокус;
Полный пример кода 



Перем ТФ, ТабФокус, Прокручиваемый1;

Процедура ПриОткрытии() Экспорт
	ТабФокус = Прокручиваемый1.ТабФокус;
	ТФ.Уведомление().Показать("ТабФокус = " + ТабФокус, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Прокручиваемый1 = ТФ.Верхний.Добавить(ТФ.Прокручиваемый(2, 2, 50, 15));
Прокручиваемый1.ЦветоваяСхема = ТФ.Цвета.Верхний;
Прокручиваемый1.РазмерСодержимого = ТФ.Размер(100, 50);
Прокручиваемый1.СмещениеСодержимого = ТФ.Точка(0, 0);

Надпись1 = Прокручиваемый1.Добавить(ТФ.Надпись("Текст для Надпись1 Текст для Надпись1 Текст для Надпись1", 0, 0, 60, 25));
Надпись1.ЦветоваяСхема = ТФ.Цвета.Ошибка;
Надпись1.АвтоРазмер = Ложь;
Для А = 0 По 29 Цикл
	Надпись1.Текст = Надпись1.Текст + "
	|Текст для Надпись1 Текст для Надпись1 Текст для Надпись1";
КонецЦикла;

Прокручиваемый1.Фокусируемый = Истина;
Прокручиваемый1.ТабФокус = Истина;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
