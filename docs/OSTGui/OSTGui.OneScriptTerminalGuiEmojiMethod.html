ТерминалФормыДляОдноСкрипта.Эмодзи (OneScriptTerminalGui.Emoji) Метод

Синтаксис

Эмодзи(<Параметр>)

Параметры

Параметр (обязательный)
Тип: Число; Строка.
Десятичное число или строка, представляющая число, шестнадцатеричную числовую константу или эскейп-последовательность символа Юникода.

Возвращаемое значение

Тип: Строка.

Описание

Возвращает эмодзи, если преобразование параметра представляется возможным. В противном случае возвращается пустая строка.

Примечание

Пример

ТФ.Верхний.Текст = ТФ.Верхний.Текст + "
|Эмодзи 169 строкой = " + ТФ.Эмодзи("169") + "
|Эмодзи 169 десятичным числом = " + ТФ.Эмодзи(169) + "
|Эмодзи -169 десятичным числом = " + ТФ.Эмодзи(-169) + "
|Эмодзи 0x169 строкой = " + ТФ.Эмодзи("0x169") + "
|Эмодзи \uA9 = " + ТФ.Эмодзи("\uA9") + "
|Эмодзи \u00A9 = " + ТФ.Эмодзи("\u00A9") + "
|Эмодзи 0xA9 = " + ТФ.Эмодзи("0xA9") + "
|Эмодзи 0x26F7 = " + ТФ.Эмодзи("0x26F7") + "
|Папка = " + ТФ.Эмодзи("📁") + "
|Корова = " + ТФ.Эмодзи("🐄") + "
|";
