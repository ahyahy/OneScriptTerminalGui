Текстовый (TextView) Конструктор

Синтаксис

Текстовый(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла поля выбора. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла поля выбора. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина поля выбора. Значение по умолчанию - 30.
Высота (необязательный)
Тип: Число.
Высота поля выбора. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса Текстовый (TextView).

Примечание

Пример

Текстовый1 = Окно1.Добавить(ТФ.Текстовый(2, 4, 50, 10));
Полный пример кода 



Перем ТФ, Окно1, Текстовый1, Текстовый;

Процедура ПриОткрытии() Экспорт

	Окно1.Текст = "Текстовый = " + Текстовый;
