Текстовый.Вставить (TextView.Paste) Метод

Синтаксис

Вставить()

Возвращаемое значение

Описание

Вставляет текст из буфера обмена.

Примечание

Пример

ТФ.ТекстБуфераОбмена = "Это ТекстБуфераОбмена ";
Текстовый1.Вставить();
Полный пример кода 



Перем ТФ, Текстовый1, Текст;

Процедура ПриОткрытии() Экспорт
	ТФ.ТекстБуфераОбмена = "Это ТекстБуфераОбмена ";
	Текстовый1.Вставить();
	Текст = Текстовый1.Текст;
	ТФ.Уведомление().Показать("Текст = " + Текст, -1);
