Атрибут.ОсновнойЦвет (Attribute.Foreground) Свойство

Использование

Только чтение.

Значение

Тип: Цвет (Color).

Цвет переднего плана.

Примечание

Пример

Окно1.Текст = "Атрибут1.ОсновнойЦвет = " + Атрибут1.ОсновнойЦвет;
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1 = ТФ.Верхний;
ЦветоваяСхема1 = ТФ.ЦветоваяСхема();

Атрибут1 = ТФ.Атрибут(ТФ.Цвет.ЯркоГолубой, ТФ.Цвет.Синий);

ЦветоваяСхема1.Нормальный = Атрибут1;
Верхний1.ЦветоваяСхема = ЦветоваяСхема1;

Окно1 = ТФ.Окно();
Окно1.Заголовок = "Окно1";
Окно1.Текст = "Атрибут1.ОсновнойЦвет = " + Атрибут1.ОсновнойЦвет;
Окно1.Икс = ТФ.Позиция.Абсолютно(1);
Окно1.Игрек = ТФ.Позиция.Абсолютно(1);
Окно1.Ширина = ТФ.Величина.Процент(50);
Окно1.Высота = ТФ.Величина.Абсолютно(6);
Верхний1.Добавить(Окно1);

ТФ.Запуск();
