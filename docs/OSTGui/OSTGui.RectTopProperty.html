Прямоугольник.Верх (Rect.Top) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает координату по оси Y верхнего края прямоугольника

Примечание

Пример

Верх = Прямоугольник1.Верх;
Полный пример кода 



Перем ТФ, Прямоугольник1;

Процедура ПриОткрытии() Экспорт
	Верх = Прямоугольник1.Верх;
	
	ТФ.Верхний.Текст = 
	"Верх = " + Верх + "
	|";
КонецПроцедуры
