Действие (Action) Конструктор

Синтаксис

Действие(<Сценарий>, <ИмяМетода>, <Параметр>)

Параметры

Сценарий (необязательный)
Тип: Произвольный.
В большинстве случаев для текущего сценария это переменная ЭтотОбъект.
ИмяМетода (необязательный)
Тип: Строка.
Параметр (необязательный)
Тип: Произвольный.

Описание

Создаёт новый экземпляр класса Действие (Action).

Примечание

Пример

Кнопка1.Нажатие = ТФ.Действие(ЭтотОбъект, "Кнопка1_Нажатие", Структура1);
