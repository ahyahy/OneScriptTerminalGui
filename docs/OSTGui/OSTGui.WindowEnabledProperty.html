Окно.Доступность (Window.Enabled) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли элемент реагировать на действия пользователя.

Примечание

Пример

Окно1.Доступность = Ложь;
Полный пример кода 



Перем Окно1;

Процедура ПриОткрытии() Экспорт
	Доступность = Окно1.Доступность;
	Окно1.Текст = "Окно1.Доступность = " + Доступность;
КонецПроцедуры
