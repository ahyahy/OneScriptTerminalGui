Окно.ЗаголовокИзменен (Window.TitleChanged) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события после того, как заголовок был изменен.

Примечание

Аргументы события

НовыйЗаголовок (NewTitle) Возвращает новый заголовок.
Отмена (Cancel) Возвращает или задает значение, указывающее, следует ли отменить событие.
Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
СтарыйЗаголовок (OldTitle) Возвращает старый заголовок.

Пример

Окно1.ЗаголовокИзменен = ТФ.Действие(ЭтотОбъект, "Окно1_ЗаголовокИзменен");
