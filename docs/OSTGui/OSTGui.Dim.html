Величина (Dim) Класс

Описание

Представляет класс для управления положением и размерами элементов относительно каких либо величин.

Этот класс используется для свойств Ширина (Width) и Высота (Height) элементов. Экземпляр этого класса представляет сам по себе значение для этих свойств. Экземпляры класса Величина (Dim) между собой могут складываться, вычитаться, обрабатывать целочисленные значения или представлять процентное отношение к другим величинам.

Важно: Объект Величина (Dim) нельзя переиспользовать. То есть при обращении к свойству элемента, имеющему тип значения Величина (Dim) или при присваивании свойству элемента значения этого типа каждый раз будет создан новый экземпляр объекта с типом Величина (Dim).
