Текстовый.ЦветВыделенного (TextView.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

Текстовый1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Текстовый1.ЦветоваяСхема.ГорячийНормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Текстовый1, ЦветВыделенного;

Процедура ПриОткрытии() Экспорт
	ЦветВыделенного = Текстовый1.ЦветВыделенного();
	Стр = "ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветВыделенного.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветВыделенного.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
