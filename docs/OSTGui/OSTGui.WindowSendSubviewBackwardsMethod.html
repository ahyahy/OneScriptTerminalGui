Окно.НаШагНазад (Window.SendSubviewBackwards) Метод

Синтаксис

НаШагНазад(<Подэлемент>)

Параметры

Подэлемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Перемещает подэлемент в системе компоновки на один шаг в обратном направлении.

Примечание

Пример

Окно0.НаШагНазад(Окно1);
