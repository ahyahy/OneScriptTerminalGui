ТерминалФормыДляОдноСкрипта.ПолеДаты (OneScriptTerminalGui.DateField) Метод

Синтаксис

ПолеДаты(<Дата>, <Икс>, <Игрек>)

Параметры

Дата (необязательный)
Тип: Дата.
Дата для поля даты, в которой содержатся год, месяц, день. Значение по умолчанию - текущая дата.
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла поля даты. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла поля даты. Значение по умолчанию - 1.

Возвращаемое значение

Тип: ПолеДаты (DateField).

Описание

Создаёт новый экземпляр класса ПолеДаты (DateField).

Примечание

Пример

ПолеДаты1 = Окно1.Добавить(ТФ.ПолеДаты(ТекущаяДата()));
Полный пример кода 



Перем ТФ, Окно1, ПолеДаты1, ПолеДаты;

Процедура ПриОткрытии() Экспорт

	Окно1.Текст = "Измените значение лет, месяцев, дней";
