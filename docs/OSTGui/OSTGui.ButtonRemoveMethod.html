Кнопка.Удалить (Button.Remove) Метод

Синтаксис

Удалить(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Удаляет подэлемент, добавленный методом Добавить (Add).

Примечание

Важно: Удаление объекта, участвующего в определении местоположения другого объекта, вызовет ошибку. Например в данном примере вызовет ошибку удаление объекта Окно1, так как он участвует в определении местоположения объекта Окно2 через метод Окно2.ВышеПравее(Окно1, -14, -53);.

Пример

Кнопка0.Удалить(Окно4);
Полный пример кода 
Копировать     Выделить всё


Перем ТФ, Кнопка0, Окно4;

Процедура ПриОткрытии() Экспорт
	ТФ.Верхний.Текст = "Осталось подэлементов = " + Кнопка0.Подэлементы.Количество;
КонецПроцедуры

Процедура Удалить() Экспорт
	Кнопка0.Удалить(Окно4);
	ТФ.Верхний.Текст = "Осталось подэлементов = " + Кнопка0.Подэлементы.Количество;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();
ТФ.Активировать();

Кнопка0 = ТФ.Кнопка();
Кнопка0.Икс = ТФ.Позиция.Абсолютно(25);
Кнопка0.Игрек = ТФ.Позиция.Абсолютно(2);
Кнопка0.Ширина = ТФ.Величина.Абсолютно(90);
Кнопка0.Высота = ТФ.Величина.Абсолютно(27);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Кнопка0.Граница = Граница1;
ТФ.Верхний.Добавить(Кнопка0);

Окно1 = ТФ.Окно();
Окно1.Заголовок = "Окно0";
Окно1.Текст = "Окно0";
Окно1.Икс = ТФ.Позиция.Абсолютно(1);
Окно1.Игрек = ТФ.Позиция.Абсолютно(10);
Окно1.Ширина = ТФ.Величина.Абсолютно(60);
Окно1.Высота = ТФ.Величина.Абсолютно(11);
Кнопка0.Добавить(Окно1);

Окно2 = ТФ.Окно("Окно1");
Окно2.Текст = "Окно1";
Окно2.Ширина = Окно1.Ширина;
Окно2.Высота = Окно1.Высота;
Окно2.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Окно2.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Черный, ТФ.Цвет.ТемноСерый);
Окно2.ВышеПравее(Окно1, -14, -53);
Кнопка0.Добавить(Окно2);

Окно3 = ТФ.Окно("Окно2");
Окно3.Текст = "Окно2";
Окно3.Ширина = Окно1.Ширина;
Окно3.Высота = Окно1.Высота;
Окно3.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Окно3.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Черный, ТФ.Цвет.Зеленый);
Кнопка0.Добавить(Окно3);
Окно3.ВышеПравее(Окно2, -7, -53);

Окно4 = ТФ.Окно("Окно3");
Окно4.Текст = "Окно3";
Окно4.Ширина = Окно1.Ширина;
Окно4.Высота = Окно1.Высота;
Окно4.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Окно4.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Черный, ТФ.Цвет.Голубой);
Кнопка0.Добавить(Окно4);
Окно4.ВышеПравее(Окно3, -10, -53);

Кнопка7 = ТФ.Кнопка("Окно0 назад");
Кнопка7.Текст = "Удалить
|Окно3";
Кнопка7.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка7.Игрек = ТФ.Позиция.Абсолютно(12);
Кнопка7.Нажатие = ТФ.Действие(ЭтотОбъект, "Удалить");
ТФ.Верхний.Добавить(Кнопка7);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
