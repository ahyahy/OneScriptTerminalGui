Окно.ПолучитьАвтоРазмер (Window.GetAutoSize) Метод

Синтаксис

ПолучитьАвтоРазмер()

Возвращаемое значение

Тип: Размер (Size).

Описание

Возвращает размер, чтобы соответствовать всему тексту, если включено свойство АвтоРазмер (AutoSize).

Примечание

Пример

Размер = Окно0.ПолучитьАвтоРазмер();
Окно0.Ширина = ТФ.Величина.Абсолютно(Размер.Ширина);
Окно0.Высота = ТФ.Величина.Абсолютно(Размер.Высота);
Полный пример кода 



Перем ТФ, Окно0;

Процедура ПриОткрытии() Экспорт
	Размер = Окно0.ПолучитьАвтоРазмер();
	Окно0.Ширина = ТФ.Величина.Абсолютно(Размер.Ширина);
	Окно0.Высота = ТФ.Величина.Абсолютно(Размер.Высота);
