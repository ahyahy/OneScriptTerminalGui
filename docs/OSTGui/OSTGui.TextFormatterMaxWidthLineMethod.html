ОформительТекста.МаксимальнаяШиринаСтроки (TextFormatter.MaxWidthLine) Метод

Синтаксис

МаксимальнаяШиринаСтроки(<Текст>)

Параметры

Текст (обязательный)
Тип: Строка.
Текст, может содержать перевод строки.

Возвращаемое значение

Тип: Число.

Ширина самой широкой строки.

Описание

Определяет ширину строки с наибольшей шириной в тексте.

Примечание

Пример

МаксимальнаяШиринаСтроки = ОформительТекста1.МаксимальнаяШиринаСтроки(ОформительТекста1.Текст);

	ТФ.Верхний.Текст = "ОформительТекста1.МаксимальнаяШиринаСтроки = " + МаксимальнаяШиринаСтроки;
