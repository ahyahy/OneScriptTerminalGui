ТерминалФормыДляОдноСкрипта.ТекстБуфераОбмена (OneScriptTerminalGui.ClipboardText) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текстовое содержимое буфера обмена операционной системы.

Примечание

Пример

ТФ.ТекстБуфераОбмена = "Это ТекстБуфераОбмена";
ТекстБуфераОбмена = ТФ.ТекстБуфераОбмена;

	ТекстБуфераОбмена = ТФ.ТекстБуфераОбмена;
	ТФ.Верхний.Текст = "ТекстБуфераОбмена = " + ТекстБуфераОбмена;
