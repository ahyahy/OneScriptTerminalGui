АргументыСобытия.Вкладка (EventArgs.Tab) Свойство

Использование

Только чтение.

Значение

Тип: Вкладка (TabPage).

Возвращает вкладку по которой кликнули мышью.

Примечание

Пример

Процедура ПанельВкладок1_ВкладкаНажата() Экспорт
	АргументыСобытия = ТФ.АргументыСобытия;
	Текст = "ПанельВкладок1_ВкладкаНажата
	|Отправитель = " + АргументыСобытия.Отправитель + "
	|Параметр = " + АргументыСобытия.Параметр + "
	|Вкладка = " + АргументыСобытия.Вкладка.Текст + "
	|Игрек = " + АргументыСобытия.Игрек + "
	|Икс = " + АргументыСобытия.Икс + "
	|ФлагиМыши = " + АргументыСобытия.ФлагиМыши + "
	|";
	ТФ.Уведомление().Показать(Текст, -1);
