Верхний (Toplevel) Класс

Описание

Представляет базовый класс для модальных/всплывающих окон.

Элементы верхнего уровня используются как для основного элемента приложения (заполнение всего экрана), так и для всплывающих элементов,таких как диалоги, окна сообщений.

Экземпляр класса Верхний (Toplevel) создается, когда приложение активируется. Доступ к верхнему уровню приложения можно получить через свойство ТерминалФормыДляОдноСкрипта.Верхний (OneScriptTerminalGui.Top).

Можно создавать и запускать дополнительные экземпляры класса Верхний (Toplevel) (например, диалоговые окна). Но это не желательно, потому что поведение некоторых классов (например класс Граница (Border)) будет не совсем корректно. Другими словами создавать экземпляр класса Верхний (Toplevel) как подэлемент ТерминалФормыДляОдноСкрипта.Верхний (OneScriptTerminalGui.Top) и размещать там основную логику событий нежелательно.
