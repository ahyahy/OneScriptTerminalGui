ПунктМеню.Нажатие (MenuBarItem.Clicked) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения при нажатии элемента.

Примечание

Если у пункта меню созданы элементы меню, тогда это событие не обрабатывается.

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.

Пример

_Справка.Нажатие = ТФ.Действие(ЭтотОбъект, "Справка_Нажатие");
