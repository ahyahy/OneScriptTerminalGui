Размер (Size) Конструктор

Синтаксис

Размер(<Ширина>, <Высота>)

Параметры

Ширина (необязательный)
Тип: Число.
Ширина. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса Размер (Size).

Размер1 = ТФ.Размер();
Размер2 = ТФ.Размер(5, 5);
