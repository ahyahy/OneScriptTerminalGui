СтильФлажкаЭлементаМеню (MenuItemCheckStyle) Перечисление

Описание

Указывает, каким образом для пункта меню отображается состояние выбора.

Члены

Имя члена Описание Значение
БезОтметки (NoCheck) Пункт меню будет отображаться как обычно, без индикатора проверки. Это значение по умолчанию. 0
Отметка (Checked) Пункт меню будет указывать на состояние Помечен/НеПомечен. 1
Переключатель (Radio) Пункт меню является частью группы переключателей и указывает на выбранное состояние. 2
