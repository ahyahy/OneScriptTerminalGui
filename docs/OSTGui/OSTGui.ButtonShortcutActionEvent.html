Кнопка.СочетаниеКлавишДействие (Button.ShortcutAction) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код обработчика события для сочетания клавиш, установленного для элемента.

Примечание

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.

Пример

Кнопка1.СочетаниеКлавишДействие = ТФ.Действие(ЭтотОбъект, "СочетаниеКлавишДействие");
Полный пример кода 



Перем ТФ;

Процедура ПриОткрытии() Экспорт
	Если ТФ.ПлатформаWin Тогда
		ТФ.ОтправитьКлавиши("s", Ложь, Ложь, Ложь);
	КонецЕсли;

	Объект1 = ТФ.АргументыСобытия.Параметр;
	Список = Объект1.ПолучитьСочетаниеКлавиш();
	Стр = "";

		Стр = Стр + "
		| " + Эл.Значение + " (" + Эл.Представление + ") " + ТекущаяУниверсальнаяДатаВМиллисекундах();
	КонецЦикла;
	
	ТФ.Верхний.Текст = "
	|
	|
	|
	|
	|
	|
	|Нажмите s в английской раскладке
	|СочетаниеКлавиш для " + Объект1.Текст + " = " + Стр;
