Окно.Фокусируемый (Window.CanFocus) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, может ли элемент управления получать фокус.

Примечание

Пример

Фокусируемый = Окно1.Фокусируемый;
Полный пример кода 



Перем Окно0, Окно1;

Процедура ПриОткрытии() Экспорт
	Стр = "";
	Подэлементы1 = Окно0.Подэлементы;
	Для А = 0 По Подэлементы1.Количество - 1 Цикл
		Стр = Стр + "Элемент(" + А + ") Фокусируемый = " + Подэлементы1.Получить(А).Фокусируемый + Символы.ПС;
	КонецЦикла;
	Окно0.Текст = Стр;
