УзелДерева.Текст (TreeNode.Text) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст для элемента.

Примечание

Пример

Узел1.Текст = "Узел1";
Текст = Узел1.Текст;
Полный пример кода 



Перем ТФ, Узел1, Текст;

Процедура ПриОткрытии() Экспорт
	Текст = Узел1.Текст; // Узлы - это массив.
	ТФ.Уведомление().Показать("Узел1.Текст = " + Текст, -1);
КонецПроцедуры

Дерево1 = ТФ.Дерево(, , 40, 15);
