ТерминалФормыДляОдноСкрипта.РазмерИзменен (OneScriptTerminalGui.Resized) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события при изменении размера терминала. Информируется о новом размере терминала.

Примечание

Отправителем в аргументах события будет экземпляр класса Верхний (Toplevel), который является значением свойства ТерминалФормыДляОдноСкрипта.Верхний (OneScriptTerminalGui.Top). И аргументы события (АргументыСобытия.Размер (EventArgs.Size)) будут приведены для этого элемента.

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
Размер (Size) Возвращает экземпляр класса Размер (Size), предсталяющий новый размер терминала.

Пример

ТФ.РазмерИзменен = ТФ.Действие(ЭтотОбъект, "ТФ_РазмерИзменен");

	|";
