Окно.Верх (Window.Top) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y верхнего края элемента.

Примечание

Пример

Верх = Окно1.Верх;
Окно1.Текст = "Окно1.Верх = " + Верх;
