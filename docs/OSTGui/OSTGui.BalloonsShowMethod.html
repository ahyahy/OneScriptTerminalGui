Уведомление.Показать (Balloons.Show) Метод

Синтаксис

Показать(<Сообщение>, <Интервал>, <Заголовок>, <Икс>, <Игрек>)

Сообщение (необязательный)
Тип: Строка.
Многострочное отображаемое сообщение. Значение по умолчанию - текущая дата.
Интервал (необязательный)
Тип: Число.
Время показа уведомления в миллисекундах. Значение по умолчанию - 3000 миллисекунд. Если установить значение -1 окно уведомления не закроется автоматически, так как время ожидания будет равно 20 годам.
Заголовок (необязательный)
Тип: Строка.
Заголовок окна уведомления. Значение по умолчанию - "Уведомление".
Икс (необязательный)
Тип: Позиция (Pos).
Позиция по оси X окна уведомления. По умолчанию окно расположено по центру.
Игрек (необязательный)
Тип: Позиция (Pos).
Позиция по оси Y окна уведомления. По умолчанию окно расположено по центру.

Возвращаемое значение

Описание

Показывает (открывает) окно уведомления.

Примечание

Пример

Уведомление1.Показать("Заголовок = " + Заголовок, -1);
Полный пример кода 



Перем ТФ, Уведомление1, Заголовок;

Процедура ПриОткрытии() Экспорт
	Заголовок = Уведомление1.Заголовок;
	Уведомление1.Показать("Заголовок = " + Заголовок, -1);
