Кнопка.СтильКомпоновки (Button.LayoutStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильКомпоновки (LayoutStyle).

Возвращает или задает стиль компоновки для элемента.

Примечание

Пример

Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Полный пример кода 



Перем ТФ, Кнопка1;

Процедура ПриОткрытии() Экспорт
	СтильКомпоновки = Кнопка1.СтильКомпоновки;
	ТФ.Верхний.Текст = "Кнопка1.СтильКомпоновки = " + ТФ.СтильКомпоновки.Имя(СтильКомпоновки);
КонецПроцедуры
