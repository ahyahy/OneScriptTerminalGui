СтрокаСостояния.ВерхнийРодитель (StatusBar.GetTopSuperView) Метод

Синтаксис

ВерхнийРодитель()

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает самого верхнего родителя.

Примечание

Пример

ВерхнийРодитель = СтрокаСостояния1.ВерхнийРодитель();
