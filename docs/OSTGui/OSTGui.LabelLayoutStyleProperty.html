Надпись.СтильКомпоновки (Label.LayoutStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильКомпоновки (LayoutStyle).

Возвращает или задает стиль компоновки для элемента.

Примечание

Пример

Надпись1.СтильКомпоновки = ТФ.СтильКомпоновки.Абсолютно;
Полный пример кода 



Перем ТФ, СтильКомпоновки, Надпись1;

Процедура ПриОткрытии() Экспорт
	СтильКомпоновки = Надпись1.СтильКомпоновки;
	ТФ.Уведомление().Показать("СтильКомпоновки = " + СтильКомпоновки, -1);
