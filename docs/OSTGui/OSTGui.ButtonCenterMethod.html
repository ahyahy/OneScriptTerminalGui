Кнопка.Центр (Button.Center) Метод

Синтаксис

Центр(<СмещениеИкс>, <СмещениеИгрек>)

Параметры

СмещениеИкс (необязательный)
Тип: Произвольный.
Величина смещения по оси икс.
СмещениеИгрек (необязательный)
Тип: Число.
Величина смещения по оси игрек.

Возвращаемое значение

Описание

Центрирует элемент управления внутри своего родителя с учетом параметров смещения.

Примечание

Важно: Перед применением метода установите свойство объекта СтильКомпоновки (LayoutStyle) в значение Вычислено (Computed).

Пример

Кнопка1.Центр();
