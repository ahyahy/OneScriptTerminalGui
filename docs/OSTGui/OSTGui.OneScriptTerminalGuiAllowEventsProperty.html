ТерминалФормыДляОдноСкрипта.РазрешитьСобытия (OneScriptTerminalGui.AllowEvents) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, показывающее, разрешена ли обработка событий.

Примечание

Значение по умолчанию - Истина. Это свойство позволяет разрешить или отменить обработку событий не останавливая выполнение скрипта.

Пример

ТФ.РазрешитьСобытия = Ложь;
