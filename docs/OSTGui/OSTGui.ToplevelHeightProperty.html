Верхний.Высота (Toplevel.Height) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает высоту элемента.

Примечание

Если используете процентное значение для установки высоты, переведите стиль компоновки в значение Вычислено (Computed).

Пример

Верхний1.Высота = ТФ.Величина.Процент(10);
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(1);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(1);
Верхний1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Верхний1.Ширина = ТФ.Величина.Процент(95);
Верхний1.Высота = ТФ.Величина.Процент(10);
Верхний1.Текст = "Верхний1.Высота = " + Верхний1.Высота;
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
Верхний1.ЦветоваяСхема = ТФ.Цвета.Диалог;
ТФ.Верхний.Добавить(Верхний1);

ТФ.Запуск();
