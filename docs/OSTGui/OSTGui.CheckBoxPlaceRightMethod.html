Флажок.Правее (CheckBox.PlaceRight) Метод

Синтаксис

Правее(<Элемент>, <Смещение>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент, относительно которого производим смещение.
Смещение (обязательный)
Тип: Число.
Величина смещения в целых числах левого края смещаемого элемента от правого края объекта, указанного в параметре Элемент.

Возвращаемое значение

Описание

Смещает правее этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.

Примечание

Пример

Флажок1.Правее(Кнопка0, 1);
Полный пример кода 



Перем ТФ, Игрек0, Игрек1, Кнопка0, Флажок1, Икс0, Икс1;

Процедура ПриОткрытии() Экспорт
	Игрек0 = Кнопка0.Кадр.Игрек;
	Игрек1 = Флажок1.Кадр.Игрек;
	Икс0 = Кнопка0.Кадр.Икс;
	Икс1 = Флажок1.Кадр.Икс;
	ТФ.Уведомление().Показать(
	"Игрек0 = " + Игрек0 + "
	|Игрек1 = " + Игрек1 + "
	|Икс0 = " + Икс0 + "
	|Икс1 = " + Икс1,
	-1);
