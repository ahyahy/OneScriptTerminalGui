СтрокаДанных.УстановитьЭлемент (DataRow.SetItem) Метод

Синтаксис

УстановитьЭлемент(<Индекс>, <Значение>)

Параметры

Индекс (обязательный)
Тип: Строка; Число.
Если Индекс является числом, отсчитываемый от нуля индекс колонки. Если Индекс является строковым выражением, имя колонки.
Значение (обязательный)
Тип: Произвольный.

Возвращаемое значение

Описание

Устанавливает данные, хранящиеся в указанной колонке.

Примечание

Пример

СтрокаДанных1 = ТаблицаДанных1.Строки.Элемент(1);
СтрокаДанных1.УстановитьЭлемент(1, "Яковлев Яков Яковлевич");
Полный пример кода 



Перем ТФ, ТаблицаДанных1, ТаблицаДанных;

	Количество = ТаблицаДанных1.Строки.Количество;
	ТФ.Уведомление().Показать("Количество = " + Количество, -1);
