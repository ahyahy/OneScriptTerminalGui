Позиция.Право (Pos.Right) Метод

Синтаксис

Право(<Элемент>)

Параметры

Элемент (обязательный)
Тип: Элемент (View).

Возвращаемое значение

Тип: Позиция (Pos).

Описание

Возвращает объект Позиция (Pos), отслеживающий правую (Икс+Ширина) координату указанного элемента.

Примечание

Пример

Позиция = ТФ.Позиция.Право(Кнопка1);
Полный пример кода

	ТФ.Верхний.Текст = 
	"Позиция = " + Позиция + "
	|";
