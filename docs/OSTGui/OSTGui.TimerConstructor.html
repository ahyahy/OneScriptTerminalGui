Таймер (Timer) Конструктор

Синтаксис

Таймер(<Интервал>)

Параметры

Интервал (необязательный)
Тип: Число.
Время в миллисекундах между отметками таймера. Значение по умолчанию - 1000.

Описание

Создаёт новый экземпляр класса Таймер (Timer).

Примечание

Пример

Таймер1 = ТФ.Таймер(1000);
