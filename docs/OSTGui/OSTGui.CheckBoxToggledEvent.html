Флажок.ПометкаИзменена (CheckBox.Toggled) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для выполнения при возникновении события переключения.

Примечание

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
СтарыйСтатус (OldToggled) Старое состояние флажка.

Пример

Флажок1.ПометкаИзменена = ТФ.Действие(ЭтотОбъект, "Флажок1_ПометкаИзменена");
