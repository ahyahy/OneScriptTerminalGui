Математика.АСинус (Math.Asin) Метод

Синтаксис

АСинус(<Синус>)

Параметры

Синус (обязательный)
Тип: Число.
Число, представляющее синус.

Возвращаемое значение

Тип: Число.

Угол, измеренный в радианах.

Описание

Возвращает числовое значение между -Пи/2 и Пи/2 радианами для значения x, лежащего в диапазоне от -1 до 1.
