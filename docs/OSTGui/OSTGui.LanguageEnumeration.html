Язык (Language) Перечисление

Описание

Указывает язык надписей для объектов некоторых классов. Например ДиалогОткрытия (OpenDialog), ДиалогСохранения (SaveDialog) или КонтекстноеМеню (ContextMenu).

Члены

Имя члена Описание Значение
Английский (English) Надписи объекта выводятся на английском языке. 1
Русский (Russian) Надписи объекта выводятся на русском языке. 0
