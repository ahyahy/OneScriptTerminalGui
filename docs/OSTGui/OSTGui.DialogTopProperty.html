Диалог.Верх (Dialog.Top) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y верхнего края элемента.

Примечание

Пример

Верх = Диалог1.Верх;
Полный пример кода 



Перем ТФ, Верх, Диалог1;

Процедура ПриОткрытии() Экспорт
	Верх = Диалог1.Верх;
	ТФ.Уведомление().Показать("Верх = " + Верх, -1);
