ПанельМеню (MenuBar) Конструктор

Синтаксис

ПанельМеню()

Параметры

Описание

Создаёт новый экземпляр класса ПанельМеню (MenuBar).

Примечание

Пример

ПанельМеню1 = ТФ.Верхний.Добавить(ТФ.ПанельМеню());
Полный пример кода 



Перем ТФ, ПанельМеню, ПанельМеню1, Текст;

Процедура ПриОткрытии() Экспорт
	ПанельМеню = ПанельМеню1;

ЭлементМеню2.Подсказка = "Выход";
ЭлементМеню2.Нажатие = ТФ.Действие(ЭтотОбъект, "Выход_Нажатие");

ПунктМеню2.Подсказка = "Справка";
ПунктМеню2.Нажатие = ТФ.Действие(ЭтотОбъект, "Справка_Нажатие");

Текст = "

|

ТФ.Запуск();
