Текстовый.Многострочный (TextView.Multiline) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, является ли элемент многострочным.

Примечание

Пример

Текстовый1.Многострочный = Ложь;
Многострочный = Текстовый1.Многострочный;
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.Многострочный = Ложь;
	Многострочный = Текстовый1.Многострочный;
	ТФ.Уведомление().Показать("Многострочный = " + Многострочный, -1);
