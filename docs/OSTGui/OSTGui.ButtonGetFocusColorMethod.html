Кнопка.ЦветФокуса (Button.GetFocusColor) Метод

Синтаксис

ЦветФокуса()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему для текущего фокуса.

Примечание

Пример

ЦветФокуса = Кнопка0.ЦветФокуса();
Полный пример кода 



Перем ТФ, Кнопка0;

Процедура Кнопка1_Нажатие() Экспорт
	ЦветФокуса = Кнопка0.ЦветФокуса();
	Кнопка0.Текст = "ЦветФокуса() = " + ЦветФокуса + "
	|ОсновнойЦвет = " + ЦветФокуса.ОсновнойЦвет + "
	|ЦветФона = " + ЦветФокуса.ЦветФона + "
	|";
