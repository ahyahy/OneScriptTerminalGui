Дерево.ПолучитьРодителя (TreeView.GetParent) Метод

Синтаксис

ПолучитьРодителя(<Узел>)

Параметры

Узел (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Тип: УзелДерева (TreeNode).

Описание

Возвращает родительский объект в дереве. Возвращает Неопределено, если объект не представлен в дереве.

Примечание

Пример

Родитель = Дерево1.ПолучитьРодителя(Узел5);
Полный пример кода 



Перем ТФ, Дерево1, Узел5;

Процедура ПриОткрытии() Экспорт
	Родитель = Дерево1.ПолучитьРодителя(Узел5);
	ТФ.Уведомление().Показать("Родитель = "+ Родитель.Текст, -1);
