ПолосаПрокрутки.Икс (ScrollBarView.X) Свойство

Использование

Чтение и запись.

Значение

Тип: Позиция (Pos).

Возвращает или задает объект Позиция (Pos), соответствующий координате по оси X.

Примечание

Пример

Икс = ПолосаПрокрутки1.Икс;
Полный пример кода 



Перем ТФ, Икс, ПолосаПрокрутки1;

Процедура ПриОткрытии() Экспорт
	Икс = ПолосаПрокрутки1.Икс;
	ТФ.Уведомление().Показать("Икс = " + Икс, -1);
