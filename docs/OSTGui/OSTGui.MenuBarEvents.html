ПанельМеню (MenuBar) События

События

ВидимостьИзменена (VisibleChanged) (унаследовано от Элемент (View)) Возвращает или задает код для события при изменении свойства Отображать (Visible).
ВсеЗакрыты (MenuAllClosed) Возвращает или задает код для события, когда все меню закрыты.
ДобавленЭлемент (Added) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда к этому элементу добавляется подэлемент.
ДоступностьИзменена (EnabledChanged) (унаследовано от Элемент (View)) Возвращает или задает код для события при изменении свойства Доступность (Enabled).
КлавишаВызоваИзменена (HotKeyChanged) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда изменилась клавиша вызова.
КлавишаНажата (KeyPress) (унаследовано от Элемент (View)) Возвращает или задает код, выполняемый при нажатии клавиши в то время как элемент имеет фокус.
МышьНадЭлементом (MouseEnter) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда мышь появилась над элементом.
МышьПокинулаЭлемент (MouseLeave) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда мышь покинула элемент.
Открыто (MenuOpened) Возвращает или задает код для события после открытия меню.
ПриВходе (Enter) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда элемент получает фокус.
ПриЗакрытии (MenuClosing) Возвращает или задает код для события перед закрытием меню.
ПриНажатииМыши (MouseClick) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда клавиша мыши нажата.
ПриОткрытии (MenuOpening) Возвращает или задает код для события перед открытием меню.
ФокусируемостьИзменена (CanFocusChanged) (унаследовано от Элемент (View)) Возвращает или задает код для события при изменении значения Фокусируемый (CanFocus).
ЭлементАктивирован (InitializedItem) (унаследовано от Элемент (View)) Возвращает или задает код в случае первой активации элемента. Позволяет выполнять конфигурации и назначения перед показом элемента.
ЭлементПокинут (Leave) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда элемент теряет фокус.
ЭлементУдален (Removed) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда элемент удален.
