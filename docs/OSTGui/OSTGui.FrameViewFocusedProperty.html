РамкаГруппы.ВФокусе (FrameView.Focused) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает элемент, который в настоящее время имеет фокус или Неопределено, если ничего не сфокусировано.

Примечание

Пример

ТекстЭлементаВФокусе = РамкаГруппы1.ВФокусе.Текст;
Полный пример кода 



Перем ТФ, ТекстЭлементаВФокусе, РамкаГруппы1, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Кнопка1.УстановитьФокус();
	ТекстЭлементаВФокусе = РамкаГруппы1.ВФокусе.Текст;
	ТФ.Верхний.Текст = "ТекстЭлементаВФокусе = " + ТекстЭлементаВФокусе;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

РамкаГруппы1 = ТФ.Верхний.Добавить(ТФ.РамкаГруппы("РамкаГруппы1", 1, 5, 25, 8));

Кнопка1 = ТФ.Кнопка("Кнопка1", 2, 1);
Кнопка1.Граница = ТФ.Граница();
Кнопка1.Граница.СтильГраницы = ТФ.СтильГраницы.Закругленная;

Кнопка2 = ТФ.Кнопка("Кнопка2", 2);
Кнопка2.Ниже(Кнопка1, 1);
Кнопка2.Граница = Кнопка1.Граница;

РамкаГруппы1.Добавить(Кнопка1);
РамкаГруппы1.Добавить(Кнопка2);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
