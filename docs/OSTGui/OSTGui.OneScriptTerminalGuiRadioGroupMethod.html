ТерминалФормыДляОдноСкрипта.Переключатель (OneScriptTerminalGui.RadioGroup) Метод

Синтаксис

Переключатель(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла переключателя. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла переключателя. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина переключателя. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота переключателя. Значение по умолчанию - 5.

Возвращаемое значение

Тип: Переключатель (RadioGroup).

Описание

Создаёт новый экземпляр класса Переключатель (RadioGroup).

Примечание

Пример

Переключатель1 = Окно1.Добавить(ТФ.Переключатель(1, 2, 15, 7));
