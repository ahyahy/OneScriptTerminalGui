Граница.ВычислитьТолщину (Border.GetSumThickness) Метод

Синтаксис

ВычислитьТолщину()

Возвращаемое значение

Тип: Толщина (Thickness).

Описание

Вычисляет сумму отступа и толщины границы.

Примечание

Пример

Толщина = Граница1.ВычислитьТолщину();
ТолщинаВерх = Толщина.Верх;

Процедура ПриОткрытии() Экспорт
	Толщина = Граница1.ВычислитьТолщину();
	ТолщинаВерх = Толщина.Верх;
	ТолщинаЛево = Толщина.Лево;
	ТолщинаНиз = Толщина.Низ;
	ТолщинаПраво = Толщина.Право;

	Верхний1.Текст = "Толщина.Верх = " + ТолщинаВерх + "
	|Толщина.Лево = " + ТолщинаЛево + "
	|Толщина.Низ = " + ТолщинаНиз + "
	|Толщина.Право = " + ТолщинаПраво + "";

Граница1.ЦветГраницы = ТФ.Цвет.ЯркоЗеленый;
Граница1.ЦветФона = ТФ.Цвет.Голубой;
