МакетПереключателя (DisplayModeLayout) Перечисление

Описание

Используется для выбора режима отображения переключателя.

Члены

Имя члена Описание Значение
Вертикально (Vertical) Отображение в вертикальном режиме. Это значение по умолчанию. 0
Горизонтально (Horizontal) Отображение в горизонтальном режиме. 1
