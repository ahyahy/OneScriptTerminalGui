Кнопка.Низ (Button.Bottom) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y, являющейся суммой значений свойств Игрек (Y) и Высота (Height) данного элемента.

Примечание

Пример

Низ = Кнопка1.Низ;
Полный пример кода 



Перем Кнопка1, Низ, Окно1;

Процедура ПриОткрытии() Экспорт
	Низ = Кнопка1.Низ;
	Окно1.Заголовок = Окно1.Заголовок + " Кнопка1.Низ = " + Низ;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Окно1 = ТФ.Окно("Окно1");
Окно1.Икс = ТФ.Позиция.Абсолютно(1);
Окно1.Игрек = ТФ.Позиция.Абсолютно(1);
Окно1.Ширина = ТФ.Величина.Абсолютно(50);
Окно1.Высота = ТФ.Величина.Абсолютно(15);
ТФ.Верхний.Добавить(Окно1);

Кнопка1 = ТФ.Кнопка("Кнопка1 - большая кнопка");
Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(1);
Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Кнопка1.Ширина = ТФ.Величина.Заполнить(1);
Кнопка1.Высота = ТФ.Величина.Заполнить(1);
Окно1.Добавить(Кнопка1);

Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Кнопка1.Граница = Граница1;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
