Верхний.Сфокусирован (Toplevel.HasFocus) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, имеет ли объект фокусировку.

Примечание

Важно: Всегда возвращает Ложь.

Пример

Верхний1.Текст = "Верхний1.Текст = " + Верхний1.Сфокусирован;
Полный пример кода 



Перем Верхний1;

Процедура ПриОткрытии() Экспорт
	Верхний1.Текст = "Верхний1.Текст = " + Верхний1.Сфокусирован;
