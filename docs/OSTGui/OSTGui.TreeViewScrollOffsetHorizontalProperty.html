Дерево.СмещениеГоризонтальное (TreeView.ScrollOffsetHorizontal) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает количество элементов в древовидном представлении, прокрученных вправо (по горизонтали).

Примечание

Установка значения меньше 0 приведет к смещению на 0. Чтобы увидеть изменения в пользовательском интерфейсе обновите экран.

Пример

Дерево1.СмещениеГоризонтальное = 4;
СмещениеГоризонтальное = Дерево1.СмещениеГоризонтальное;
Полный пример кода 



Перем ТФ, Дерево1;

Процедура ПриОткрытии() Экспорт
	Массив = Новый Массив();
	Для А = 9 По 60 Цикл
		Массив.Добавить(ТФ.УзелДерева("Узел" + А));
	КонецЦикла;
	Дерево1.ДобавитьУзлы(Массив);
	
	Дерево1.СмещениеГоризонтальное = 4;
	СмещениеГоризонтальное = Дерево1.СмещениеГоризонтальное;
	ТФ.Уведомление().Показать("СмещениеГоризонтальное = " + СмещениеГоризонтальное, -1);
