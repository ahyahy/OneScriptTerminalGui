ВыборЦвета.ТекущийСверху (ColorPicker.IsCurrentTop) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, является ли этот элемент в настоящее время на переднем плане (активен).
