ОформительТекста (TextFormatter) Свойства

Свойства

ВертикальноеВыравнивание (VerticalAlignment) Возвращает или задает выравнивание текста по вертикали.
Выравнивание (Alignment) Возвращает или задает, как текст элемента выровнен горизонтально.
Линии (Lines) Возвращает массив строк, содержащий строки текста.
НаправлениеТекста (Direction) Возвращает или задает вертикальное выравнивание текста.
Текст (Text) Возвращает или задает текст, который будет отображаться. Этот текст будет неизменным.
