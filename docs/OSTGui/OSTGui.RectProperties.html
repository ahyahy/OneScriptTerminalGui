Прямоугольник (Rect) Свойства

Свойства

Верх (Top) Возвращает координату по оси Y верхнего края прямоугольника
Высота (Height) Возвращает или задает высоту прямоугольника.
Игрек (Y) Возвращает или задает координату по оси Y левого верхнего угла прямоугольника.
Икс (X) Возвращает или задает координату по оси X левого верхнего угла прямоугольника.
Лево (Left) Возвращает координату по оси X левого края прямоугольника.
Низ (Bottom) Возвращает координату по оси Y нижнего края прямоугольника.
Право (Right) Возвращает координату по оси X правого края прямоугольника.
Ширина (Width) Возвращает или задает ширину прямоугольника.
