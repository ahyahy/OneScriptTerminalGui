ТерминалФормыДляОдноСкрипта.НоваяСтрока (OneScriptTerminalGui.NewLine) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает строку, обозначающую в данной среде начало новой строки.

Примечание

Пример

ВыделенныйТекст = "ка 4" + ТФ.НоваяСтрока + 
"Строка 5" + ТФ.НоваяСтрока + 
"Строка 6" + ТФ.НоваяСтрока + 
"Ст"
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.ПозицияКурсора = ТФ.Точка(4, 4);
	Текстовый1.КолонкаНачалаВыделения = 2;
	Текстовый1.СтрокаНачалаВыделения = 7;
	ВыделенныйТекст = Текстовый1.ВыделенныйТекст;
	ТФ.Уведомление().Показать("ВыделенныйТекст = " + ВыделенныйТекст, -1);
