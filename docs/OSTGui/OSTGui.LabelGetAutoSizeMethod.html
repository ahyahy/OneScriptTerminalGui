Надпись.ПолучитьАвтоРазмер (Label.GetAutoSize) Метод

Синтаксис

ПолучитьАвтоРазмер()

Возвращаемое значение

Тип: Размер (Size).

Описание

Возвращает размер, чтобы соответствовать всему тексту, если включено свойство АвтоРазмер (AutoSize).

Примечание

Пример

Размер = Надпись0.ПолучитьАвтоРазмер();
Полный пример кода 



Перем ТФ, Надпись0;

Процедура ПриОткрытии() Экспорт
	Размер = Надпись0.ПолучитьАвтоРазмер();
	Надпись0.Ширина = ТФ.Величина.Абсолютно(Размер.Ширина);
	Надпись0.Высота = ТФ.Величина.Абсолютно(Размер.Высота);
