АргументыСобытия.НоваяВкладка (EventArgs.NewTab) Свойство

Использование

Только чтение.

Значение

Тип: Вкладка (TabPage).

Возвращает вновь открытую вкладку.

Примечание

Пример

Процедура ПанельВкладок1_ВкладкаИзменена() Экспорт
	АргументыСобытия = ТФ.АргументыСобытия;
	Текст = "ПанельВкладок1_ВкладкаИзменена
	|СтараяВкладка = " + АргументыСобытия.СтараяВкладка.Текст + "
	|НоваяВкладка = " + АргументыСобытия.НоваяВкладка.Текст + "
	|";
	ТФ.Уведомление().Показать(Текст, -1);
