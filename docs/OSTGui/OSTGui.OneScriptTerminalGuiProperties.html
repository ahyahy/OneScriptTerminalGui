ТерминалФормыДляОдноСкрипта (OneScriptTerminalGui) Свойства

Свойства

АргументыСобытия (EventArgs) Возвращает аргументы события, связанные с действием, которое вызвало последнее событие.
Величина (Dim) Возвращает новый экземпляр класса Величина (Dim).
Верхний (Top) Возвращает экземпляр класса Верхний (Toplevel), создаваемый при активации приложения.
Высота (Rows) Текущее количество строк в терминале.
КлавишаВыхода (QuitKey) Возвращает или задает клавишу для выхода из приложения.
ЛогФайл (PathLog) Возвращает или задает путь до файла записей ошибок. По умолчанию это файл logtui.txt в каталоге стартового сценария.
НоваяСтрока (NewLine) Возвращает строку, обозначающую в данной среде начало новой строки.
Отправитель (Sender) Возвращает источник (объект) последнего события.
ПлатформаWin (WinPlatform) Возвращает значение, показывающее, используется ли операционная система семейства Windows.
Позиция (Pos) Возвращает новый экземпляр класса Позиция (Pos).
РазрешитьСобытия (AllowEvents) Возвращает или задает значение, показывающее, разрешена ли обработка событий.
ТекстБуфераОбмена (ClipboardText) Возвращает или задает текстовое содержимое буфера обмена операционной системы.
Цвета (Colors) Возвращает объект Цвета (Colors), содержащий цветовые схемы по умолчанию для приложения.
Ширина (Cols) Текущее количество столбцов в терминале.
