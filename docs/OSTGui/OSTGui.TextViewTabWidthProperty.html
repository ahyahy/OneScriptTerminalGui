Текстовый.ШиринаТаб (TextView.TabWidth) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает значение, указывающее количество пробелов при нажатии клавиши Tab.

Примечание

Пример

Текстовый1.ШиринаТаб = 6;
ШиринаТаб = Текстовый1.ШиринаТаб;
