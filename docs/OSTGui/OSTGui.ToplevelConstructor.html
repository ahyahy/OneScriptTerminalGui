Верхний (Toplevel) Конструктор

Синтаксис

Верхний(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла объекта. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла объекта. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина объекта. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота объекта. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса Верхний (Toplevel).

Примечание

Пример

Верхний1 = ТФ.Верхний();
Верхний2 = ТФ.Верхний(1, 9, 30, 5);
