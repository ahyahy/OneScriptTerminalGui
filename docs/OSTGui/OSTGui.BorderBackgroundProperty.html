Граница.ЦветФона (Border.Background) Свойство

Использование

Чтение и запись.

Значение

Тип: Цвет (Color).

Возвращает или задает цвет фона границы.

Примечание

Пример

Граница1.ЦветФона = ТФ.Цвет.Голубой;
Полный пример кода 



Перем ТФ, Верхний1, ЦветФона, Граница1;

Процедура ПриОткрытии() Экспорт
	ЦветФона = Граница1.ЦветФона;
	Верхний1.Текст = "Граница1.ЦветФона = " + ЦветФона;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(10);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(5);
Верхний1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Верхний1.Ширина = ТФ.Величина.Процент(60);
Верхний1.Высота = ТФ.Величина.Процент(20);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Граница1.Заголовок = "Заголовок границы";
Верхний1.Граница = Граница1;
Верхний1.Текст = "В это время к Джузеппе зашёл его старинный приятель, шарманщик по имени Карло.";
ТФ.Верхний.Добавить(Верхний1);
ТФ.Верхний.Текст = "Верхний";

Граница1.Эффект3D = Истина;
Граница1.Смещение3D = ТФ.Точка(2, 1);
Граница1.Цвет3D = ТФ.Атрибут(ТФ.Цвет.ЯркоГолубой);
Граница1.ЦветГраницы = ТФ.Цвет.ЯркоЗеленый;
Граница1.ЦветФона = ТФ.Цвет.Голубой;

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
