ОкноСообщений (MessageBox) Свойства

Свойства

АвтоРазмер (AutoSize) Возвращает или задает возможность автоподбора размера окна сообщений исходя из длины заголовка, размера сообщения и количества заданных кнопок. Значение по умолчанию - Истина.
Высота (Height) Возвращает или задает высоту окна сообщений.
Заголовок (Title) Возвращает или задает заголовок окна сообщений.
ИндексКнопкиПоУмолчанию (DefaultButtonIndex) Возвращает или задает индекс кнопки по умолчанию.
Кнопки (Buttons) Возвращает или задает массив кнопок для добавления.
Сообщение (Message) Возвращает или задает многострочное отображаемое сообщение.
Ширина (Width) Возвращает или задает ширину окна сообщений.
