Верхний.Кадр (Toplevel.Frame) Свойство

Использование

Чтение и запись.

Значение

Тип: Прямоугольник (Rect).

Возвращает или задает кадр для элемента. Кадр определяется относительно родителя.

Примечание

Пример

Верхний2.Кадр = ТФ.Прямоугольник(1, 13, 60, 8);
Полный пример кода 



Перем ТФ, Верхний1, Верхний2;

Процедура ПриОткрытии() Экспорт

	|Верхний1.Кадр.Икс = " + Верхний1.Кадр.Икс + "
	|Верхний1.Кадр.Игрек = " + Верхний1.Кадр.Игрек + "
	|Верхний1.Кадр.Ширина = " + Верхний1.Кадр.Ширина + "
	|Верхний1.Кадр.Высота = " + Верхний1.Кадр.Высота + "
	|";
