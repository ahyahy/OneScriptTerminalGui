Переключатель.Игрек (RadioGroup.Y) Свойство

Использование

Чтение и запись.

Значение

Тип: Позиция (Pos).

Возвращает или задает объект Позиция (Pos), соответствующий координате по оси Y.

Примечание

Пример

Игрек = Переключатель1.Игрек;
Полный пример кода 



Перем ТФ, Игрек, Переключатель1;

Процедура ПриОткрытии() Экспорт
	Игрек = Переключатель1.Игрек;
	ТФ.Уведомление().Показать("Игрек = " + Игрек, -1);
