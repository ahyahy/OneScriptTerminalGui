Прямоугольник.Содержит (Rect.Contains) Метод

Синтаксис

Содержит(<Икс>, <Игрек>)

Содержит(<Прямоугольник>)

Содержит(<Точка>)

Параметры

Икс (обязательный)
Тип: Число.
Координата икс.
Игрек (обязательный)
Тип: Число.
Координата игрек.
Прямоугольник (обязательный)
Тип: Прямоугольник (Rect).
Проверяемый прямоугольник.
Точка (обязательный)
Тип: Точка (Point).
Проверяемая точка.

Возвращаемое значение

Тип: Булево.

Описание

Проверяет, находится ли x, y координата, Точка (Point) или Прямоугольник (Rect) полностью в этом прямоугольнике.

Примечание

Пример

СодержитТочку = Прямоугольник1.Содержит(Точка1);
СодержитПрямоугольник = Прямоугольник1.Содержит(Прямоугольник2);
Содержит = Прямоугольник1.Содержит(7, 7);
Полный пример кода 



Перем ТФ, Прямоугольник1, Прямоугольник2, Точка1;

Процедура ПриОткрытии() Экспорт
	СодержитТочку = Прямоугольник1.Содержит(Точка1);
	СодержитПрямоугольник = Прямоугольник1.Содержит(Прямоугольник2);
	Содержит = Прямоугольник1.Содержит(7, 7);
	
	ТФ.Верхний.Текст = 
	"СодержитТочку = " + СодержитТочку + "
	|СодержитПрямоугольник = " + СодержитПрямоугольник + "
	|Содержит = " + Содержит + "
	|";
