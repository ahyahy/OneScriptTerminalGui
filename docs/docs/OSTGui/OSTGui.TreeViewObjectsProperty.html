Дерево.Корневые (TreeView.Objects) Свойство

Использование

Только чтение.

Значение

Тип: Массив, доступный только для чтения.

Возвращает корневые объекты в дереве. Обратите внимание, что в этой коллекции представлены только корневые объекты.

Примечание

Пример

Корневые = Дерево1.Корневые;
Стр = "";
Для А = 0 По Корневые.ВГраница() Цикл
	Эл = Корневые.Получить(А);
	Стр = Стр + Эл.Текст + ТФ.НоваяСтрока;
КонецЦикла;
ТФ.Уведомление().Показать("Корневые = " + ТФ.НоваяСтрока + Стр, -1);
