СтильДерева.ВыделятьТолькоТекст (TreeStyle.HighlightModelTextOnly) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, выделять ли только текст узла или же всю строку. Установите значение Истина, чтобы выбранный элемент отображался с выделением только текста. Если установлено значение Ложь (по умолчанию), будет выделена вся строка.

Примечание

Пример

СтильДерева1.ВыделятьТолькоТекст = Ложь;
Полный пример кода 



Перем ТФ, СтильДерева1;

Процедура ПриОткрытии() Экспорт
	ВыделятьТолькоТекст = СтильДерева1.ВыделятьТолькоТекст;
	ТФ.Уведомление().Показать("ВыделятьТолькоТекст = " + ВыделятьТолькоТекст, -1);
