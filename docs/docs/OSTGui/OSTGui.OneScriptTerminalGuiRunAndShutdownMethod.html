ТерминалФормыДляОдноСкрипта.ЗапуститьИЗавершить (OneScriptTerminalGui.RunAndShutdown) Метод

Синтаксис

ЗапуститьИЗавершить(<НеВыводитьОшибки>)

Параметры

НеВыводитьОшибки (необязательный)
Тип: Булево.
Не выводить сообщения об ошибке, возникшие при работе библиотеки. Значение по умолчанию - Ложь.

Возвращаемое значение

Описание

Запускает приложение и после выполнения возникших событий завершается работа приложения.

Примечание

Метод создан для упрощения тестирования библиотеки и выполняет завершение работы приложения после выполнения события ТерминалФормыДляОдноСкрипта.ПриОткрытии (OneScriptTerminalGui.NotifyNewRunState).

По разным причинам после завершения приложения TUI могут быть вызваны на выполнение назначенные обработчики событий. При установке параметра НеВыводитьОшибки в значение Истина сбой при выполнении этих событий не приведет к выводу сообщения об ошибке.

Параметр введен в целях улучшения, упрощения тестирования библиотеки.

Пример

ТФ.ЗапуститьИЗавершить();
