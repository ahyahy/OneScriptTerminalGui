Верхний.Идентификатор (Toplevel.Id) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает идентификатор для элемента.

Примечание

Пример

Верхний1.Идентификатор = "Идент1";
Полный пример кода 



Перем ТФ, Верхний1;

Процедура ПриОткрытии() Экспорт
	Верхний1.Текст = "Верхний1.Идентификатор = " + Верхний1.Идентификатор;
	ТФ.Обновить();
