Толщина (Thickness) Конструктор

Синтаксис

Толщина(<Длина>)

Толщина(<Левая>, <Верхняя>, <Правая>, <Нижняя>)

Параметры

Длина (обязательный)
Тип: Число.
Одинаковая длина каждой стороны.
Левая (обязательный)
Тип: Число.
Длина левой стороны.
Верхняя (обязательный)
Тип: Число.
Длина верхней стороны.
Правая (обязательный)
Тип: Число.
Длина правой стороны.
Нижняя (обязательный)
Тип: Число.
Длина нижней стороны.

Описание

Создаёт новый экземпляр класса Толщина (Thickness).

Примечание

Пример

Толщина1 = ТФ.Толщина(3);
Толщина2 = ТФ.Толщина(2, 2, 2, 2);
