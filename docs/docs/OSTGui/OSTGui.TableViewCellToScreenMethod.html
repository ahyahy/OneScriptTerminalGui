Таблица.ТочкаНаЯчейке (TableView.CellToScreen) Метод

Синтаксис

ТочкаНаЯчейке(<Колонка>, <Строка>)

Параметры

Колонка (обязательный)
Тип: Число.
Индекс колонки.
Строка (обязательный)
Тип: Число.
Индекс строки.

Возвращаемое значение

Тип: Точка (Point).

Значение Икс найденной точки соответствует координате икс начала ячейка.
Значение Игрек найденной точки соответствует координате игрек ячейки.

Описание

Вычисляет местоположение ячейки на экране (относительно клиентской области элемента управления), в которой отображается заданная ячейка, или значение Неопределено, если ячейка находится за пределами текущей области прокрутки или таблица не загружена.

Примечание

Пример

Точка = Таблица1.ТочкаНаЯчейке(1, 5);
Полный пример кода 



Перем ТФ, Таблица1;

Процедура ПриОткрытии() Экспорт
	Точка = Таблица1.ТочкаНаЯчейке(1, 5);
	ТФ.Уведомление().Показать("Точка.Икс = " + Точка.Икс + "
	|Точка.Игрек = " + Точка.Игрек + "
	|", -1);
