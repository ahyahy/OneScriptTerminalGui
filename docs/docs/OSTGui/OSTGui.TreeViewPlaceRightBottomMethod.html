Дерево.ПравееНиже (TreeView.PlaceRightBottom) Метод

Синтаксис

ПравееНиже(<Элемент>, <СмещениеИкс>, <СмещениеИгрек>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент, относительно которого производим смещение.
СмещениеИкс (обязательный)
Тип: Число.
Величина смещения в целых числах левого края смещаемого элемента от правого края объекта, указанного в параметре Элемент.
СмещениеИгрек (обязательный)
Тип: Число.
Величина смещения в целых числах верхнего края смещаемого элемента от нижнего края объекта, указанного в параметре Элемент.

Возвращаемое значение

Описание

Смещает правее и ниже этот элемент относительно объекта, указанного в параметре Элемент, на величины, указанные в параметрах.

Примечание

Пример

Дерево1.ПравееНиже(Кнопка0, 1, 1);
Полный пример кода 



Перем ТФ, Игрек0, Игрек1, Кнопка0, Дерево1, Икс0, Икс1;

Процедура ПриОткрытии() Экспорт
	Игрек0 = Кнопка0.Кадр.Игрек;
	Игрек1 = Дерево1.Кадр.Игрек;
	Икс0 = Кнопка0.Кадр.Икс;
	Икс1 = Дерево1.Кадр.Икс;
	ТФ.Уведомление().Показать(
	"Игрек0 = " + Игрек0 + "
	|Игрек1 = " + Игрек1 + "
	|Икс0 = " + Икс0 + "
	|Икс1 = " + Икс1,
	-1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");

Кнопка0 = ТФ.Верхний.Добавить(ТФ.Кнопка("Кнопка0"));
Кнопка0.Икс = ТФ.Позиция.Центр();
Кнопка0.Игрек = ТФ.Позиция.Центр();
Кнопка0.Ширина = ТФ.Величина.Абсолютно(10);
Кнопка0.Высота = ТФ.Величина.Абсолютно(1);

Дерево1 = ТФ.Верхний.Добавить(ТФ.Дерево(, , 40, 7));
Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Дерево1.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);

Узел1 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел1"));
Узел2 = Дерево1.ДобавитьУзел(ТФ.УзелДерева("Узел2"));

Узел1_1 = Узел1.Добавить(ТФ.УзелДерева("Узел1_1"));
Узел1_2 = Узел1.Добавить(ТФ.УзелДерева("Узел1_2"));
Узел1_1_1 = Узел1_1.Добавить(ТФ.УзелДерева("Узел1_1_1"));

Дерево1.РазвернутьВсе();

Дерево1.Икс = ТФ.Позиция.Центр();
Дерево1.ПравееНиже(Кнопка0, 1, 1);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
