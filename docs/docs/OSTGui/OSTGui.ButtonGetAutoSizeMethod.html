Кнопка.ПолучитьАвтоРазмер (Button.GetAutoSize) Метод

Синтаксис

ПолучитьАвтоРазмер()

Возвращаемое значение

Тип: Размер (Size).

Описание

Возвращает размер, чтобы соответствовать всему тексту, если включено свойство АвтоРазмер (AutoSize).

Примечание

Пример

Размер = Кнопка0.ПолучитьАвтоРазмер();
Полный пример кода 



Перем ТФ, Кнопка0;

Процедура ПриОткрытии() Экспорт
	Размер = Кнопка0.ПолучитьАвтоРазмер();
	Кнопка0.Ширина = ТФ.Величина.Абсолютно(Размер.Ширина);
	Кнопка0.Высота = ТФ.Величина.Абсолютно(Размер.Высота);
