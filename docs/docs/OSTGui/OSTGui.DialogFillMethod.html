Диалог.Заполнить (Dialog.Fill) Метод

Синтаксис

Заполнить(<ОтступСправа>, <ОтступСнизу>)

Параметры

ОтступСправа (необязательный)
Тип: Число.
Отступ между правой границей объекта и правой границей окна родителя.
ОтступСнизу (необязательный)
Тип: Число.
Отступ между нижней границей объекта и нижней границей окна родителя.

Возвращаемое значение

Описание

Заполняет размерность объекта, но оставляет указанные отступы.

Примечание

Пример

Диалог1.Заполнить(1, 1);
Полный пример кода 



Перем ТФ, Ширина, Диалог1;

Процедура ПриОткрытии() Экспорт

	ТФ.Уведомление().Показать("Ширина = " + Ширина, -1);
КонецПроцедуры
