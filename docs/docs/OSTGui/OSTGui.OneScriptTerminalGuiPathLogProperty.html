ТерминалФормыДляОдноСкрипта.ЛогФайл (OneScriptTerminalGui.PathLog) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает путь до файла записей ошибок. По умолчанию это файл logtui.txt в каталоге стартового сценария.

Примечание

В файл будет записана информация об ошибке. При отсутствии файла он будет создан. Запись об ошибке с указанием времени будет добавлена в файл.

Пример

ТекПуть = Новый Файл(ТекущийСценарий().Источник).Путь;
ТФ.ЛогФайл = ОбъединитьПути(ТекПуть, "ЛогФайл.txt");
Полный пример кода 



Перем ТФ, ЛогФайл;

Процедура ПриОткрытии() Экспорт
	ЛогФайл = ТФ.ЛогФайл;
	ТФ.Уведомление().Показать("ЛогФайл = " + ЛогФайл, -1);
