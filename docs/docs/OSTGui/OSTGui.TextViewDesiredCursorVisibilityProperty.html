Текстовый.Курсор (TextView.DesiredCursorVisibility) Свойство

Использование

Чтение и запись.

Значение

Тип: ВидКурсора (CursorVisibility).

Возвращает или задает нужный курсор при фокусировке на элементе.

Примечание

В некоторых терминалах возможно не соответствие заданному значению.

Пример

Текстовый1.Курсор = ТФ.ВидКурсора.Блок;
Полный пример кода 



Перем ТФ, Текстовый1, Курсор;

Процедура ПриОткрытии() Экспорт
	Курсор = Текстовый1.Курсор;
	ТФ.Уведомление().Показать("Курсор = " + ТФ.ВидКурсора.Имя(Курсор), -1);
