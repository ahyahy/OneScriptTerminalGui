Переключатель.Низ (RadioGroup.Bottom) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y, являющейся суммой значений свойств Игрек (Y) и Высота (Height) данного элемента.

Примечание

Пример

Низ = Переключатель1.Низ;
Полный пример кода 



Перем ТФ, Низ, Переключатель1;

Процедура ПриОткрытии() Экспорт
	Низ = Переключатель1.Низ;
	ТФ.Уведомление().Показать("Низ = " + Низ, -1);
