Индикатор.Доступность (ProgressBar.Enabled) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли элемент реагировать на действия пользователя.

Примечание

Пример

Доступность = Индикатор1.Доступность;
Полный пример кода 



Перем ТФ, Доступность, Индикатор1;

Процедура ПриОткрытии() Экспорт
	Доступность = Индикатор1.Доступность;
	ТФ.Уведомление().Показать("Доступность = " + Доступность, -1);
