Текстовый.ПрокрутитьДо (TextView.ScrollTo) Метод

Синтаксис

ПрокрутитьДо(<ИндексКолонки>, <ЭтоСтрока>)

Параметры

ИндексКолонки (обязательный)
Тип: Число.
ЭтоСтрока (необязательный)
Тип: Булево.
Значение по умолчанию - Истина.

Возвращаемое значение

Описание

Прокручивает текст для отображения указанной строки вверху, если параметр ЭтоСтрока имеет значение Истина. Иначе прокручивает текст для отображения указанной в параметре колонки слева, если значение ЭтоСтрока установлено в Ложь.

Примечание

Пример

Текстовый1.ПрокрутитьДо(10, Истина);
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.ПрокрутитьДо(10, Истина);
	ВерхняяСтрока = Текстовый1.ВерхняяСтрока;
	ТФ.Уведомление().Показать("ВерхняяСтрока = " + ВерхняяСтрока, -1);
