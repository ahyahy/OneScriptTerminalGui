ВыравниваниеКнопок (ButtonAlignments) Перечисление

Описание

Указывает горизонтальное выравнивание кнопок диалога.

Члены

Имя члена Описание Значение
Лево (Left) Кнопки выровнены по левому краю. 2
Подобранный (Justify) Подобранное выравнивание кнопок. 1
Право (Right) Кнопки выровнены по правому краю. 3
Центр (Center) Кнопки выровнены по центру. Это значение по умолчанию. 0
