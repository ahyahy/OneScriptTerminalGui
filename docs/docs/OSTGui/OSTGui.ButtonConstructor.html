Кнопка (Button) Конструктор

Синтаксис

Кнопка(<Текст>, <Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст для кнопки. Значение по умолчанию - "Кнопка".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла кнопки. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла кнопки. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина кнопки. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота кнопки. Значение по умолчанию - 1.

Описание

Создаёт новый экземпляр класса Кнопка (Button).

Примечание

Пример

Кнопка1 = ТФ.Кнопка("Кнопка1", , , 25, 3);
Полный пример кода 



Перем ТФ, Кнопка, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Кнопка = Кнопка1;
	ТФ.Уведомление().Показать("Кнопка = " + Кнопка, -1);
КонецПроцедуры
