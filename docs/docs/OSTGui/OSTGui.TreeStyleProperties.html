СтильДерева (TreeStyle) Свойства

Свойства

ВыделятьТолькоТекст (HighlightModelTextOnly) Возвращает или задает значение, указывающее, выделять ли только текст узла или же всю строку. Установите значение Истина, чтобы выбранный элемент отображался с выделением только текста. Если установлено значение Ложь (по умолчанию), будет выделена вся строка.
ИнверсияСимволов (InvertExpandSymbolColors) Возвращает или задает значение, указывающее, инвертировать ли цвета консоли, используемые для отображения символа раскрытия.
ПоказатьВертикальные (ShowBranchLines) Возвращает или задает значение, указывающее отображение вертикальных линий под развёрнутыми узлами. Установите значение Истина для отображения вертикальных линий под развёрнутыми узлами, чтобы показать, к какому родительскому узлу относится тот или иной узел. Ложь - для использования только пробелов.
ПоследняяСтрока (LeaveLastRow) Возвращает или задает значение, указывающее, состояние последней строки узла дерева. Установите значение Истина, чтобы последняя строка элемента управления оставалась свободной для перезаписи (например, полосой прокрутки). Если Истина, прокрутка будет запускаться на предпоследней строке элемента управления, а не на последней.
СимволРазворачивания (ExpandableSymbol) Возвращает или задает символ, который используется для обозначения узлов ветвления, которые можно развернуть, чтобы показать пользователю. По умолчанию используется '+'. Чтобы скрыть символ, установите значение Неопределено.
СимволСворачивания (CollapseableSymbol) Возвращает или задает символ, который используется для узлов ветвления, которые можно свернуть, если они в данный момент они развёрнуты. По умолчанию используется '-'. Чтобы скрыть узел, установите значение Неопределено.
ЦветныеСимволы (ColorExpandSymbol) Возвращает или задает значение, указывающее, выделять ли выделять символы разворачивания-сворачивания цветом горячих клавиш. Установите значение Истина, чтобы выделить символы разворачивания-сворачивания цветом горячих клавиш.
