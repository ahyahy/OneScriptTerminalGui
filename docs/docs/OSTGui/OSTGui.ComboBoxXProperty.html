ПолеВыбора.Икс (ComboBox.X) Свойство

Использование

Чтение и запись.

Значение

Тип: Позиция (Pos).

Возвращает или задает объект Позиция (Pos), соответствующий координате по оси X.

Примечание

Пример

ПолеВыбора1.Икс = ТФ.Позиция.Абсолютно(5);
Полный пример кода 



Перем ТФ, ПолеВыбора1, Икс, Игрек;

Процедура ПриОткрытии() Экспорт
	Игрек = ПолеВыбора1.Кадр.Игрек;
	Икс = ПолеВыбора1.Кадр.Икс;
	ТФ.Уведомление().Показать("Икс = " + Икс + " Игрек = " + Игрек, 3000);
