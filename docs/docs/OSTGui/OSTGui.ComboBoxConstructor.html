ПолеВыбора (ComboBox) Конструктор

Синтаксис

ПолеВыбора(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла поля выбора. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла поля выбора. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина поля выбора. Значение по умолчанию - 30.
Высота (необязательный)
Тип: Число.
Высота поля выбора. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса ПолеВыбора (ComboBox).

Примечание

Поле выбора не рисует границу. Чтобы обозначить границы для поля выбора, поместите его в объект Верхний (Toplevel), у которого есть нарисованные границы.

Пример

ПолеВыбора1 = Верхний1.Добавить(ТФ.ПолеВыбора(0, 0));
