СписокЭлементов.ПерейтиВверхСтраницы (ListView.MovePageUp) Метод

Синтаксис

ПерейтиВверхСтраницы()

Возвращаемое значение

Тип: Булево.

Возвращает Истина если действие выполнено успешно.

Описание

Переходит на элемент, расположенный в верхней части видимого списка.

Примечание

Пример

СписокЭлементов1.ПерейтиВверхСтраницы();
