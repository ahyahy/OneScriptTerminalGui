Флажок.Граница (CheckBox.Border) Свойство

Использование

Чтение и запись.

Значение

Тип: Граница (Border).

Экземпляр класса Граница Border для данного элемента.

Примечание

Важно: Каждый раз при установке значения этого свойства создается новый экземпляр класса Граница (Border).

Пример

Флажок1.Граница = ТФ.Граница(ТФ.СтильГраницы.Закругленная);
Полный пример кода 



Перем ТФ, Граница, Флажок1;

Процедура ПриОткрытии() Экспорт
	Граница = Флажок1.Граница;
	ТФ.Уведомление().Показать("Граница = " + Граница, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();
