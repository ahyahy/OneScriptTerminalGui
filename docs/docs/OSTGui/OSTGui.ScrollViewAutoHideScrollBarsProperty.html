Прокручиваемый.АвтоскрытиеПрокрутки (ScrollView.AutoHideScrollBars) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, скрывать ли вертикальную или горизонтальную полосы прокрутки, если в них нет необходимости.

Примечание

Пример

Прокручиваемый1.АвтоскрытиеПрокрутки = Истина;
Полный пример кода 



Перем ТФ, Прокручиваемый1;

Процедура ПриОткрытии() Экспорт
	АвтоскрытиеПрокрутки = Прокручиваемый1.АвтоскрытиеПрокрутки;
	ТФ.Уведомление().Показать("АвтоскрытиеПрокрутки = " + АвтоскрытиеПрокрутки, -1);
