ОформительТекста (TextFormatter) Методы

Методы

МаксимальнаяШиринаСтроки (MaxWidthLine) Определяет ширину строки с наибольшей шириной в тексте.
МаксимальноСтрок (MaxLines) Вычисляет количество строк, необходимых для отображения указанного текста с учётом ширины.
ПереносСлов (WordWrap) Форматирует предоставленный текст так, чтобы он соответствовал заданной ширине, используя перенос слов.
ПолучитьАвтоРазмер (GetAutoSize) Возвращает размер, чтобы соответствовать всему тексту.
ШиринаТекста (GetTextWidth) Возвращает общую ширину передаваемого текста.
