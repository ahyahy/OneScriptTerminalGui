Текстовый.Данные (TextView.Data) Свойство

Использование

Чтение и запись.

Значение

Тип: Произвольный.

Возвращает или задает произвольные данные для элемента.

Примечание

Пример

Структура1 = Новый Структура("Планета","Земля");
Структура1.Вставить("Звезда", "Солнце");
Структура1.Вставить("Спутник", "Луна");
Текстовый1.Данные = Структура1;
Полный пример кода 



Перем ТФ, Данные, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Данные = Текстовый1.Данные;
	ТФ.Уведомление().Показать("Данные = " + Данные, -1);
КонецПроцедуры
