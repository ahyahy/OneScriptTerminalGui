Верхний.Ширина (Toplevel.Width) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает ширину элемента.

Примечание

Если используете процентное значение для установки ширины, переведите стиль компоновки в значение Вычислено (Computed).

Пример

Верхний1.Ширина = ТФ.Величина.Абсолютно(60);
Полный пример кода 



Перем Верхний1;

Функция Тест() Экспорт
	Верхний1.Текст = "Верхний1.Ширина = " + Верхний1.Ширина;
	Возврат Верхний1.Ширина;
