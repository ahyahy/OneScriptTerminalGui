КолонкиДанных (DataColumnCollection) Методы

Методы

Добавить (Add) Создает и добавляет указанный объект КолонкаДанных (DataColumn) в коллекцию КолонкиДанных (DataColumnCollection).
ДобавитьЭлемент (AddItem) Создает и добавляет объект КолонкаДанных (DataColumn) с именем, указанным в КолонкиДанных (DataColumnCollection).
Очистить (Clear) Удаляет все колонки из коллекции.
Получить (Get) Возвращает значение по указанному индексу из коллекции.
Удалить (Remove) Удаляет объект КолонкаДанных (DataColumn) из коллекции.
УдалитьПоИндексу (RemoveAt) Удаляет колонку с указанным индексом из коллекции.
