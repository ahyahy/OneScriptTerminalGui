Кнопка.ЦветОбычного (Button.GetNormalColor) Метод

Синтаксис

ЦветОбычного()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветОбычного = Кнопка0.ЦветОбычного();
Полный пример кода 



Перем ТФ, Кнопка0;

Процедура Кнопка1_Нажатие() Экспорт
	ЦветОбычного = Кнопка0.ЦветОбычного();
	Кнопка0.Текст = "ЦветОбычного() = " + ЦветОбычного + "
	|ОсновнойЦвет = " + ЦветОбычного.ОсновнойЦвет + "
	|ЦветФона = " + ЦветОбычного.ЦветФона + "
	|";
