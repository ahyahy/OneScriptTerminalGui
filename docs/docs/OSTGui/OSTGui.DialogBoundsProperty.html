Диалог.Границы (Dialog.Bounds) Свойство

Использование

Только чтение.

Значение

Тип: Прямоугольник (Rect).

Возвращает прямоугольник, используемый для элемента.

Примечание

Пример

Границы = Диалог1.Границы;
Полный пример кода 



Перем ТФ, Границы, Диалог1;

Процедура ПриОткрытии() Экспорт
	Границы = Диалог1.Границы;
	ТФ.Уведомление().Показать("Границы = " + Границы, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");

Диалог1 = ТФ.Верхний.Добавить(ТФ.Диалог("Диалог1", 3, 3, 30, 5));
