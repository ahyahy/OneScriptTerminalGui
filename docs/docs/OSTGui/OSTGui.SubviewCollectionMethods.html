КоллекцияПодэлементов (SubviewCollection) Методы

Методы

Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс в коллекции для найденного элемента.
Получить (Get) Возвращает значение по указанному индексу из коллекции.
Содержит (Contains) Определяет, является ли указанный элемент членом коллекции.
