Дерево.ПолучитьИндексУзла (TreeView.GetObjectRow) Метод

Синтаксис

ПолучитьИндексУзла(<Узел>)

Параметры

Узел (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Тип: Число.

Описание

Возвращает координату Y в пределах границ дерева, в которой узел будет отображаться, или значение Неопределено, если в данный момент узел скрыт (например, его родительский элемент свернут). Обратите внимание, что возвращаемое значение может быть отрицательным, если дерево прокручено вниз, а искомый узел находится за пределами представления.

Примечание

Пример

Индекс = Дерево1.ПолучитьИндексУзла(Узел);
Полный пример кода 



Перем ТФ, Дерево1, Узел1;

Процедура ПриОткрытии() Экспорт
	Узел = Дерево1.ПолучитьПоИндексу(0);
	Индекс = Дерево1.ПолучитьИндексУзла(Узел);
	ТФ.Уведомление().Показать("Индекс = " + Индекс, -1);
КонецПроцедуры
