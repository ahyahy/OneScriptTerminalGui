Прокручиваемый.ПрокрутитьВправо (ScrollView.ScrollRight) Метод

Синтаксис

ПрокрутитьВправо(<Количество>)

Параметры

Количество (обязательный)
Тип: Число.
Количество колонок для прокрутки вправо.

Возвращаемое значение

Тип: Булево.

Возвращает Истина, если действие выполнено успешно.

Описание

Прокручивает содержимое вправо на указанную величину.

Примечание

Пример

Прокручиваемый1.ПрокрутитьВправо(5);
Полный пример кода 



Перем ТФ, Прокручиваемый1;

Процедура ПриОткрытии() Экспорт

	СмещениеСодержимого = Прокручиваемый1.СмещениеСодержимого;
	ТФ.Уведомление().Показать("СмещениеСодержимого.Икс = " + СмещениеСодержимого.Икс, -1);
