Свойства

БылоДень (OldDay) Возвращает предыдущее значение дня.
БылоМесяц (OldMonth) Возвращает предыдущее значение месяца.
БылоМинут (OldMinutes) Возвращает предыдущее значение минут.
БылоСекунд (OldSeconds) Возвращает предыдущее значение секунд.
БылоТактов (OldTicks) Возвращает предыдущее значение тактов (тиков).
БылоЧасов (OldHours) Возвращает предыдущее значение часов.
Вкладка (Tab) Возвращает вкладку по которой кликнули мышью.
Дерево (TreeView) Дерево, содержащее активированный узел.
Значение (Value) Возвращает значение элемента.
Игрек (Y) Возвращает координату Y для события мыши.
Икс (X) Возвращает координату X для события мыши.
ИндексВыбранного (SelectedItem) Возвращает индекс выбранного элемента.
ИндексКолонки (ColumnIndex) Возвращает индекс колонки.
ИндексПредыдущегоВыбранного (PreviousSelectedItem) Возвращает индекс предыдущего выбранного элемента.
ИндексСтроки (RowIndex) Возвращает индекс строки.
Клавиша (KeyValue) Возвращает значение для клавиши в виде целого числа.
КлавишаСтрокой (KeyToString) Возвращает значение для клавиши в виде строки вместе со значением модификатора, если таковой был применен.
НоваяВкладка (NewTab) Возвращает вновь открытую вкладку.
НоваяКолонка (NewCol) Возвращает индекс новой колонки.
НоваяСтрока (NewRow) Возвращает индекс новой строки.
НовыйПунктМеню (NewMenuBarItem) Пункт меню для подмены.
НовыйТекст (NewText) Возвращает текст элемента после изменения.
НовыйУзел (NewNode) Новый узел дерева.
Отмена (Cancel) Возвращает или задает значение, указывающее, следует ли отменить событие.
Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
ПутьКаталога (DirectoryPath) Возвращает абсолютный путь до каталога.
ПутьФайла (FilePath) Возвращает абсолютный путь до файла.
Размер (Size) Возвращает экземпляр класса Размер (Size), предсталяющий новый размер терминала.
РезультатДиалога (DialogResult) Возвращает значение перечисления РезультатДиалога (DialogResult), указывающее, какие кнопки были нажаты в диалоговом окне.
СталоГод (NewYear) Возвращает новое значение года.
СталоДень (NewDay) Возвращает новое значение дня.
СталоМесяц (NewMonth) Возвращает новое значение месяца.
СталоМинут (NewMinutes) Возвращает новое значение минут.
СталоСекунд (NewSeconds) Возвращает новое значение секунд.
СталоТактов (NewTicks) Возвращает новое значение тактов (тиков).
СталоЧасов (NewHours) Возвращает новое значение часов.
СтараяВкладка (OldTab) Возвращает предыдущую открытую вкладку.
СтараяКолонка (OldCol) Возвращает индекс старой колонки.
СтараяСтрока (OldRow) Возвращает индекс старой строки.
СтарыйСтатус (OldToggled) Старое состояние флажка.
СтарыйТекст (OldText) Возвращает текст элемента до изменения.
СтарыйУзел (OldNode) Старый узел дерева.
ТаблицаДанных (DataTable) Возвращает таблицу данных (источник) для редактируемой ячейки.
ТекущийПунктМеню (CurrentMenu) Текущий родитель пункта меню.
УзелДерева (TreeNode) Активированный узел дерева.
Файлы (FilePaths) Возвращает массив путей выбранных файлов.
ФлагиМыши (Flags) Возвращает флаги мыши для события мыши.
ФорматВремени (TimeFormat) Возвращает формат времени.
ФорматДаты (DateFormat) Возвращает формат даты.
Элемент (View) Возвращает элемент.
ЭлементМеню (MenuItem) Возвращает элемент меню.
ЭтоAlt (IsAlt) Возвращает значение, указывающее, была ли нажата клавиша Alt (реальная или синтезированная).
ЭтоCapslock (IsCapslock) Возвращает значение, указывающее, была ли нажата клавиша CapsLock (реальная или синтезированная).
ЭтоCtrl (IsCtrl) Возвращает значение, указывающее, была ли нажата клавиша Ctrl (реальная или синтезированная).
ЭтоNumlock (IsNumlock) Возвращает значение, указывающее, была ли нажата клавиша NumLock (реальная или синтезированная).
ЭтоScrolllock (IsScrolllock) Возвращает значение, указывающее, была ли нажата клавиша ScrollLock (реальная или синтезированная).
ЭтоShift (IsShift) Возвращает значение, указывающее, была ли нажата клавиша Shift (реальная или синтезированная).
