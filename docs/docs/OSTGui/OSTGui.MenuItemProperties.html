ЭлементМеню (MenuItem) Свойства

Свойства

Данные (Data) Возвращает или задает произвольные данные для элемента.
Заголовок (Title) Возвращает или задает заголовок элемента меню.
КлавишаВызова (HotKey) Возвращает клавишу вызова, определенную для элемента. Событие Нажатие (Clicked) сработает, если элемент имеет фокус.
Метка (Tag) Возвращает или задает объект, содержащий дополнительные данные.
Подсказка (Help) Возвращает или задает текст подсказки, которая отображается справа от заголовка элемента меню.
Помечен (Checked) Возвращает или задает информацию о том, помечен ли элемент меню.
СтильФлажка (CheckType) Возвращает или задает стиль элемента меню.
