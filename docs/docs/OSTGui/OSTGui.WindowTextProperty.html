Окно.Текст (Window.Text) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст для элемента.

Примечание

Пример

Стр = "Давным-давно в городке на берегу Средиземного моря жил старый столяр Джузеппе по прозванию Сизый Нос.
|Однажды ему попалось под руку полено, обыкновенное полено для топки очага в зимнее время.
|– Неплохая вещь, – сказал сам себе Джузеппе, – можно смастерить из него что-нибудь вроде ножки для стола…";
Окно0.Текст = Стр;
