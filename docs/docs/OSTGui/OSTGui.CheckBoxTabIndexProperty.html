Флажок.ПорядокОбхода (CheckBox.TabIndex) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает индекс текущего элемента в списке порядка обхода элементов.

Возвращает -1, если элементу не присвоен порядок обхода.

Примечание

Пример

ПорядокОбхода = Флажок1.ПорядокОбхода;
Полный пример кода 



Перем ТФ, Флажок1, ПорядокОбхода;

Процедура ПриОткрытии() Экспорт
	ПорядокОбхода = Флажок1.ПорядокОбхода;
	ТФ.Уведомление().Показать("ПорядокОбхода = " + ПорядокОбхода, -1);
КонецПроцедуры
