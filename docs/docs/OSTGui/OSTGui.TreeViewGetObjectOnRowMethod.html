Дерево.ПолучитьПоИндексу (TreeView.GetObjectOnRow) Метод

Синтаксис

ПолучитьПоИндексу(<ИндексСтроки>)

Параметры

ИндексСтроки (обязательный)
Тип: Число.

Возвращаемое значение

Тип: УзелДерева (TreeNode).

Описание

Возвращает объект из древовидного списка, который виден в данный момент в указанной строке. Возвращает значение Неопределено, если в указанном месте нет объекта.

Примечание

Пример

Узел = Дерево1.ПолучитьПоИндексу(0);
Полный пример кода 



Перем ТФ, Дерево1, Узел1;

Процедура ПриОткрытии() Экспорт
	Узел = Дерево1.ПолучитьПоИндексу(0);
	Дерево1.Перейти(Узел);
	Дерево1.Развернуть();
	Развернут = Дерево1.Развернут(Узел);
	ТФ.Уведомление().Показать("Развернут = " + Развернут, -1);
