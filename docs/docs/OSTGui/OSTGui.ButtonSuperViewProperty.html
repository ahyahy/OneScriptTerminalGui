Кнопка.Родитель (Button.SuperView) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает контейнер (родителя) для элемента или Неопределено, если этот элемент не был добавлен в контейнер.

Примечание

Пример

Родитель = Кнопка1.Родитель;
Полный пример кода 



Перем ТФ, Родитель, Кнопка1;

Процедура ПриОткрытии() Экспорт
	Родитель = Кнопка1.Родитель;
	ТФ.Верхний.Текст = ТФ.Верхний.Текст + "
	|Кнопка1.Родитель.Текст = " + Родитель.Текст;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка();
Кнопка1.Текст = "Кнопка1";
Кнопка1.Икс = ТФ.Позиция.Абсолютно(1);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(4);
Кнопка1.Ширина = ТФ.Величина.Абсолютно(25);
Кнопка1.Высота = ТФ.Величина.Абсолютно(3);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Кнопка1.Граница = Граница1;
ТФ.Верхний.Добавить(Кнопка1);

ТФ.Верхний.Текст = "ТФ.Верхний";

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
