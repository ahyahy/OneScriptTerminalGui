Библиотека OneScriptTerminalGui Перечисления

Перечисления

Перечисление Описание
ВертикальноеВыравниваниеТекста (VerticalTextAlignment) Указывает вертикальное выравнивание текста.
ВидКурсора (CursorVisibility) Указывает вид курсора.
ВыравниваниеКнопок (ButtonAlignments) Указывает горизонтальное выравнивание кнопок диалога.
ВыравниваниеТекста (TextAlignment) Указывает выравнивание текста.
Клавиши (Keys) Задает коды и модификаторы клавиш.
КлавишиКонсоли (ConsoleKey) Задает коды клавиш соответственно перечислению System.ConsoleKey фреймворка .NET.
КомандаTUI (CommandTUI) Действия, которые могут быть выполнены приложением или привязаны к клавишам в элементе.
МакетПереключателя (DisplayModeLayout) Используется для выбора режима отображения переключателя.
НаправлениеТекста (TextDirection) Указывает как отображается текст.
ПорядокСортировки (SortOrder) Указывает порядок сортировки элементов в ПолеВыбора (ComboBox) и СписокЭлементов (ListView).
РезультатДиалога (DialogResult) Указывает, какие кнопки были нажаты в диалоговом окне.
СтильГраницы (BorderStyle) Указывает стиль границы.
СтильИндикатора (ProgressBarStyle) Указывает стиль, который используется в индикаторе для отображения хода выполнения операции.
СтильКомпоновки (LayoutStyle) Указывает как выполняется компоновка элементов.
СтильФлажкаЭлементаМеню (MenuItemCheckStyle) Указывает, каким образом для пункта меню отображается состояние выбора.
ТипДанных (DataType) Указывает тип данных для объекта КолонкаДанных (DataColumn).
ФлагиМыши (MouseFlags) Задает константы, перечисляющие действия мыши.
ФорматИндикатора (ProgressBarFormat) Указывает формат, который используется в индикаторе для визуального отображения.
Цвет (Color) Цвета, которые можно использовать для установки переднего плана и фоновых цветов в приложениях консоли.
Язык (Language) Указывает язык надписей для объектов некоторых классов. Например ДиалогОткрытия (OpenDialog), ДиалогСохранения (SaveDialog) или КонтекстноеМеню (ContextMenu).
