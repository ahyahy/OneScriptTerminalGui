Дерево.ПолучитьРазвернутые (TreeView.GetExpandChildren) Метод

Синтаксис

ПолучитьРазвернутые(<Узел>)

Параметры

Узел (обязательный)
Тип: УзелДерева (TreeNode).

Возвращаемое значение

Описание

Возвращает раскрытые дочерние элементы переданного объекта. Возвращает пустую коллекцию, если ветка скрыта или не раскрыта.

Примечание

Пример

Развернутые = Дерево1.ПолучитьРазвернутые(Узел6);
Стр = "";
Для А = 0 По Развернутые.ВГраница() Цикл
	Эл = Развернутые.Получить(А);
	Стр = Стр + Эл.Текст + ТФ.НоваяСтрока;
КонецЦикла;
ТФ.Уведомление().Показать("Развернутые = "+ ТФ.НоваяСтрока + Стр, -1);
