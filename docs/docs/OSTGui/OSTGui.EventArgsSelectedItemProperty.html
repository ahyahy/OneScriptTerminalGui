АргументыСобытия.ИндексВыбранного (EventArgs.SelectedItem) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает индекс выбранного элемента.

Примечание

Пример

Процедура Переключатель1_ПометкаИзменена() Экспорт

	Текст = "Переключатель1_ПометкаИзменена = " + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
	|ИндексВыбранного = " + АргументыСобытия.ИндексВыбранного + "
	|ИндексПредыдущегоВыбранного = " + АргументыСобытия.ИндексПредыдущегоВыбранного + "
	|";
	ТФ.Уведомление().Показать(Текст, 2000);
