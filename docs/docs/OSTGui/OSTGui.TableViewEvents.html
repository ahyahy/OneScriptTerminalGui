Таблица (TableView) События

События

КлавишаНажата (KeyPress) (унаследовано от Элемент (View)) Возвращает или задает код, выполняемый при нажатии клавиши в то время как элемент имеет фокус.
МышьНадЭлементом (MouseEnter) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда указатель мыши находится над элементом.
МышьПокинулаЭлемент (MouseLeave) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда мышь покинула элемент.
ПриАктивацииЯчейки (CellActivated) Возвращает или задает код для события, когда ячейка активируется, например, при двойном щелчке или нажатии клавиши активации ячейки.
ПриВходе (Enter) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда элемент получает фокус.
ПриНажатииМыши (MouseClick) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда клавиша мыши нажата.
ПриУходе (Leave) (унаследовано от Элемент (View)) Возвращает или задает код для события, когда элемент теряет фокус.
СочетаниеКлавишДействие (ShortcutAction) (унаследовано от Элемент (View)) Возвращает или задает код обработчика события для сочетания клавиш, установленного для элемента.
ЯчейкаИзменена (SelectedCellChanged) Возвращает или задает код для события при изменении выбранной ячейки в таблице.
