Дерево.Низ (TreeView.Bottom) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y, являющейся суммой значений свойств Игрек (Y) и Высота (Height) данного элемента.

Примечание

Пример

Низ = Дерево1.Низ;
Полный пример кода 



Перем ТФ, Низ, Дерево1;

Процедура ПриОткрытии() Экспорт
	Низ = Дерево1.Низ;
	ТФ.Уведомление().Показать("Низ = " + Низ, -1);
КонецПроцедуры

Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
