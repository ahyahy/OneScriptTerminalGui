АргументыСобытия.Игрек (EventArgs.Y) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает координату Y для события мыши.

Примечание

Пример

АргументыСобытия = ТФ.АргументыСобытия;
ТФ.Верхний.Текст = "Верхний_ПриНажатииМыши = " + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
|ФлагиМыши = " + АргументыСобытия.ФлагиМыши + "
|Элемент = " + АргументыСобытия.Элемент + "
|Икс = " + АргументыСобытия.Икс + "
|Игрек = " + АргументыСобытия.Игрек + "
|";
