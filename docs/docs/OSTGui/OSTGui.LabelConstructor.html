Надпись (Label) Конструктор

Синтаксис

Надпись(<Текст>, <Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Текст (необязательный)
Тип: Строка.
Текст для надписи. Значение по умолчанию - "Надпись".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла надписи. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла надписи. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина надписи. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота надписи. Значение по умолчанию - 1.

Описание

Создаёт новый экземпляр класса Надпись (Label).

Примечание

Пример

Надпись1 = Окно1.Добавить(ТФ.Надпись("Текст для Надпись1", 0, 0));

Процедура ПриОткрытии() Экспорт
