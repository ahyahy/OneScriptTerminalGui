СписокЭлементов.СнятьДляВсех (ListView.AllowsAll) Метод

Синтаксис

СнятьДляВсех()

Возвращаемое значение

Описание

Если свойства Пометки (AllowsMarking) и МножественныйВыбор (AllowsMultipleSelection) имеют значение Истина, то все отмеченные элементы, кроме выбранного в данный момент, будут сняты с пометки.

Примечание

Пример

СписокЭлементов1.СнятьДляВсех();
Полный пример кода 



Перем ТФ, СписокЭлементов1;

Процедура ПриОткрытии() Экспорт
	СписокЭлементов1.Пометки = Истина;
	СписокЭлементов1.МножественныйВыбор = Истина;
	СписокЭлементов1.Пометить(0);
	СписокЭлементов1.Пометить(1);
	СписокЭлементов1.Пометить(2);
	СписокЭлементов1.Пометить(3);
	СписокЭлементов1.Пометить(4);
	СписокЭлементов1.ИндексВыбранного = 3;
	СписокЭлементов1.СнятьДляВсех();
	Помечен = СписокЭлементов1.Помечен(3);
	ТФ.Верхний.Текст = "Помечен = " + Помечен;
