Текстовый.КоличествоСтрок (TextView.Lines) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает количество строк.

Примечание

Пример

КоличествоСтрок = Текстовый1.КоличествоСтрок;
Полный пример кода 



Перем ТФ, Текстовый1;

	ТФ.Уведомление().Показать("КоличествоСтрок = " + КоличествоСтрок, -1);
