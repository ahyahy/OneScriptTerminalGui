ПолеВыбора.ТекстВыбранного (ComboBox.SelectedText) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает текст выбранного в настоящее время элемента списка.

Примечание

Пример

ТекстВыбранного = ПолеВыбора1.ТекстВыбранного;
Полный пример кода 



Перем ТФ, ПолеВыбора1, ТекстВыбранного;
