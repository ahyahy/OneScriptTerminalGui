Флажок.ЦветФокуса (CheckBox.GetFocusColor) Метод

Синтаксис

ЦветФокуса()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему для текущего фокуса.

Примечание

Пример

Флажок1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Флажок1.ЦветоваяСхема.Фокус = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Флажок1, ЦветФокуса;

Процедура ПриОткрытии() Экспорт
	ЦветФокуса = Флажок1.ЦветФокуса();
	Стр = "ЦветФокуса() = " + ЦветФокуса + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветФокуса.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветФокуса.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
