МеткиПереключателя.Очистить (RadioGroupItems.Clear) Метод

Синтаксис

Очистить()

Возвращаемое значение

Описание

Удаляет все элементы из коллекции.

Примечание

Пример

Переключатель1.Метки.Очистить();
Полный пример кода 



Перем ТФ, Переключатель1, Количество;

Процедура ПриОткрытии() Экспорт
	Переключатель1.Метки.Очистить();
	Количество = Переключатель1.Метки.Количество;
	ТФ.Уведомление().Показать("Количество = " + Количество, -1);
