ТерминалФормыДляОдноСкрипта.ПриОткрытии (OneScriptTerminalGui.NotifyNewRunState) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код, когда приложение запустилось.

Примечание

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.

Пример

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");
