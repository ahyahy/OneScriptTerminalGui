ОформительТекста.МаксимальноСтрок (TextFormatter.MaxLines) Метод

Синтаксис

МаксимальноСтрок(<Текст>, <Ширина>)

Параметры

Текст (обязательный)
Тип: Строка.
Текст, может содержать перевод строки.
Ширина (обязательный)
Тип: Число.
Минимальная ширина текста.

Возвращаемое значение

Тип: Число.

Описание

Вычисляет количество строк, необходимых для отображения указанного текста с учётом ширины.

Примечание

Пример

МаксимальноСтрок = ОформительТекста1.МаксимальноСтрок(ОформительТекста1.Текст, Окно1.Кадр.Ширина);
Полный пример кода

	ТФ.Верхний.Текст = "ОформительТекста1.МаксимальноСтрок = " + МаксимальноСтрок;
