Надпись.АвтоРазмер (Label.AutoSize) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, будет ли размер элемента автоматически изменяться в соответствии с его текстом. По умолчанию установлено значение Ложь. Если это свойство имеет значение Истина, то ширина и высота останутся неизменными, если размер текста меньше. Если размер текста больше, границы будут изменены в соответствии с текстом.

Примечание

Пример

АвтоРазмер = Надпись1.АвтоРазмер;
Полный пример кода 



Перем ТФ, АвтоРазмер, Надпись1;

Процедура ПриОткрытии() Экспорт
	АвтоРазмер = Надпись1.АвтоРазмер;
	ТФ.Уведомление().Показать("АвтоРазмер = " + АвтоРазмер, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Надпись1 = ТФ.Верхний.Добавить(ТФ.Надпись("Текст для Надпись1", 2, 2));
Надпись1.ЦветоваяСхема = ТФ.Цвета.Ошибка;

|Однажды ему попалось под руку полено, обыкновенное полено для топки очага в зимнее время.
|– Неплохая вещь, – сказал сам себе Джузеппе, – можно смастерить из него что-нибудь вроде ножки для стола…";
Надпись1.Текст = Стр;
