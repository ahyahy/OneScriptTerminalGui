СписокЗначений (ValueList) Методы

Методы

Вставить (Insert) Возвращает элемент из списка значений.
Добавить (Add) Добавляет элемент в список значений.
Индекс (IndexOf) Возвращает отсчитываемый от нуля индекс в коллекции для найденного элемента.
НайтиПоЗначению (FindByValue) Возврашает элемент списка значений, найденный по его значению.
Очистить (Clear) Удаляет все элементы из списка значений.
Получить (Get) Возвращает элемент из списка значений.
Сдвинуть (Move) Сдвигает элемент на указанное количество позиций.
СортироватьПоЗначению (SortByValue) Сортирует элементы в списке по порядку значений.
СортироватьПоПредставлению (SortByPresentation) Сортирует элементы в списке по порядку представления.
Удалить (Delete) Удаляет элемент по указанному индексу.
