СписокЭлементов.Идентификатор (ListView.Id) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает идентификатор для элемента.

Примечание

Пример

СписокЭлементов1.Идентификатор = "Идент";
Полный пример кода 



Перем ТФ, СписокЭлементов1, Идентификатор, Окно1;

Процедура ПриОткрытии() Экспорт
	Идентификатор = СписокЭлементов1.Идентификатор;
	ТФ.Уведомление().Показать("Идентификатор = " + Идентификатор, 3000);
