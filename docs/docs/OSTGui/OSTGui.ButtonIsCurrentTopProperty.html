Кнопка.ТекущийСверху (Button.IsCurrentTop) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, является ли этот элемент в настоящее время на переднем плане (активен).

Примечание

Анализ для элемента производится в пределах своего родителя.

Пример

ТекущийСверху = Кнопка1.ТекущийСверху;
Полный пример кода 



Перем ТФ, Кнопка1, ТекущийСверху;

Процедура ПриОткрытии() Экспорт
	ТекущийСверху = Кнопка1.ТекущийСверху;
	ТФ.Верхний.Текст = "Кнопка1.ТекущийСверху = " + ТекущийСверху;
