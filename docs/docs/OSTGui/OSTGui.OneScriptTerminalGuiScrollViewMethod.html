ТерминалФормыДляОдноСкрипта.Прокручиваемый (OneScriptTerminalGui.ScrollView) Метод

Синтаксис

Прокручиваемый(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла элемента. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла элемента. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина элемента. Значение по умолчанию - 20.
Высота (необязательный)
Тип: Число.
Высота элемента. Значение по умолчанию - 5.

Возвращаемое значение

Тип: Прокручиваемый (ScrollView).

Описание

Создаёт новый экземпляр класса Прокручиваемый (ScrollView).

Примечание

Пример

Прокручиваемый1 = Окно1.Добавить(ТФ.Прокручиваемый(2, 2, 50, 15));
