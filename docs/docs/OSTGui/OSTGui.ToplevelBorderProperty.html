Верхний.Граница (Toplevel.Border) Свойство

Использование

Чтение и запись.

Значение

Тип: Граница (Border).

Экземпляр класса Граница Border для данного элемента.

Примечание

Важно: Каждый раз при установке значения этого свойства создается новый экземпляр класса Граница (Border).

Пример

Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Верхний1 = ТФ.Верхний();
Верхний1.Икс = ТФ.Позиция.Абсолютно(1);
Верхний1.Игрек = ТФ.Позиция.Абсолютно(1);
Верхний1.Ширина = ТФ.Величина.Абсолютно(95);
Верхний1.Высота = ТФ.Величина.Абсолютно(10);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Двойная;
Верхний1.Граница = Граница1;
Верхний1.ЦветоваяСхема = ТФ.Цвета.Диалог;
ТФ.Верхний.Добавить(Верхний1);

Верхний1.Текст = "Верхний1.Граница = " + Верхний1.Граница;

ТФ.Запуск();
