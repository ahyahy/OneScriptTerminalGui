ПанельВкладок.ВкладкаНажата (TabView.TabClicked) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события при нажатии на вкладку. Может использоваться для отмены навигации, отображения контекстного меню (например, при щелчке правой кнопкой мыши) и т. д.

Примечание

Аргументы события

Вкладка (Tab) Возвращает вкладку по которой кликнули мышью.
Игрек (Y) Возвращает координату Y для события мыши.
Икс (X) Возвращает координату X для события мыши.
Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.
ФлагиМыши (Flags) Возвращает флаги мыши для события мыши.

Пример

ПанельВкладок1.ВкладкаНажата = ТФ.Действие(ЭтотОбъект, "ПанельВкладок1_ВкладкаНажата");
Полный пример кода 



Перем ТФ, ПанельВкладок1, АргументыСобытия;

Процедура ПриОткрытии() Экспорт
	ТФ.ОтправитьКлавишуКонсоли(ТФ.КлавишиКонсоли.Enter);
КонецПроцедуры

Процедура ПанельВкладок1_ВкладкаНажата() Экспорт
	АргументыСобытия = ТФ.АргументыСобытия;
	Текст = "ПанельВкладок1_ВкладкаНажата
	|Отправитель = " + АргументыСобытия.Отправитель + "
	|Параметр = " + АргументыСобытия.Параметр + "
	|Вкладка = " + АргументыСобытия.Вкладка.Текст + "
	|Игрек = " + АргументыСобытия.Игрек + "
	|Икс = " + АргументыСобытия.Икс + "
	|ФлагиМыши = " + АргументыСобытия.ФлагиМыши + "
	|";
	ТФ.Уведомление().Показать(Текст, -1);
