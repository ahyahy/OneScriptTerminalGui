СтильКомпоновки (LayoutStyle) Перечисление

Описание

Указывает как выполняется компоновка элемента.

Библиотека ТерминалФормыДляОдноСкрипта поддерживает две различные системы компоновки: абсолютную и вычисляемую. Абсолютная система используется, когда вы хотите, чтобы представление располагалось точно в указанном месте, и хотите вручную управлять его расположением. Для этого нужно вызвать конструктор элемента с аргументом типа Прямоугольник (Rect). После этого для изменения положения представления можно изменять свойство Кадр (Frame) элемента.

Система компоновки не поддерживает целые числа. Вместо этого свойства Икс (X) и Игрек (Y) имеют тип Позиция (Pos), а свойства Ширина (Width) и Высота (Height) имеют тип Величина (Dim), которые могут быть созданы неявно на основе целочисленных значений.

Члены

Имя члена Описание Значение
Абсолютно (Absolute) Положение и размер элемента основаны на значении кадра. 0
Вычислено (Computed) Положение и размер элемента будут рассчитаны на основе свойств X, Y, ширины и высоты и установлены на кадре. 1
