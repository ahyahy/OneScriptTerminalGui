Верхний.СтильКомпоновки (Toplevel.LayoutStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильКомпоновки (LayoutStyle).

Возвращает или задает стиль компоновки для элемента.

Примечание

Пример

Верхний1.Текст = "Верхний1.СтильКомпоновки =  " + Верхний1.СтильКомпоновки;
Верхний2.Текст = "Верхний2.СтильКомпоновки =  " + Верхний2.СтильКомпоновки;
Полный пример кода 



Перем ТФ, Верхний1, Верхний2;

Процедура ПриОткрытии() Экспорт
	Верхний1.Текст = "Верхний1.СтильКомпоновки =  " + Верхний1.СтильКомпоновки;
	Верхний2.Текст = "Верхний2.СтильКомпоновки =  " + Верхний2.СтильКомпоновки;
