Величина.Вычесть (Dim.Subtract) Метод

Синтаксис

Вычесть(<Уменьшаемое>, <Вычитаемое>)

Параметры

Уменьшаемое (обязательный)
Тип: Величина (Dim).
Вычитаемое (обязательный)
Тип: Величина (Dim).

Возвращаемое значение

Тип: Величина (Dim).

Описание

Вычитает из одного значения экземпляра класса Величина (Dim) другое значение экземпляра класса Величина (Dim), получая в результате экземпляр класса Величина (Dim).

Примечание

Пример

Окно1.Ширина = ТФ.Величина.Вычесть(ТФ.Величина.Абсолютно(110), ТФ.Величина.Абсолютно(20));
