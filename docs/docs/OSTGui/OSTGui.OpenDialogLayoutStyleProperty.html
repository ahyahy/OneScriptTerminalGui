ДиалогОткрытия.СтильКомпоновки (OpenDialog.LayoutStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильКомпоновки (LayoutStyle).

Возвращает или задает стиль компоновки для элемента.

Примечание

Пример

ДиалогОткрытия1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Полный пример кода 



Перем ТФ, ДиалогОткрытия1, СтильКомпоновки;

Процедура ПриОткрытии() Экспорт
	СтильКомпоновки = ДиалогОткрытия1.СтильКомпоновки;
	ТФ.Уведомление().Показать("СтильКомпоновки = " + ТФ.СтильКомпоновки.Имя(СтильКомпоновки), -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

ДиалогОткрытия1 = ТФ.Верхний.Добавить(ТФ.ДиалогОткрытия("Диалог открытия", "Сообщение", 2, 2));

ТипыФайлов = Новый Массив();
ТипыФайлов.Добавить(".txt;.bin;.xml;.json");
ТипыФайлов.Добавить(".txt");
ТипыФайлов.Добавить(".bin");
ТипыФайлов.Добавить(".xml");
ТипыФайлов.Добавить(".json");
ДиалогОткрытия1.ДопустимыеФайлы = ТипыФайлов;

ДиалогОткрытия1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
