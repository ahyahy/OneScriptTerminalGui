Дерево (TreeView) Конструктор

Синтаксис

Дерево(<Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла дерева. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла дерева. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина дерева. Значение по умолчанию - 10.
Высота (необязательный)
Тип: Число.
Высота дерева. Значение по умолчанию - 5.

Описание

Создаёт новый экземпляр класса Дерево (TreeView).

Примечание

Пример

Дерево1 = ТФ.Дерево(, , 40, 15);
Полный пример кода 



Перем ТФ, Дерево1, Дерево;

Процедура ПриОткрытии() Экспорт
	Дерево = Дерево1;
	ТФ.Уведомление().Показать("Дерево = " + Дерево, -1);
