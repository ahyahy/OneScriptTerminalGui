Текстовый.ПереместитьВНачало (TextView.MoveHome) Метод

Синтаксис

ПереместитьВНачало()

Возвращаемое значение

Описание

Прокручивает текст до первой строки и установит курсор в это место.

Примечание

Пример

Текстовый1.ПереместитьВНачало();
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.ПереместитьВКонец();
	Текстовый1.ПереместитьВНачало();
	ТекущаяСтрока = Текстовый1.ТекущаяСтрока;
	ТФ.Уведомление().Показать("ТекущаяСтрока = " + ТекущаяСтрока, -1);
