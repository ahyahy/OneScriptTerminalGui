Окно.ТочкаНаЭлементе (Window.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Игрек (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Точка = Окно0.ТочкаНаЭлементе(45, 10);
Полный пример кода 



Перем ТФ, Окно0, Икс, Игрек;

Процедура ПриОткрытии() Экспорт
	Точка = Окно0.ТочкаНаЭлементе(2, 2);
	Стр = "Левый верхний угол Окно0 имеет X = 0 и Y = 0 
	|Для точки на терминале X = 2 и Y = 2 
	|Окно0.ТочкаНаЭлементе(2, 2).Икс = " + Точка.Икс + "
	|Окно0.ТочкаНаЭлементе(2, 2).Игрек = " + Точка.Игрек;
	Окно0.Текст = Стр;
	
	Икс = Точка.Икс;
	Игрек = Точка.Игрек;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Окно0 = ТФ.Окно("Окно0");
Окно0.Икс = ТФ.Позиция.Абсолютно(0);
Окно0.Игрек = ТФ.Позиция.Абсолютно(0);
Окно0.Ширина = ТФ.Величина.Абсолютно(70);
Окно0.Высота = ТФ.Величина.Абсолютно(20);
Граница1 = ТФ.Граница();
