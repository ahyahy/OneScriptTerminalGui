АргументыСобытия.ИндексСтроки (EventArgs.RowIndex) Свойство

Использование

Только чтение.

Значение

Тип: Число.

Возвращает индекс строки.

Примечание

Пример

АргументыСобытия = ТФ.АргументыСобытия;
ТФ.Верхний.Текст = "СодержимоеИзменено = " + ТекущаяУниверсальнаяДатаВМиллисекундах() + "
|Отправитель = " + АргументыСобытия.Отправитель + "
|ИндексСтроки = " + АргументыСобытия.ИндексСтроки + "
|ИндексКолонки = " + АргументыСобытия.ИндексКолонки + "
|
|Это простой случай для ввода символа.
|Для других случаев, например удаления символа, требуется написать свой другой алгоритм.
|";
