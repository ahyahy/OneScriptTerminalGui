Индикатор.ПолучитьСочетаниеКлавиш (ProgressBar.GetShortcut) Метод

Синтаксис

ПолучитьСочетаниеКлавиш()

Возвращаемое значение

Тип: СписокЗначений.

Список значений из пространства имен односкрипта.

Описание

Возвращает сочетания клавиш, назначенные для элемента.

Примечание

Пример

Список = Индикатор1.ПолучитьСочетаниеКлавиш();
Полный пример кода 



Перем ТФ;

Процедура ПриОткрытии() Экспорт
	ТФ.ОтправитьКлавиши("s", Ложь, Ложь, Ложь);
КонецПроцедуры

Процедура СочетаниеКлавишДействие() Экспорт
	Объект = ТФ.АргументыСобытия.Параметр;
	Список = Объект.ПолучитьСочетаниеКлавиш();
	Стр = "";
	Для Каждого Эл Из Список Цикл
		Стр = Стр + "
		| " + Эл.Значение + " (" + Эл.Представление + ") " + ТекущаяУниверсальнаяДатаВМиллисекундах();
	КонецЦикла;
	ТФ.Уведомление().Показать("СочетаниеКлавиш для " + Объект.Текст + " = " + Стр, -1);
КонецПроцедуры
