ПанельМеню.Кадр (MenuBar.Frame) Свойство

Использование

Чтение и запись.

Значение

Тип: Прямоугольник (Rect).

Возвращает или задает кадр для элемента. Кадр определяется относительно родителя.

Примечание

Пример

Кадр = ПанельМеню1.Кадр;
Полный пример кода 



Перем ТФ, ПанельМеню1, Текст;

Процедура ПриОткрытии() Экспорт
	Кадр = ПанельМеню1.Кадр;
	ТФ.Верхний.Текст = ТФ.Верхний.Текст + "
	|Кадр = " + Кадр + "
	|";
КонецПроцедуры

Процедура Файл_Нажатие() Экспорт
