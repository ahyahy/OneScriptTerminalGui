Текстовый.ПринятиеВозврат (TextView.AllowsReturn) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, создает ли нажатие клавиши Enter новую строку текста или активирует кнопку по умолчанию для верхнего уровня. Если Истина - клавиша Enter создает новую строку текста, Ложь - клавиша Enter активирует кнопку по умолчанию. Значение по умолчанию: Истина.

Примечание

Пример

Текстовый1.ПринятиеВозврат = Ложь;
ПринятиеВозврат = Текстовый1.ПринятиеВозврат;
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Текстовый1.ПринятиеВозврат = Ложь;
	ПринятиеВозврат = Текстовый1.ПринятиеВозврат;
	ТФ.Уведомление().Показать("ПринятиеВозврат = " + ПринятиеВозврат, -1);
