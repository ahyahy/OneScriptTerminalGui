ЦветоваяСхема.Отключено (ColorScheme.Disabled) Свойство

Использование

Чтение и запись.

Значение

Тип: Атрибут (Attribute).

Возвращает или задает цвет переднего плана и фона по умолчанию для текста, когда просмотр отключен.

Примечание

Пример

Отключено = ЦветоваяСхема1.Отключено;
Полный пример кода 



Перем ТФ, ЦветоваяСхема1;

Процедура ПриОткрытии() Экспорт
	Отключено = ЦветоваяСхема1.Отключено;
	ТФ.Верхний.Текст = "ЦветоваяСхема1.Отключено = " + Отключено;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

ЦветоваяСхема1 = ТФ.ЦветоваяСхема();
ЦветоваяСхема1.Отключено = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);

ТФ.Запуск();
