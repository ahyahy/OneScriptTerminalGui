Дерево.ЦветОбычного (TreeView.GetNormalColor) Метод

Синтаксис

ЦветОбычного()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

Дерево1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
Дерево1.ЦветоваяСхема.Нормальный = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, Дерево1, ЦветОбычного;

Процедура ПриОткрытии() Экспорт
	ЦветОбычного = Дерево1.ЦветОбычного();
	Стр = "ЦветОбычного() = " + ЦветОбычного + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветОбычного.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветОбычного.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
