Кнопка.Сфокусирован (Button.HasFocus) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, имеет ли объект фокусировку.

Примечание

Пример

Сфокусирован = Кнопка1.Сфокусирован;
Полный пример кода 



Перем ТФ, Кнопка1, Сфокусирован;

Процедура ПриОткрытии() Экспорт

	ТФ.Верхний.Текст = "Кнопка1.Сфокусирован = " + Сфокусирован;
