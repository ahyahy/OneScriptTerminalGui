Синтаксис

ПолучитьСочетаниеКлавиш()

Возвращаемое значение

Тип: СписокЗначений.

Список значений из пространства имен односкрипта.

Описание

Возвращает сочетания клавиш, назначенные для элемента.

Примечание

Пример

Список = ПолеДаты1.ПолучитьСочетаниеКлавиш();
