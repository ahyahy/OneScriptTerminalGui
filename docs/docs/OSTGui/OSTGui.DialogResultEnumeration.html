РезультатДиалога (DialogResult) Перечисление

Описание

Указывает, какие кнопки были нажаты в диалоговом окне.

Члены

Имя члена Описание Значение
ОК (OK) В диалоговом окне нажата кнопка Открыть (Open) или Сохранить (Save). 1
Отмена (Cancel) В диалоговом окне нажата кнопка Отмена (Cancel). 0
