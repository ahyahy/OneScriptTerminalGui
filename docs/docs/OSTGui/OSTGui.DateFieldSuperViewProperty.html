ПолеДаты.Родитель (DateField.SuperView) Свойство

Использование

Только чтение.

Значение

Тип: Произвольный.

Возвращает контейнер (родителя) для элемента или Неопределено, если этот элемент не был добавлен в контейнер.

Примечание

Пример

Родитель = ПолеДаты1.Родитель;
Полный пример кода 



Перем ТФ, Родитель, ПолеДаты1;

Процедура ПриОткрытии() Экспорт
	Родитель = ПолеДаты1.Родитель;
	ТФ.Уведомление().Показать("Родитель = " + Родитель, -1);
