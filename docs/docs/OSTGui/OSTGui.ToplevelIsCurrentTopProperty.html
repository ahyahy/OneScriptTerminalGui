Верхний.ТекущийСверху (Toplevel.IsCurrentTop) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, указывающее, является ли этот элемент в настоящее время на переднем плане (активен).

Примечание

Анализ для элемента производится в пределах своего родителя.

Пример

ТекущийСверху = Верхний1.ТекущийСверху;
Полный пример кода 



Перем ТФ, Верхний1, Верхний2, Верхний3;

Процедура ПриОткрытии() Экспорт
	ТекущийСверху = Верхний1.ТекущийСверху;
	ТФ.Верхний.Текст = "Верхний1.ТекущийСверху = " + Верхний1.ТекущийСверху + "
	|Верхний2.ТекущийСверху = " + Верхний2.ТекущийСверху + "
	|Верхний3.ТекущийСверху = " + Верхний3.ТекущийСверху + "
	|";
