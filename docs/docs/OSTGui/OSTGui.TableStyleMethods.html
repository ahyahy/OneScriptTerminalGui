СтильТаблицы (TableStyle) Методы

Методы

ПолучитьСтильКолонки (GetColumnStyleIfAny) Возвращает стиль колонки для указанной в параметре колонки или Неопределено, если для колонки не задан пользовательский стиль.
ЦветДесяткаСтрок (RowColorByDecimal) Окрашивает строки в указанные цвета с периодичностью в десять строк. Отсчет ведется со строки с индексом 0.
ЦветСтрок (RowsColor) Окрашивает строки в указанные цвета исходя из переданного в параметре соответствия.
ЧерезСтрочныйЦветСтрок (RowColorThroughLine) Окрашивает строки в указанный цвет с шагом в n строк, исходя из переданного в параметре числа.
