ПунктМеню.ПолучитьСочетаниеКлавиш (MenuBarItem.GetShortcut) Метод

Синтаксис

ПолучитьСочетаниеКлавиш()

Возвращаемое значение

Тип: СписокЗначений.

Список значений из пространства имен односкрипта.

Описание

Возвращает сочетания клавиш, назначенные для элемента.

Примечание

Пример

Список = Объект1.ПолучитьСочетаниеКлавиш();
Полный пример кода 



Перем ТФ, ПунктМеню1;

Процедура ПриОткрытии() Экспорт
	ПунктМеню1.УдалитьСочетаниеКлавиш(ТФ.Клавиши.d_low);

	ТФ.ОтправитьКлавиши("s", Ложь, Ложь, Ложь);
КонецПроцедуры

Процедура СочетаниеКлавишДействие() Экспорт
	Объект1 = ТФ.АргументыСобытия.Параметр;
	Список = Объект1.ПолучитьСочетаниеКлавиш();
	Стр = "";
	Для Каждого Эл Из Список Цикл
		Стр = Стр + "
		| " + Эл.Значение + " (" + Эл.Представление + ") " + ТекущаяУниверсальнаяДатаВМиллисекундах();
	КонецЦикла;
	
	ТФ.Верхний.Текст = "
	|
	|
	|
	|
	|
	|
	|
	|СочетаниеКлавиш для " + Объект1.Заголовок + " = " + Стр;
