ТерминалФормыДляОдноСкрипта.ТаймерНачатьИОстановить (OneScriptTerminalGui.TimerStartAndStop) Метод

Синтаксис

ТаймерНачатьИОстановить(<Сценарий>, <ИмяМетода>, <Интервал>, <КоличествоИтераций>)

Параметры

Сценарий (обязательный)
Тип: Произвольный.
В большинстве случаев для текущего сценария это переменная ЭтотОбъект.
ИмяМетода (обязательный)
Тип: Строка.
Имя обработчика события. Экспортная функция или процедура.
Интервал (необязательный)
Тип: Число.
Время в миллисекундах между отметками таймера. Значение по умолчанию - 1000.
КоличествоИтераций (необязательный)
Тип: Число.
Количество срабатываний таймера до его закрытия. Значение по умолчанию - 1.

Возвращаемое значение

Описание

Запускает вновь созданный объект Таймер (Timer) указанное количество раз и останавливает его по достижении указанного количества итераций.

Примечание

Можно вызвать цепную реакцию запусков, снова вызывая в обработчике события этот метод.

Можно завершить приложение через определенное время.

Можно использовать метод при тестировании, потому что некоторые действия и события возможно выполнить только после перерисовки объектов.

Пример

ТФ.ТаймерНачатьИОстановить(ЭтотОбъект, "ТаймерТик");
