ПолеВыбора.СтильКомпоновки (ComboBox.LayoutStyle) Свойство

Использование

Чтение и запись.

Значение

Тип: СтильКомпоновки (LayoutStyle).

Возвращает или задает стиль компоновки для элемента.

Примечание

Пример

ПолеВыбора1.СтильКомпоновки = ТФ.СтильКомпоновки.Абсолютно;
