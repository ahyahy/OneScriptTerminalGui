Текстовый.ТекстИзменен (TextView.TextChanged) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события, возникающего при изменении свойства Текст (Text). Свойство Текст (Text) объекта Текстовый (TextView) изменяется только в том случае, если оно задано явно, а не по мере ввода пользователем текста. Чтобы получать уведомления об изменении содержимого пользователем, используйте свойство ЕстьИзменения (IsDirty).

Примечание

Аргументы события

Отправитель (Sender) Возвращает источник (объект) последнего события.
Параметр (Parameter) Возвращает объект, заданный при установке обработчика события, для передачи в обработчик события.

Пример

Текстовый1.ТекстИзменен = ТФ.Действие(ЭтотОбъект, "ТекстИзменен");
