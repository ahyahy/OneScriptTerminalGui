Кнопка.ТочкаНаЭлементе (Button.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Игрек (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Точка = Кнопка0.ТочкаНаЭлементе(45, 10);
Полный пример кода 



Перем ТФ, Кнопка0;

Процедура ПриОткрытии() Экспорт
	Точка = Кнопка0.ТочкаНаЭлементе(45, 10);
	Стр = "Левый верхний угол Кнопка0 имеет X = 40 и Y = 5 
	|Для точки на терминале X = 45 и Y = 10 
	|Кнопка0.ТочкаНаЭлементе(45, 10).Икс = " + Точка.Икс + "
	|Кнопка0.ТочкаНаЭлементе(45, 10).Игрек = " + Точка.Игрек;
	ТФ.Верхний.Текст = Стр;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка0 = ТФ.Кнопка("Кнопка0");
Кнопка0.Икс = ТФ.Позиция.Абсолютно(40);
Кнопка0.Игрек = ТФ.Позиция.Абсолютно(5);
Кнопка0.Ширина = ТФ.Величина.Абсолютно(70);
Кнопка0.Высота = ТФ.Величина.Абсолютно(20);
Граница1 = ТФ.Граница();
Граница1.СтильГраницы = ТФ.СтильГраницы.Закругленная;
Кнопка0.Граница = Граница1;
ТФ.Верхний.Добавить(Кнопка0);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
