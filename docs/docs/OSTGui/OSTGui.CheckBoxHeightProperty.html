Флажок.Высота (CheckBox.Height) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает высоту элемента.

Примечание

Пример

Высота = Флажок1.Высота;
Полный пример кода 



Перем ТФ, Высота, Флажок1;

Процедура ПриОткрытии() Экспорт
	Высота = Флажок1.Высота;
	ТФ.Уведомление().Показать("Высота = " + Высота, -1);
