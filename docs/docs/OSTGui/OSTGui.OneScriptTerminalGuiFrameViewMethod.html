ТерминалФормыДляОдноСкрипта.РамкаГруппы (OneScriptTerminalGui.FrameView) Метод

Синтаксис

РамкаГруппы(<Заголовок>, <Икс>, <Игрек>, <Ширина>, <Высота>)

Параметры

Заголовок (необязательный)
Тип: Строка.
заголовок для рамки группы. Значение по умолчанию - "Рамка группы".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла рамки группы. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла рамки группы. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина рамки группы. Значение по умолчанию - 17.
Высота (необязательный)
Тип: Число.
Высота рамки группы. Значение по умолчанию - 5.

Возвращаемое значение

Тип: РамкаГруппы (FrameView).

Описание

Создаёт новый экземпляр класса РамкаГруппы (FrameView).

Примечание

Пример

РамкаГруппы1 = Окно1.Добавить(ТФ.РамкаГруппы("РамкаГруппы1", 1, 1));
