Верхний.Доступность (Toplevel.Enabled) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, может ли элемент реагировать на действия пользователя.

Примечание

Пример

Верхний2.Доступность = Ложь;
Полный пример кода 



Перем ТФ, Верхний1, Верхний2;

Процедура ПриОткрытии() Экспорт
	Верхний1.Текст = "Верхний1.Доступность = " + Верхний1.Доступность;
	Верхний2.Текст = "Верхний2.Доступность = " + Верхний2.Доступность;
