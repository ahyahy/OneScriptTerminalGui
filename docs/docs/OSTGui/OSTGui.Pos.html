Позиция (Pos) Класс

Описание

Описывает положение элемента, которое может быть абсолютным значением, процентом, центрированным или относительным по отношению к конечному размеру.

Этот класс используется для свойств Икс (X) и Игрек (Y) элементов. Экземпляр этого класса представляет сам по себе значение для этих свойств. Экземпляры класса Позиция (Pos) между собой могут складываться, вычитаться, обрабатывать целочисленные значения или представлять процентное отношение к другим величинам.

Важно: Объект Позиция (Pos) нельзя переиспользовать. То есть при обращении к свойству элемента, имеющему тип значения Позиция (Pos) или при присваивании свойству элемента значения этого типа каждый раз будет создан новый экземпляр объекта с типом Позиция (Pos).
