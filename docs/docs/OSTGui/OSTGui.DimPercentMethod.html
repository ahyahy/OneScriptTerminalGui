Величина.Процент (Dim.Percent) Метод

Синтаксис

Процент(<Процент>, <ВычислятьНаОсновеВсегоИсходногоПространства>)

Параметры

Процент (обязательный)
Тип: Число.
ВычислятьНаОсновеВсегоИсходногоПространства (необязательный)
Тип: Булево.
Если Истина процент вычисляется на основе оставшегося пространства после привязки к осям X/Y. Если Ложь вычисляется на основе всего исходного пространства. Значение по умолчанию - Ложь.

Возвращаемое значение

Тип: Величина (Dim).

Описание

Возвращает объект Величина (Dim) на основе указанного процента.

Примечание

Пример

Окно1.Ширина = ТФ.Величина.Процент(50);
