ПолеВвода.ЦветФокуса (TextField.GetFocusColor) Метод

Синтаксис

ЦветФокуса()

Возвращаемое значение

Тип: Атрибут (Attribute).

Описание

Возвращает текущую цветовую схему для текущего фокуса.

Примечание

Пример

ПолеВвода1.ЦветоваяСхема = ТФ.ЦветоваяСхема();
ПолеВвода1.ЦветоваяСхема.Фокус = ТФ.Атрибут(ТФ.Цвет.Синий, ТФ.Цвет.Серый);
Полный пример кода 



Перем ТФ, ПолеВвода1, ЦветФокуса;

Процедура ПриОткрытии() Экспорт
	ЦветФокуса = ПолеВвода1.ЦветФокуса();
	Стр = "ЦветФокуса() = " + ЦветФокуса + "
	|ОсновнойЦвет = " + ТФ.Цвет.Имя(ЦветФокуса.ОсновнойЦвет) + "
	|ЦветФона = " + ТФ.Цвет.Имя(ЦветФокуса.ЦветФона) + "
	|";
	ТФ.Уведомление().Показать(Стр, -1);
