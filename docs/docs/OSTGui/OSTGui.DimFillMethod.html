Величина.Заполнить (Dim.Fill) Метод

Синтаксис

Заполнить(<Отступ>)

Параметры

Отступ (необязательный)
Тип: Число.
Величина отступа. Значение по умолчанию - 0.

Возвращаемое значение

Тип: Величина (Dim).

Описание

Возвращает объект Величина (Dim), который заполняет размерность, но оставляет указанный отступ.

Отступ будет сделан между правой границей объекта и правой границей окна родителя.

Примечание

Пример

Окно1.Ширина = ТФ.Величина.Заполнить(25);
Полный пример кода 



ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Окно1 = ТФ.Окно();
Окно1.Заголовок = "Окно1";
Окно1.Текст = "Окно1.Ширина = ТФ.Величина.Заполнить(25);";
Окно1.Икс = ТФ.Позиция.Абсолютно(1);
Окно1.Игрек = ТФ.Позиция.Абсолютно(1);
Окно1.Ширина = ТФ.Величина.Заполнить(25);
Окно1.Высота = ТФ.Величина.Абсолютно(4);
ТФ.Верхний.Добавить(Окно1);

ТФ.Запуск();
