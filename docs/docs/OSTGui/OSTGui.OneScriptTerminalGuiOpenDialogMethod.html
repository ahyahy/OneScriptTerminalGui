ТерминалФормыДляОдноСкрипта.ДиалогОткрытия (OneScriptTerminalGui.OpenDialog) Метод

Синтаксис

ДиалогОткрытия(<Заголовок>, <Сообщение>, <Икс>, <Игрек>, <Ширина>, <Высота>, <ЯзыкНадписей>)

Параметры

Заголовок (необязательный)
Тип: Строка.
Заголовок для диалога открытия. Значение по умолчанию - "Диалог открытия".
Сообщение (необязательный)
Тип: Строка.
Сообщение для диалога открытия. Значение по умолчанию - "Сообщение".
Икс (необязательный)
Тип: Число.
Координата по оси икс левого верхнего угла диалога открытия. Значение по умолчанию - 1.
Игрек (необязательный)
Тип: Число.
Координата по оси игрек левого верхнего угла диалога открытия. Значение по умолчанию - 1.
Ширина (необязательный)
Тип: Число.
Ширина диалога открытия. Значение по умолчанию - 80.
Высота (необязательный)
Тип: Число.
Высота диалога открытия. Значение по умолчанию - 20.
ЯзыкНадписей (необязательный)
Тип: ЯзыкНадписей (LabelLanguage).
Язык надписей на элементах. Значение по умолчанию - Русский (Russian).

Возвращаемое значение

Тип: ДиалогОткрытия (OpenDialog).

Описание

Создаёт новый экземпляр класса ДиалогОткрытия (OpenDialog).

Примечание

Пример

ДиалогОткрытия1 = ТФ.ДиалогОткрытия(, , , 5);

Перем ТФ, ДиалогОткрытия1;

Процедура ПриОткрытии() Экспорт

	|Выберите какой либо файл и нажмите кнопку Открыть";
КонецПроцедуры

Процедура ДиалогЗакрыт() Экспорт
	АргументыСобытия = ТФ.АргументыСобытия;

		|Элемент = " + АргументыСобытия.Элемент + "
		|";
	Иначе
		Текст = "
		|РезультатДиалога = " + АргументыСобытия.РезультатДиалога;
	КонецЕсли;
	ТФ.Верхний.Текст = Текст;
	ДиалогОткрытия = АргументыСобытия.Элемент;
