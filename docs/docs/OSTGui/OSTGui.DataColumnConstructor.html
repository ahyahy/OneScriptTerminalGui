КолонкаДанных (DataColumn) Конструктор

Синтаксис

КолонкаДанных(<ИмяКолонки>, <ТипДанных>)

Параметры

ИмяКолонки (необязательный)
Тип: Строка.
Строка, представляющая имя создаваемой колонки. Если установлено значение Неопределено или пустая строка (""), то при добавлении объекта в коллекцию колонок будет задано имя по умолчанию.
ТипДанных (необязательный)
Тип: ТипДанных (DataType).
Одно из значений перечисления ТипДанных (DataType). По умолчанию используется строковый тип данных.

Описание

Создаёт новый экземпляр класса КолонкаДанных (DataColumn).

Примечание

Пример

КолонкаДанных1 = ТФ.КолонкаДанных("№ н/п", ТФ.ТипДанных.Число);

		М.Добавить(СтрПолучитьСтроку(Стр,Ч));
	КонецЦикла;
	Возврат М;
