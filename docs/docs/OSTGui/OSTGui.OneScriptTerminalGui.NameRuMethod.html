ТерминалФормыДляОдноСкрипта.[Перечисление].Имя (OneScriptTerminalGui.[Перечисление].Имя) Метод

Синтаксис

[Перечисление].Имя(<Значение>)

Параметры

Значение (обязательный)
Тип: Число.

Возвращаемое значение

Тип: Строка.

Описание

Возвращает строковое представление значения перечисления на русском языке.

Примечание

Пример

МигающийБлок = ТФ.ВидКурсора.Имя(ТФ.ВидКурсора.МигающийБлок);

	|ВидКурсора на русском = " + ТФ.ВидКурсора.Имя(ТФ.ВидКурсора.МигающийБлок) + "
	|", -1);
