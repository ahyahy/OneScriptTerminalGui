ПанельМеню.ВерхнийРодитель (MenuBar.GetTopSuperView) Метод

Синтаксис

ВерхнийРодитель()

Возвращаемое значение

Тип: Произвольный.

Описание

Возвращает самого верхнего родителя.

Примечание

Пример

ВерхнийРодитель = ПанельМеню1.ВерхнийРодитель();
