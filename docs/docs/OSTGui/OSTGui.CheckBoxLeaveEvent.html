Флажок.ПриУходе (CheckBox.Leave) Событие

Использование

Чтение и запись.

Значение

Тип: Действие (Action).

Возвращает или задает код для события, когда элемент теряет фокус.

Примечание

Пример

Флажок1.ПриУходе = ТФ.Действие(ЭтотОбъект, "Флажок1_ПриУходе");

Перем ТФ, Кнопка;

	Кнопка.УстановитьФокус();

	|Флажок1_ПриУходе = " + ТекущаяУниверсальнаяДатаВМиллисекундах();
