Дерево.ПрокрутитьВниз (TreeView.ScrollDown) Метод

Синтаксис

ПрокрутитьВниз()

Возвращаемое значение

Описание

Прокручивает область просмотра вниз на одну строку, не изменяя текущий выбор.

Примечание

Пример

Дерево1.ПрокрутитьВниз();
Полный пример кода 



Перем ТФ, Дерево1;

Процедура ПриОткрытии() Экспорт
	Дерево1.ПерейтиВНачало();
	Дерево1.ПрокрутитьВниз();
	Дерево1.ПрокрутитьВниз();
	Дерево1.ПрокрутитьВниз();
КонецПроцедуры
