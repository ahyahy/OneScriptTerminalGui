Толщина.Лево (Thickness.Left) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает ширину левой стороны ограничивающего прямоугольника.

Примечание

Пример

Лево = Толщина2.Лево;

Перем ТФ, Толщина2;
