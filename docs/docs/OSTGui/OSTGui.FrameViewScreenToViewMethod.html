РамкаГруппы.ТочкаНаЭлементе (FrameView.ScreenToView) Метод

Синтаксис

ТочкаНаЭлементе(<Икс>, <Игрек>)

Параметры

Икс (обязательный)
Тип: Число.
Координата икс.
Игрек (обязательный)
Тип: Число.
Координата игрек.

Возвращаемое значение

Тип: Точка (Point).

Описание

Вычисляет местоположение указанной точки на терминале в координаты на элементе.

Примечание

Пример

Точка = РамкаГруппы1.ТочкаНаЭлементе(45, 10);
Полный пример кода 



Перем ТФ, Точка, РамкаГруппы1;

Процедура ПриОткрытии() Экспорт
	Точка = РамкаГруппы1.ТочкаНаЭлементе(45, 10);
	Стр = "Левый верхний угол РамкаГруппы1 имеет X = 40 и Y = 5 
	|Для точки на терминале X = 45 и Y = 10 
	|РамкаГруппы1.ТочкаНаЭлементе(45, 10).Икс = " + Точка.Икс + "
	|РамкаГруппы1.ТочкаНаЭлементе(45, 10).Игрек = " + Точка.Игрек;
	ТФ.Уведомление().Показать(Стр, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

РамкаГруппы1 = ТФ.Верхний.Добавить(ТФ.РамкаГруппы("РамкаГруппы1", 1, 1));
РамкаГруппы1.Ширина = ТФ.Величина.Процент(25);
РамкаГруппы1.Высота = ТФ.Величина.Процент(35);

Кнопка1 = ТФ.Кнопка("Кнопка1", 2, 1);
Кнопка1.Граница = ТФ.Граница();
Кнопка1.Граница.СтильГраницы = ТФ.СтильГраницы.Закругленная;

Кнопка2 = ТФ.Кнопка("Кнопка2", 2);
Кнопка2.Ниже(Кнопка1, 1);
Кнопка2.Граница = Кнопка1.Граница;

РамкаГруппы1.Добавить(Кнопка1);
РамкаГруппы1.Добавить(Кнопка2);

РамкаГруппы1.Икс = ТФ.Позиция.Абсолютно(40);
РамкаГруппы1.Игрек = ТФ.Позиция.Абсолютно(5);
РамкаГруппы1.Ширина = ТФ.Величина.Абсолютно(70);
РамкаГруппы1.Высота = ТФ.Величина.Абсолютно(20);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
