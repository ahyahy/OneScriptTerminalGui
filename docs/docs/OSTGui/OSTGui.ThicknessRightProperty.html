Толщина.Право (Thickness.Right) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает ширину правой стороны ограничивающего прямоугольника.

Примечание

Пример

Право = Толщина2.Право;
Полный пример кода 



Перем ТФ, Толщина2;

Процедура ПриОткрытии() Экспорт
	Право = Толщина2.Право;
	ТФ.Верхний.Текст = "Толщина2.Право = " + Право;
