Текстовый.ЕстьИстория (TextView.HasHistoryChanges) Свойство

Использование

Только чтение.

Значение

Тип: Булево.

Возвращает значение, показывающее, есть ли в тексте изменения, внесённые в историю. Истина, если в тексте есть изменения, внесённые в историю, Ложь в противном случае.

Примечание

Пример

ЕстьИстория = Текстовый1.ЕстьИстория;
Полный пример кода 



Перем ТФ, Текстовый1;

Процедура ПриОткрытии() Экспорт
	ТФ.ТаймерНачатьИОстановить(ЭтотОбъект, "ТаймерТик");
КонецПроцедуры

Процедура ТаймерТик() Экспорт
	Текстовый1.ПозицияКурсора = ТФ.Точка(2, 0);
	Текстовый1.Выборка = Ложь;
	Текстовый1.УдалитьСимволСлева();
	ЕстьИстория = Текстовый1.ЕстьИстория;
	ТФ.Уведомление().Показать("ЕстьИстория = " + ЕстьИстория, -1);
