СтрокаСостояния.ЦветВыделенного (StatusBar.GetHotNormalColor) Метод

Синтаксис

ЦветВыделенного()

Возвращаемое значение

Тип: ЦветоваяСхема (ColorScheme).

Описание

Возвращает текущую цветовую схему если свойство Доступность (Enabled) включено.

Примечание

Пример

ЦветВыделенного = СтрокаСостояния1.ЦветВыделенного();
Полный пример кода 



Перем ТФ, СтрокаСостояния1;

Процедура ПриОткрытии() Экспорт
	ЦветВыделенного = СтрокаСостояния1.ЦветВыделенного();
	ТФ.Верхний.Текст = ТФ.Верхний.Текст + "
	|СтрокаСостояния1.ЦветВыделенного() = " + ЦветВыделенного + "
	|ОсновнойЦвет = " + ЦветВыделенного.ОсновнойЦвет + "
	|ЦветФона = " + ЦветВыделенного.ЦветФона + "
	|";
