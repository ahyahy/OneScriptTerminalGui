Кнопка.Ширина (Button.Width) Свойство

Использование

Чтение и запись.

Значение

Тип: Величина (Dim).

Возвращает или задает ширину кнопки.

Примечание

Пример

Кнопка1.Ширина = ТФ.Величина.Процент(50);
Полный пример кода 



Перем ТФ, Кнопка1, Ширина;

Процедура ПриОткрытии() Экспорт
	Ширина = Кнопка1.Ширина;
	ТФ.Верхний.Текст = "Кнопка1.Ширина = " + Ширина;
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка("Кнопка1");
Кнопка1.Икс = ТФ.Позиция.Абсолютно(4);
Кнопка1.Игрек = ТФ.Позиция.Абсолютно(4);
Кнопка1.СтильКомпоновки = ТФ.СтильКомпоновки.Вычислено;
Кнопка1.Ширина = ТФ.Величина.Процент(50);
Кнопка1.Высота = ТФ.Величина.Процент(18);
ТФ.Верхний.Добавить(Кнопка1);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
