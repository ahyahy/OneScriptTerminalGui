СтильИндикатора (ProgressBarStyle) Перечисление

Описание

Указывает стиль, который используется в индикаторе для отображения хода выполнения операции.

Члены

Имя члена Описание Значение
БегущаяПолоса (MarqueeContinuous) Отображает прогресс путем непрерывной прокрутки полосы индикатора. 3
БегущиеБлоки (MarqueeBlocks) Отображает прогресс путем непрерывной прокрутки блоков индикатора. 2
Блоки (Blocks) Отображает прогресс, увеличивая количество сегментированных блоков. 0
Полоса (Continuous) Отображает прогресс, увеличивая размер плавной непрерывной полосы. 1
