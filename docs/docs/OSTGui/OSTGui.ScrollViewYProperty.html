Прокручиваемый.Игрек (ScrollView.Y) Свойство

Использование

Чтение и запись.

Значение

Тип: Позиция (Pos).

Возвращает или задает объект Позиция (Pos), соответствующий координате по оси Y.

Примечание

Пример

Игрек = Прокручиваемый1.Игрек;
Полный пример кода 



Перем ТФ, Игрек, Прокручиваемый1;

Процедура ПриОткрытии() Экспорт
	Игрек = Прокручиваемый1.Игрек;
	ТФ.Уведомление().Показать("Игрек = " + Игрек, -1);
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Прокручиваемый1 = ТФ.Верхний.Добавить(ТФ.Прокручиваемый(2, 2, 50, 15));
Прокручиваемый1.ЦветоваяСхема = ТФ.Цвета.Верхний;
Прокручиваемый1.РазмерСодержимого = ТФ.Размер(100, 50);
Прокручиваемый1.СмещениеСодержимого = ТФ.Точка(0, 0);

Надпись1 = Прокручиваемый1.Добавить(ТФ.Надпись("Текст для Надпись1 Текст для Надпись1 Текст для Надпись1", 0, 0, 60, 25));
Надпись1.ЦветоваяСхема = ТФ.Цвета.Ошибка;
Надпись1.АвтоРазмер = Ложь;
