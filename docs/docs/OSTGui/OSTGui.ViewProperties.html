Элемент (View) Свойства

Свойства

АвтоРазмер (AutoSize) Возвращает или задает значение, указывающее, будет ли размер элемента автоматически изменяться в соответствии с его текстом. По умолчанию установлено значение Ложь. Если это свойство имеет значение Истина, то ширина и высота останутся неизменными, если размер текста меньше. Если размер текста больше, границы будут изменены в соответствии с текстом.
ВертикальноеВыравниваниеТекста (VerticalTextAlignment) Возвращает или задает, как текст элемента выровнен вертикально. Изменение этого свойства повторно отобразит элемент.
Верх (Top) Возвращает объект Позиция (Pos), соответствующий координате по оси Y верхнего края элемента.
ВФокусе (Focused) Возвращает элемент, который в настоящее время имеет фокус или Неопределено, если ничего не сфокусировано.
ВыравниваниеТекста (TextAlignment) Возвращает или задает, как текст элемента выровнен горизонтально. Изменение этого свойства повторно отобразит элемент.
Высота (Height) Возвращает или задает высоту элемента.
Граница (Border) Экземпляр класса Граница Border для данного элемента.
Границы (Bounds) Возвращает прямоугольник, используемый для элемента.
Данные (Data) Возвращает или задает произвольные данные для элемента.
Добавлено (IsAdded) Возвращает информацию о том добавлен ли элемент к родителю.
Доступность (Enabled) (унаследовано от Респондент (Responder)) Возвращает или задает значение, указывающее, может ли элемент реагировать на действия пользователя.
Игрек (Y) Возвращает или задает объект Позиция (Pos), соответствующий координате по оси Y.
Идентификатор (Id) Возвращает или задает идентификатор для элемента.
Икс (X) Возвращает или задает объект Позиция (Pos), соответствующий координате по оси X.
Кадр (Frame) Возвращает или задает кадр для элемента. Кадр определяется относительно родителя.
КлавишаВызова (HotKey) Возвращает клавишу вызова, определенную для элемента. Событие нажатия сработает, если элемент имеет фокус.
Лево (Left) Возвращает объект Позиция (Pos), соответствующий координате по оси X левого края элемента.
Метка (Tag) Возвращает или задает объект, содержащий дополнительные данные.
НаправлениеТекста (TextDirection) Возвращает или задает направление текста элемента. Изменение этого свойства повторно отобразит элемент.
Низ (Bottom) Возвращает объект Позиция (Pos), соответствующий координате по оси Y, являющейся суммой значений свойств Игрек (Y) и Высота (Height) данного элемента.
ОтключитьКлавишуВызова (DisableHotKey) Отключает поддержку клавиши вызова для элемента. Значение по умолчанию - Ложь.
Отображать (Visible) (унаследовано от Респондент (Responder)) Возвращает или задает значение, указывающее, отображается ли этот элемент и все его дочерние элементы.
ОформительТекста (TextFormatter) Возвращает или задает экземпляр класса ОформительТекста TextFormatter.
Подэлементы (Subviews) Возвращает коллекцию подэлементов, содержащую дочерние элементы этого элемента.
ПорядокОбхода (TabIndex) Возвращает или задает индекс текущего элемента в списке порядка обхода элементов.
Право (Right) Возвращает объект Позиция (Pos), соответствующий координате по оси X, являющейся суммой значений свойств Икс (X) и Ширина (Width) данного элемента.
Родитель (SuperView) Возвращает контейнер (родителя) для элемента или Неопределено, если этот элемент не был добавлен в контейнер.
СтильКомпоновки (LayoutStyle) Возвращает или задает стиль компоновки для элемента.
Сфокусирован (HasFocus) (унаследовано от Респондент (Responder)) Возвращает значение, указывающее, имеет ли объект фокусировку.
ТабФокус (TabStop) Возвращает или задает значение, указывающее, может ли пользователь перевести фокус на данный элемент при помощи клавиши Tab. Применимо только если свойство Фокусируемый CanFocus также установлено в Истина.
Текст (Text) Возвращает или задает текст для элемента.
ТекущийСверху (IsCurrentTop) Возвращает значение, указывающее, является ли этот элемент в настоящее время на переднем плане (активен).
Фокусируемый (CanFocus) (унаследовано от Респондент (Responder)) Возвращает или задает значение, указывающее, может ли элемент управления получать фокус.
ЦветоваяСхема (ColorScheme) Возвращает или задает цветовую схему для элемента. Если она не определена, будет возвращена цветовая схема родителя.
Ширина (Width) Возвращает или задает ширину элемента.
