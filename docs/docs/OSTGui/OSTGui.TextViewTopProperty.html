Текстовый.Верх (TextView.Top) Свойство

Использование

Только чтение.

Значение

Тип: Позиция (Pos).

Возвращает объект Позиция (Pos), соответствующий координате по оси Y верхнего края элемента.

Примечание

Пример

Верх = Текстовый1.Верх;
Полный пример кода 



Перем ТФ, Верх, Текстовый1;

Процедура ПриОткрытии() Экспорт
	Верх = Текстовый1.Верх;
	ТФ.Уведомление().Показать("Верх = " + Верх, -1);
