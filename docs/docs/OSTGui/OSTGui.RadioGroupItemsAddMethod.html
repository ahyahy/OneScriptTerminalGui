МеткиПереключателя.Добавить (RadioGroupItems.Add) Метод

Синтаксис

Добавить(<Метка>)

Параметры

Метка (обязательный)
Тип: Строка.
Добавляемая метка для переключателя.

Возвращаемое значение

Описание

Добавляет метку в конец коллекции.

Примечание

Пример

Метки1 = Переключатель1.Метки;
Метки1.Добавить("Метка1");
