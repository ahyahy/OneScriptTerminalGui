ДиалогОткрытия.ВыборКаталога (OpenDialog.CanChooseDirectories) Свойство

Использование

Чтение и запись.

Значение

Тип: Булево.

Возвращает или задает значение, указывающее, можно ли выбирать каталоги.

Значение Истина, если можно выбирать каталоги, в противном случае Ложь. Значение по умолчанию - Ложь.

Примечание

Пример

ДиалогОткрытия1.ВыборКаталога = Истина;
Полный пример кода 



Перем ТФ, ДиалогОткрытия1, ВыборКаталога;

Процедура ПриОткрытии() Экспорт
	ВыборКаталога = ДиалогОткрытия1.ВыборКаталога;
	ТФ.Уведомление().Показать("ВыборКаталога = " + ВыборКаталога, -1);
