СтильГраницы (BorderStyle) Перечисление

Описание

Указывает стиль границы.

Члены

Имя члена Описание Значение
Двойная (Double) Граница нарисована двойной линией. 2
Закругленная (Rounded) Граница нарисована одинарной линией и закругленными углами. 3
Одинарная (Single) Граница нарисована одинарной линией. 1
Отсутствие (None) Ни одна граница не нарисована. 0
