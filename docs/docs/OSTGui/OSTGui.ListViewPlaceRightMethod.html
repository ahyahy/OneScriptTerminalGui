СписокЭлементов.Правее (ListView.PlaceRight) Метод

Синтаксис

Правее(<Элемент>, <Смещение>)

Параметры

Элемент (обязательный)
Тип: Произвольный.
Элемент, относительно которого производим смещение.
Смещение (обязательный)
Тип: Число.
Величина смещения в целых числах левого края смещаемого элемента от правого края объекта, указанного в параметре Элемент.

Возвращаемое значение

Описание

Смещает правее этот элемент относительно объекта, указанного в параметре Элемент, на величину, указанную в параметре Смещение.

Примечание

Пример

СписокЭлементов1.Правее(Кнопка0, 1);
