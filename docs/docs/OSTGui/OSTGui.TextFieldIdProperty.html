ПолеВвода.Идентификатор (TextField.Id) Свойство

Использование

Чтение и запись.

Значение

Тип: Строка.

Возвращает или задает идентификатор для элемента.

Примечание

Пример

ПолеВвода1.Идентификатор = "Идент";
Полный пример кода 



Перем ТФ, ПолеВвода1, Идентификатор, Окно1;

Процедура ПриОткрытии() Экспорт
	Идентификатор = ПолеВвода1.Идентификатор;
	ТФ.Уведомление().Показать("Идентификатор = " + Идентификатор, 3000);
