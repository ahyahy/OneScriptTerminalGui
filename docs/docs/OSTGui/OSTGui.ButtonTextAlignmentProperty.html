Кнопка.ВыравниваниеТекста (Button.TextAlignment) Свойство

Использование

Чтение и запись.

Значение

Тип: ВыравниваниеТекста (TextAlignment).

Возвращает или задает, как текст элемента выровнен горизонтально. Изменение этого свойства повторно отобразит элемент.

Примечание

Пример

Кнопка1.ВыравниваниеТекста = ТФ.ВыравниваниеТекста.Подобранный;
