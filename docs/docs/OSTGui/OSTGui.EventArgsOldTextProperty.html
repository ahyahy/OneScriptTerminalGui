АргументыСобытия.СтарыйТекст (EventArgs.OldText) Свойство

Использование

Только чтение.

Значение

Тип: Строка.

Возвращает текст элемента до изменения.

Примечание

Пример

Процедура ТекстИзменен() Экспорт
	АргументыСобытия = ТФ.АргументыСобытия;
	ТФ.Верхний.Текст = Текст + "
	|СтарыйТекст = " + АргументыСобытия.СтарыйТекст + "
	|";
