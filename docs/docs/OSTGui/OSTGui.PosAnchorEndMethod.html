Позиция.ЯкорьКонец (Pos.AnchorEnd) Метод

Синтаксис

ЯкорьКонец(<Отступ>)

Параметры

Отступ (необязательный)
Тип: Число.

Возвращаемое значение

Тип: Позиция (Pos).

Описание

Возвращает объект Позиция (Pos), привязанный к концу (правой стороне или нижней части) измерения, что полезно для выравнивания макета по правому краю или нижней части.

Примечание

Пример

Кнопка1.Икс = ТФ.Позиция.ЯкорьКонец(Кнопка1.Кадр.Ширина + 2);
Полный пример кода 



Перем ТФ, Кнопка1, Позиция;

Процедура ПриОткрытии() Экспорт
	Позиция = Кнопка1.Игрек;
	
	ТФ.Верхний.Текст = 
	"Позиция = " + Позиция + "
	|";
КонецПроцедуры

ПодключитьВнешнююКомпоненту("C:\444\111\OneScriptTerminalGui\OneScriptTerminalGui\bin\Debug\OneScriptTerminalGui.dll");
ТФ = Новый ТерминалФормыДляОдноСкрипта();

Кнопка1 = ТФ.Кнопка("Кнопка1");
Кнопка1.Икс = ТФ.Позиция.ЯкорьКонец(Кнопка1.Кадр.Ширина + 2);
Кнопка1.Игрек = ТФ.Позиция.ЯкорьКонец(Кнопка1.Кадр.Высота + 1);
Кнопка1.Граница = ТФ.Граница();
Кнопка1.Граница.СтильГраницы = ТФ.СтильГраницы.Закругленная;
ТФ.Верхний.Добавить(Кнопка1);

ТФ.ПриОткрытии = ТФ.Действие(ЭтотОбъект, "ПриОткрытии");

ТФ.Запуск();
