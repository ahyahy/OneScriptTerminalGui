Переключатель.МакетПереключателя (RadioGroup.DisplayMode) Свойство

Использование

Чтение и запись.

Значение

Тип: МакетПереключателя (DisplayModeLayout).

Возвращает или задает способ компоновки для группы меток переключателя.

Примечание

Пример

Переключатель1.МакетПереключателя = ТФ.МакетПереключателя.Горизонтально;
МакетПереключателя = Переключатель1.МакетПереключателя;
ТФ.Уведомление().Показать("МакетПереключателя = " + ТФ.МакетПереключателя.Имя(МакетПереключателя), -1);
