Дерево.ПереместитьВыбор (TreeView.AdjustSelection) Метод

Синтаксис

Переместить(<Величина>, <РасширитьВыбор>)

Параметры

Величина (обязательный)
Тип: Число.
Количество строк на экране, на которое нужно переместить область выбора.
РасширитьВыбор (необязательный)
Тип: Булево.
Установите значение Истина для расширения области выбора (при условии, что свойство МножественныйВыбор (MultiSelect) установлено в Истина). Если Ложь - будет произведена замена области выбора. Значение по умолчанию - Ложь.

Возвращаемое значение

Описание

Перемещает область выбора.

Примечание

Если в данный момент ничего не выбрано или выбранный объект больше не отображается в дереве, вместо него выбирается первый объект в дереве.

Пример

Дерево1.ПереместитьВыбор(3, Истина);
Полный пример кода 



Перем ТФ, Дерево1, Узел6;

Процедура ПриОткрытии() Экспорт
	Дерево1.Перейти(Узел6);
	Дерево1.ПереместитьВыбор(3, Истина);
	
	ВсеВыбранные = Дерево1.ПолучитьВсеВыбранные();
	Стр = "";
	Для А = 0 По ВсеВыбранные.ВГраница() Цикл
		Эл = ВсеВыбранные.Получить(А);
		Стр = Стр + Эл.Текст + ТФ.НоваяСтрока;
	КонецЦикла;
	ТФ.Уведомление().Показать("ВсеВыбранные = "+ ТФ.НоваяСтрока + Стр, -1);
