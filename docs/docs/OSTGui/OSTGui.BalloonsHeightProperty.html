Уведомление.Высота (Balloons.Height) Свойство

Использование

Чтение и запись.

Значение

Тип: Число.

Возвращает или задает высоту окна уведомления.

Примечание

Пример

Высота = Уведомление1.Высота;
Полный пример кода 



Перем ТФ, Высота, Уведомление1;

Процедура ПриОткрытии() Экспорт
	Высота = Уведомление1.Высота;
	Уведомление1.Показать("Высота = " + Высота, -1);
