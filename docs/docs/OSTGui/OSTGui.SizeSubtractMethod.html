Размер.Вычесть (Size.Subtract) Метод

Синтаксис

Вычесть(<Размер1>, <Размер2>)

Параметры

Размер1 (обязательный)
Тип: Размер (Size).
Структура Размер (Size) из которой вычитаются значения высоты и ширины.
Размер2 (обязательный)
Тип: Размер (Size).
Структура Размер (Size) высота и ширина которой вычитаются из первой структуры.

Возвращаемое значение

Тип: Размер (Size).

Структура Размер (Size).

Описание

Вычитает ширину и высоту структуры одного размера из ширины и высоты структуры другого размера.

Примечание

Пример

Вычесть = ТФ.Размер().Вычесть(Размер1, Размер2);
