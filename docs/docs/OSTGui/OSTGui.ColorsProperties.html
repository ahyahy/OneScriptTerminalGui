Цвета (Colors) Свойства

Свойства

Верхний (TopLevel) Возвращает или задает цветовую схему верхнего уровня приложения по умолчанию.
Диалог (Dialog) Возвращает или задает цветовую схему диалогового окна для стандартных всплывающих диалоговых окон.
Меню (Menu) Возвращает или задает цветовую схему для строки меню.
Основа (Base) Возвращает или задает базовую цветовую схему для видов верхнего уровня по умолчанию.
Ошибка (Error) Возвращает или задает цветовую схему для отображения ошибок.
